    Section 3.2 Limits and Continuity

    In this section, we will explore the concepts of limits and continuity for functions of several variables. Mastering these foundational ideas is crucial for analyzing the localized behavior of multivariable functions before extending our study to rates of change.
    🔗
    We will begin by defining the rigorous condition for the limit of a multivariate function. Next, we will introduce the two-path rule to definitively prove when a limit fails to exist along distinct linear or parabolic geometric pathways. We will then explore methods for evaluating existing limits, including direct algebraic substitution and the Squeeze Theorem. Finally, we will establish the mathematical criteria required for a function to be continuous on its domain, examining both piecewise definitions and the continuity profiles of composite functions.
    🔗
    By the end of this section, you will have a solid conceptual framework for evaluating limits and determining the continuity of functions of several variables, providing the essential toolkit required to study multivariable derivatives.
    🔗

    Definition 3.11.

    We write
    \begin{equation*} \lim_{(x,y)\to (a,b)}f(x,y)=L \end{equation*}
    and say that the limit of \(f(x,y)\) as \((x,y)\) approaches \((a,b)\) is \(L\) if we can make the values of \(f(x,y)\) as close to \(L\) as we please by taking the point \((x,y)\) sufficiently close to the point \((a,b)\) but not equal to \((a,b)\text{.}\)
    🔗
    This is equivalent to stating that for every \(\epsilon > 0\text{,}\) there exists a \(\delta > 0\) such that if
    \begin{equation*} 0 < \sqrt{(x-a)^2+(y-b)^2} < \delta \end{equation*}
    then
    \begin{equation*} |f(x,y)-L| < \epsilon \end{equation*}
    🔗
    🔗
    Alternately, this may be denoted as
    \begin{equation*} \lim_{\substack{x \to a \\ y \to b}} f(x,y)=L \quad \text{and} \quad f(x,y) \to L \text{ as } (x,y)\to (a,b) \end{equation*}
    🔗

    Remark 3.12. Two-Path Rule.

    If \(f(x,y)\to L_1\) as \((x,y)\to (a,b)\) along a path \(C_1\) and \(f(x,y)\to L_2\) as \((x,y)\to (a,b)\) along a path \(C_2\) where \(L_1 \neq L_2\text{,}\) then
    \begin{equation*} \lim_{(x,y)\to (a,b)}f(x,y) \end{equation*}
    does not exist.
    🔗
    🔗

    Example 3.13.

    Find the following limits:
    🔗
    1. \(\displaystyle \displaystyle\lim_{(x,y)\to (2,-3)}(x^3-4xy^2+5y-7)\)
      🔗
      🔗
    2. \(\displaystyle \displaystyle\lim_{(x,y)\to (3,4)}\frac{x^2-y^2}{\sqrt{x^2+y^2}}\)
      🔗
      🔗
    🔗
    Solution.
    1. Since the function \(f(x,y) = x^3-4xy^2+5y-7\) is a polynomial function of two variables, it is continuous everywhere on its domain \(\mathbb{R}^2\text{.}\) Therefore, we can evaluate the limit directly by substituting \(x = 2\) and \(y = -3\) into the expression:
      \begin{align*} \lim_{(x,y)\to (2,-3)}(x^3-4xy^2+5y-7) \amp = (2)^3 - 4(2)(-3)^2 + 5(-3) - 7\\ \amp = 8 - 4(2)(9) - 15 - 7\\ \amp = 8 - 72 - 15 - 7\\ \amp = -86 \end{align*}
      🔗
      🔗
    2. The function \(f(x,y) = \frac{x^2-y^2}{\sqrt{x^2+y^2}}\) is a rational function involving a radical. The denominator is non-zero at the limit point since \(\sqrt{3^2+4^2} = \sqrt{25} = 5 \neq 0\text{.}\) Because the denominator does not equal zero at \((3,4)\text{,}\) the function is continuous at this point, and we can find the limit using direct substitution of \(x = 3\) and \(y = 4\text{:}\)
      \begin{align*} \lim_{(x,y)\to (3,4)}\frac{x^2-y^2}{\sqrt{x^2+y^2}} \amp = \frac{3^2 - 4^2}{\sqrt{3^2 + 4^2}}\\ \amp = \frac{9 - 16}{\sqrt{9 + 16}}\\ \amp = \frac{-7}{\sqrt{25}}\\ \amp = -\frac{7}{5} \end{align*}
      🔗
      🔗
    🔗
    🔗
    🔗

    Example 3.14.

    Show that
    \begin{equation*} \lim_{(x,y)\to (0,0)}\frac{x^2-y^2}{x^2+y^2} \end{equation*}
    does not exist.
    🔗
    Solution.
    Let \(f(x,y) = \frac{x^2-y^2}{x^2+y^2}\text{.}\) To show that the limit does not exist as \((x,y) \to (0,0)\text{,}\) we can analyze the behavior of the function along different paths approaching the origin.
    🔗
    First, let’s approach \((0,0)\) along the \(x\)-axis. On this path, \(y = 0\text{,}\) which simplifies the function to:
    \begin{equation*} f(x,0) = \frac{x^2 - 0^2}{x^2 + 0^2} = \frac{x^2}{x^2} = 1 \quad \text{for all } x \neq 0 \end{equation*}
    Therefore, as we approach along the \(x\)-axis, the limit is:
    \begin{equation*} \lim_{x \to 0} f(x,0) = 1 \end{equation*}
    🔗
    Next, we approach along the \(y\)-axis by setting \(x = 0\text{.}\) This simplifies the function to:
    \begin{equation*} f(0,y) = \frac{0^2 - y^2}{0^2 + y^2} = \frac{-y^2}{y^2} = -1 \quad \text{for all } y \neq 0 \end{equation*}
    Therefore, as we approach along the \(y\)-axis, the limit is:
    \begin{equation*} \lim_{y \to 0} f(0,y) = -1 \end{equation*}
    🔗
    Since the function approaches two different limiting values (\(1\) and \(-1\)) along two distinct directional paths, the two-path rule applies. We conclude that the given limit
    \begin{equation*} \lim_{(x,y)\to (0,0)}\frac{x^2-y^2}{x^2+y^2} \end{equation*}
    does not exist.
    🔗
    🔗
    🔗

    Example 3.15.

    Show that
    \begin{equation*} \lim_{(x,y)\to (0,0)} \frac{xy}{x^2+y^2} \end{equation*}
    does not exist.
    🔗
    Solution.
    Let \(f(x,y) = \frac{xy}{x^2+y^2}\text{.}\) To test the existence of the limit as we approach the origin, we check the behavior of the function along multiple distinct paths.
    🔗
    First, let’s approach \((0,0)\) along the \(x\)-axis. Setting \(y = 0\) gives:
    \begin{equation*} f(x,0) = \frac{x(0)}{x^2+0^2} = 0 \quad \text{for all } x \neq 0 \end{equation*}
    Thus, the limit along this path is \(0\text{.}\)
    🔗
    Next, we approach along the \(y\)-axis by setting \(x = 0\text{:}\)
    \begin{equation*} f(0,y) = \frac{(0)y}{0^2+y^2} = 0 \quad \text{for all } y \neq 0 \end{equation*}
    The limit along this path is also \(0\text{.}\)
    🔗
    Although we have obtained identical limits along both coordinate axes, this is not sufficient to prove that the overall limit is \(0\text{.}\) We must test other paths, such as the diagonal line \(y = x\text{.}\) Substituting \(y = x\) into our function for \(x \neq 0\) yields:
    \begin{equation*} f(x,x) = \frac{x(x)}{x^2+x^2} = \frac{x^2}{2x^2} = \frac{1}{2} \end{equation*}
    Therefore, as we approach the origin along the line \(y = x\text{,}\) the function value remains constantly equal to \(\frac{1}{2}\text{,}\) giving a path limit of \(\frac{1}{2}\text{.}\)
    🔗
    Since we have obtained different limiting values along two different paths (\(0\) along the axes and \(\frac{1}{2}\) along the diagonal), the two-path rule applies. We conclude that the given limit does not exist.
    🔗
    🔗
    🔗

    Checkpoint 3.17.

    If \(f(x,y) = \frac{x^2y}{x^4+y^2}\text{,}\) does
    \begin{equation*} \lim_{(x,y)\to (0,0)}\frac{x^2y}{x^4+y^2} \end{equation*}
    exist?
    🔗
    Solution.
    To test if the limit exists, let’s analyze the behavior of the function along different families of paths approaching the origin.
    🔗
    First, let’s approach \((0,0)\) along an arbitrary straight line path \(y=mx\) (where \(m \neq 0\) is the slope). Substituting \(y=mx\) into the function yields:
    \begin{align*} f(x,mx) \amp = \frac{x^2(mx)}{x^4+(mx)^2}\\ \amp = \frac{mx^3}{x^4+m^2x^2}\\ \amp = \frac{mx^3}{x^2(x^2+m^2)}\\ \amp = \frac{mx}{x^2+m^2} \quad \text{for } x \neq 0 \end{align*}
    Taking the limit as \(x \to 0\text{,}\) we find:
    \begin{equation*} \lim_{x \to 0} f(x,mx) = \frac{0}{0+m^2} = 0 \end{equation*}
    If we approach along the coordinate axes (\(x=0\) or \(y=0\)), the function is identically \(0\text{,}\) so the limit is also \(0\text{.}\) Thus, \(f(x,y)\) has the same limiting value of \(0\) along every single straight-line path passing through the origin.
    🔗
    However, this still does not guarantee that the overall limit is \(0\text{.}\) Let’s approach \((0,0)\) along a parabolic path by setting \(y=x^2\) for \(x \neq 0\text{.}\) Substituting this curve into our expression yields:
    \begin{equation*} f(x,x^2) = \frac{x^2(x^2)}{x^4+(x^2)^2} = \frac{x^4}{x^4+x^4} = \frac{x^4}{2x^4} = \frac{1}{2} \end{equation*}
    Therefore, as we approach the origin along the parabola \(y=x^2\text{,}\) the limit value is consistently \(\frac{1}{2}\text{.}\)
    🔗
    Since we have obtained different limiting values along two different paths (\(0\) along all linear paths and \(\frac{1}{2}\) along the parabolic path), the two-path rule applies. We conclude that the given limit does not exist.
    🔗
    🔗
    🔗

    Checkpoint 3.18.

    Find
    \begin{equation*} \lim_{(x,y)\to (0,0)} \frac{3x^2y}{x^2+y^2} \end{equation*}
    🔗
    Solution.
    We can determine this limit by analyzing the absolute distance from \(f(x,y)\) to \(0\text{:}\)
    \begin{equation*} \left|\frac{3x^2y}{x^2+y^2}-0\right| = \left|\frac{3x^2y}{x^2+y^2}\right| = \frac{3x^2|y|}{x^2+y^2} \end{equation*}
    🔗
    We notice an inequality involving the components of the denominator: since \(y^2 \geq 0\text{,}\) it must be true that \(x^2 \leq x^2+y^2\) for all real numbers. Dividing both sides by \(x^2+y^2\) (where \((x,y) \neq (0,0)\)) establishes that:
    \begin{equation*} \frac{x^2}{x^2+y^2} \leq 1 \end{equation*}
    🔗
    Multiplying this bounded inequality by \(3|y|\) sets up our boundary sandwich:
    \begin{equation*} 0 \leq \frac{3x^2|y|}{x^2+y^2} \leq 3|y|(1) = 3|y| \end{equation*}
    🔗
    We evaluate the limits of our bounding bounds as \((x,y) \to (0,0)\text{.}\) Since \(\lim_{(x,y)\to(0,0)} 0 = 0\) and \(\lim_{(x,y)\to(0,0)} 3|y| = 0\text{,}\) the Squeeze Theorem applies. Therefore, we conclude that:
    \begin{equation*} \lim_{(x,y)\to (0,0)} \frac{3x^2y}{x^2+y^2}=0 \end{equation*}
    🔗
    🔗
    🔗

    Definition 3.19.

    A function \(f\) of two variables is continuous at \((a,b)\) if
    \begin{equation*} \lim_{(x,y)\to (a,b)}f(x,y)=f(a,b) \end{equation*}
    We say \(f\) is continuous on \(D\) if \(f\) is continuous at every point \((a,b)\) in \(D\text{.}\)
    🔗
    🔗

    Example 3.20.

    Find
    \begin{equation*} \lim_{(x,y)\to (1,2)} (x^2y^3-x^3y^2+3x+2y)\text{.} \end{equation*}
    🔗
    Solution.
    Since \(f(x,y)=x^2y^3-x^3y^2+3x+2y\) is a polynomial function, it is continuous everywhere on \(\mathbb{R}^2\text{.}\) Therefore, we can find the limit via direct substitution:
    \begin{align*} \lim_{(x,y)\to (1,2)} (x^2y^3-x^3y^2+3x+2y) \amp = (1)^2(2)^3 - (1)^3(2)^2 + 3(1) + 2(2)\\ \amp = 1(8) - 1(4) + 3 + 4\\ \amp = 8 - 4 + 3 + 4\\ \amp = 11 \end{align*}
    🔗
    🔗
    🔗

    Example 3.21.

    Where is the function \(f(x,y)= \frac{x^2-y^2}{x^2+y^2}\) continuous?
    🔗
    Solution.
    Since \(f\) is a rational function, it is continuous at every point on its domain where the denominator is non-zero. The denominator \(x^2+y^2\) equals zero only at the origin. Thus, the function is continuous on its domain:
    \begin{equation*} D = \{(x,y) \mid (x,y) \neq (0,0)\} \end{equation*}
    🔗
    🔗
    🔗

    Example 3.22.

    Find the limit of \(g(x,y)\) as \((x,y)\to (0,0)\) if
    \begin{equation*} g(x,y) = \begin{cases} \frac{x^2-y^2}{x^2+y^2} \amp \text{if } (x,y) \neq (0,0) \\ 0 \amp \text{if } (x,y) = (0,0) \end{cases} \end{equation*}
    🔗
    Solution.
    To find the limit as \((x,y) \to (0,0)\text{,}\) we examine the behavior of the function for points near, but not equal to, the origin. Therefore, we evaluate using the first piece of our function definition:
    \begin{equation*} \lim_{(x,y)\to (0,0)} g(x,y) = \lim_{(x,y)\to (0,0)} \frac{x^2-y^2}{x^2+y^2} \end{equation*}
    🔗
    As shown previously, approaching the origin along the \(x\)-axis yields a path limit of \(1\text{,}\) while approaching along the \(y\)-axis yields a path limit of \(-1\text{.}\) Because the limits along these two paths are different, the overall limit does not exist.
    🔗
    Consequently, while the function value is defined at the origin (\(g(0,0) = 0\)), the function \(g\) is discontinuous at \((0,0)\) because the limit fails to exist.
    🔗
    🔗
    🔗

    Example 3.23.

    Is the function
    \begin{equation*} f(x,y) = \begin{cases} \frac{3x^2y}{x^2+y^2} \amp \text{if } (x,y) \neq (0,0) \\ 0 \amp \text{if } (x,y) = (0,0) \end{cases} \end{equation*}
    continuous?
    🔗
    Solution.
    To determine if \(f(x,y)\) is continuous, we check the three criteria for continuity at the point of interest, \((0,0)\text{:}\)
    🔗
    1. The function is defined at the origin: \(f(0,0) = 0\text{.}\)
      🔗
      🔗
    2. Using the Squeeze Theorem (as shown in the earlier checkpoint), the limit of the non-zero piece as we approach the origin is:
      \begin{equation*} \lim_{(x,y)\to (0,0)} \frac{3x^2y}{x^2+y^2} = 0 \end{equation*}
      🔗
      🔗
    3. Since the limiting value equals the defined function value (\(0 = 0\)), we have:
      \begin{equation*} \lim_{(x,y)\to (0,0)} f(x,y) = f(0,0) \end{equation*}
      🔗
      🔗
    🔗
    Because it is continuous at \((0,0)\) and consists of a continuous rational expression at all other points, \(f\) is continuous on all of \(\mathbb{R}^2\text{.}\)

    Example 3.24.

    Show that
    \begin{equation*} \lim_{(x,y,z)\to (1,2,-1)} \frac{(x+y+3z)^3}{(x-1)(y-2)(z+1)} \end{equation*}
    does not exist.
    🔗
    Solution.
    Let \(P(x,y,z)\) approach the point \((1,2,-1)\) along an arbitrary line \(L\) passing through \((1,2,-1)\) with direction vector \(\langle a,b,c \rangle\) where \(a, b, c \neq 0\text{.}\) The parametric equations of this line are given by:
    \begin{equation*} x = 1+at, \quad y = 2+bt, \quad z = -1+ct \quad \text{for } t \in \mathbb{R} \end{equation*}
    🔗
    Substituting these parametric expressions into the function as \(t \to 0\) yields:
    \begin{align*} \frac{(x+y+3z)^3}{(x-1)(y-2)(z+1)} \amp = \frac{\left((1+at) + (2+bt) + 3(-1+ct)\right)^3}{(1+at-1)(2+bt-2)(-1+ct+1)}\\ \amp = \frac{(1 + 2 - 3 + at + bt + 3ct)^3}{(at)(bt)(ct)}\\ \amp = \frac{(at + bt + 3ct)^3}{abc \cdot t^3}\\ \amp = \frac{t^3(a+b+3c)^3}{abc \cdot t^3}\\ \amp = \frac{(a+b+3c)^3}{abc} \quad \text{for } t \neq 0 \end{align*}
    🔗
    Since the value of this expression depends directly on the chosen direction components \(a\text{,}\) \(b\text{,}\) and \(c\text{,}\) approaching along different lines will produce different limiting values. By the two-path rule extended to three dimensions, the given limit does not exist.
    🔗
    🔗

    Theorem 3.25.

    If a function \(f\) of two variables is continuous at \((a,b)\) and a function \(g\) of one variable is continuous at \(f(a,b)\text{,}\) then the composite function \(h(x,y)=g(f(x,y))\) is continuous at \((a,b)\text{.}\)
    🔗
    🔗

    Example 3.26.

    Find \(h(x,y)=g(f(x,y))\) and the set on which \(h\) is continuous for each of the following:
    🔗
    1. \(\displaystyle g(t)=3t^2+t+1 \quad \text{and} \quad f(x,y)=xe^y\)
      🔗
      🔗
    2. \(\displaystyle g(t)=\sin\sqrt{t} \quad \text{and} \quad f(x,y)=y-4x^2\)
      🔗
      🔗
    🔗
    Solution.
    1. Substituting \(f(x,y)\) into \(g(t)\) gives:
      \begin{equation*} h(x,y) = g(f(x,y)) = 3(xe^y)^2 + (xe^y) + 1 = 3x^2e^{2y} + xe^y + 1 \end{equation*}
      Since polynomials and exponential functions are continuous everywhere on their domains, this composite function is continuous at every point on \(\mathbb{R}^2\text{.}\)
      🔗
      🔗
    2. Substituting \(f(x,y)\) into \(g(t)\) gives:
      \begin{equation*} h(x,y) = g(f(x,y)) = \sin\sqrt{y-4x^2} \end{equation*}
      The outer sine function is continuous everywhere, but the square root function is only defined and continuous for non-negative inputs (\(t \geq 0\)). Therefore, the expression inside the radical must be greater than or equal to zero. The composite function \(h\) is continuous on the set:
      \begin{equation*} D = \{(x,y) \mid y \geq 4x^2 \} \end{equation*}
