    Worksheet Worksheet 02: Vector Functions
    

    1.

    State the rules for differentiating and integrating vector functions. Give examples.
    🔗
    Solution.
    Let \(\mathbf{u}(t)\) and \(\mathbf{v}(t)\) be differentiable vector functions, and let \(f(t)\) be a differentiable scalar function. The key derivative rules are:
    🔗
    1. Sum Rule: \(\frac{d}{dt}[\mathbf{u}(t) + \mathbf{v}(t)] = \mathbf{u}'(t) + \mathbf{v}'(t)\)
      🔗
      🔗
    2. Scalar Product Rule: \(\frac{d}{dt}[f(t)\mathbf{u}(t)] = f'(t)\mathbf{u}(t) + f(t)\mathbf{u}'(t)\)
      🔗
      🔗
    3. Dot Product Rule: \(\frac{d}{dt}[\mathbf{u}(t) \cdot \mathbf{v}(t)] = \mathbf{u}'(t) \cdot \mathbf{v}(t) + \mathbf{u}(t) \cdot \mathbf{v}'(t)\)
      🔗
      🔗
    4. Cross Product Rule: \(\frac{d}{dt}[\mathbf{u}(t) \times \mathbf{v}(t)] = \mathbf{u}'(t) \times \mathbf{v}(t) + \mathbf{u}(t) \times \mathbf{v}'(t)\)
      🔗
      🔗
    For integration, if \(\mathbf{r}(t) = \langle f(t), g(t), h(t) \rangle\text{,}\) the definite integral is calculated component-wise:
    \begin{equation*} \int_{a}^{b} \mathbf{r}(t)\, dt = \left\langle \int_{a}^{b} f(t)\, dt, \, \int_{a}^{b} g(t)\, dt, \, \int_{a}^{b} h(t)\, dt \right\rangle \end{equation*}
    🔗
    Example: If \(\mathbf{r}(t) = \langle t^2, e^t, \sin t \rangle\text{,}\) then \(\mathbf{r}'(t) = \langle 2t, e^t, \cos t \rangle\) and \(\int \mathbf{r}(t)\, dt = \langle \frac{t^3}{3}, e^t, -\cos t \rangle + \mathbf{C}\text{.}\)
    🔗
    🔗
    🔗

    2.

    How do you define and calculate the length of a segment of a smooth space curve? Give an example.
    🔗
    Solution.
    The arc length \(L\) of a smooth space curve traced by \(\mathbf{r}(t)\) from \(t = a\) to \(t = b\) is defined as the integral of its scalar speed:
    \begin{equation*} L = \int_{a}^{b} \|\mathbf{r}'(t)\|\, dt = \int_{a}^{b} \sqrt{[x'(t)]^2 + [y'(t)]^2 + [z'(t)]^2}\, dt \end{equation*}
    🔗
    Example: Find the length of \(\mathbf{r}(t) = \langle 3t, \sin t, \cos t \rangle\) from \(t=0\) to \(t=2\text{.}\) The derivative is \(\mathbf{r}'(t) = \langle 3, \cos t, -\sin t \rangle\text{.}\) Its magnitude is \(\|\mathbf{r}'(t)\| = \sqrt{3^2 + \cos^2 t + (-\sin t)^2} = \sqrt{9 + 1} = \sqrt{10}\text{.}\) Integrating over the interval yields:
    \begin{equation*} L = \int_{0}^{2} \sqrt{10}\, dt = 2\sqrt{10} \end{equation*}
    🔗
    🔗
    🔗

    3.

    What is a differentiable curve’s unit tangent vector? Give an example.
    🔗
    Solution.
    The unit tangent vector \(\mathbf{T}(t)\) represents the direction of motion along a smooth curve, normalized to have a constant length of \(1\text{.}\) It is defined as:
    \begin{equation*} \mathbf{T}(t) = \frac{\mathbf{r}'(t)}{\|\mathbf{r}'(t)\|} \end{equation*}
    provided that \(\mathbf{r}'(t) \neq \mathbf{0}\text{.}\)
    🔗
    Example: For \(\mathbf{r}(t) = \langle t, t^2 \rangle\) at \(t=1\text{.}\) The tangent vector is \(\mathbf{r}'(t) = \langle 1, 2t \rangle \implies \mathbf{r}'(1) = \langle 1, 2 \rangle\text{.}\) Its magnitude is \(\|\mathbf{r}'(1)\| = \sqrt{1^2 + 2^2} = \sqrt{5}\text{.}\) Thus, the unit tangent vector is:
    \begin{equation*} \mathbf{T}(1) = \frac{1}{\sqrt{5}}\langle 1, 2 \rangle = \left\langle \frac{1}{\sqrt{5}}, \frac{2}{\sqrt{5}} \right\rangle \end{equation*}
    🔗
    🔗

    4.

    Write \(\mathbf{a}\) in the form \(\mathbf{a}=a_T\mathbf{T}+a_N\mathbf{N}\) without finding \(\mathbf{T}\) and \(\mathbf{N}\text{,}\) and find the value of \(\kappa\) at the given value of \(t\text{:}\)
    \begin{equation*} \mathbf{r}(t)=(4\cos t)\mathbf{i}+(\sqrt{2}\sin t)\mathbf{j}, \qquad t=\frac{\pi}{4} \end{equation*}
    🔗
    Solution.
    We differentiate the position vector function to find velocity \(\mathbf{v}(t)\) and acceleration \(\mathbf{a}(t)\text{:}\)
    \begin{equation*} \mathbf{v}(t) = \langle -4\sin t, \sqrt{2}\cos t \rangle \implies \mathbf{v}\left(\frac{\pi}{4}\right) = \left\langle -4\left(\frac{\sqrt{2}}{2}\right), \sqrt{2}\left(\frac{\sqrt{2}}{2}\right) \right\rangle = \langle -2\sqrt{2}, 1 \rangle \end{equation*}
    \begin{equation*} \mathbf{a}(t) = \langle -4\cos t, -\sqrt{2}\sin t \rangle \implies \mathbf{a}\left(\frac{\pi}{4}\right) = \left\langle -4\left(\frac{\sqrt{2}}{2}\right), -\sqrt{2}\left(\frac{\sqrt{2}}{2}\right) \right\rangle = \langle -2\sqrt{2}, -1 \rangle \end{equation*}
    🔗
    Compute the scalar speed tracking parameters at \(t = \frac{\pi}{4}\text{:}\)
    \begin{equation*} v = \|\mathbf{v}\| = \sqrt{(-2\sqrt{2})^2 + 1^2} = \sqrt{8 + 1} = 3 \end{equation*}
    The tangential component of acceleration \(a_T\) is found using a projection dot product:
    \begin{equation*} a_T = \frac{\mathbf{v} \cdot \mathbf{a}}{\|\mathbf{v}\|} = \frac{(-2\sqrt{2})(-2\sqrt{2}) + (1)(-1)}{3} = \frac{8 - 1}{3} = \frac{7}{3} \end{equation*}
    🔗
    To find the normal acceleration component \(a_N\text{,}\) use the relationship \(\|\mathbf{a}\|^2 = a_T^2 + a_N^2\text{.}\) First, find \(\|\mathbf{a}\|^2 = (-2\sqrt{2})^2 + (-1)^2 = 8 + 1 = 9\text{:}\)
    \begin{equation*} a_N = \sqrt{\|\mathbf{a}\|^2 - a_T^2} = \sqrt{9 - \left(\frac{7}{3}\right)^2} = \sqrt{9 - \frac{49}{9}} = \sqrt{\frac{32}{9}} = \frac{4\sqrt{2}}{3} \end{equation*}
    Thus, the acceleration decomposition is:
    \begin{equation*} \mathbf{a} = \frac{7}{3}\mathbf{T} + \frac{4\sqrt{2}}{3}\mathbf{N} \end{equation*}
    🔗
    Finally, calculate the curvature using the normal acceleration relationship \(a_N = \kappa v^2\text{:}\)
    \begin{equation*} \kappa = \frac{a_N}{v^2} = \frac{(4\sqrt{2})/3}{3^2} = \frac{4\sqrt{2}}{27} \end{equation*}
    🔗
    🔗

    5.

    The position of the particle in the plane at time \(t\) is:
    \begin{equation*} \mathbf{r}=\frac{1}{\sqrt{1+t^2}}\mathbf{i}+\frac{t}{\sqrt{1+t^2}}\mathbf{j} \end{equation*}
    Find the particle’s highest speed.
    🔗
    Solution.
    Notice that the coordinates describe a unit circle since \(x^2 + y^2 = \frac{1}{1+t^2} + \frac{t^2}{1+t^2} = 1\text{.}\) Differentiate each component to evaluate the velocity vector:
    \begin{equation*} x'(t) = -\frac{1}{2}(1+t^2)^{-3/2}(2t) = \frac{-t}{(1+t^2)^{3/2}} \end{equation*}
    \begin{equation*} y'(t) = \frac{1 \cdot \sqrt{1+t^2} - t \cdot \frac{t}{\sqrt{1+t^2}}}{1+t^2} = \frac{(1+t^2) - t^2}{(1+t^2)^{3/2}} = \frac{1}{(1+t^2)^{3/2}} \end{equation*}
    \begin{equation*} \mathbf{v}(t) = \left\langle \frac{-t}{(1+t^2)^{3/2}}, \, \frac{1}{(1+t^2)^{3/2}} \right\rangle \end{equation*}
    🔗
    Compute the scalar speed function \(v(t) = \|\mathbf{v}(t)\|\text{:}\)
    \begin{equation*} v(t) = \sqrt{\frac{t^2}{(1+t^2)^3} + \frac{1}{(1+t^2)^3}} = \sqrt{\frac{t^2+1}{(1+t^2)^3}} = \sqrt{\frac{1}{(1+t^2)^2}} = \frac{1}{1+t^2} \end{equation*}
    🔗
    To find the maximum speed value, look at the denominator of \(v(t) = \frac{1}{1+t^2}\text{.}\) The fraction reaches its largest value when its denominator is minimized, which occurs exactly at \(t=0\text{.}\) Therefore, the particle’s highest speed is:
    \begin{equation*} v_{\max} = v(0) = \frac{1}{1+0^2} = 1 \end{equation*}

    6.

    A shot leaves the thrower’s hand \(6.5\) ft above the ground at a \(45^\circ\) angle at \(44\) ft/sec. Where is it \(2\) sec later?
    🔗
    Solution.
    Using standard ground kinematics parameters for English units, we have an initial position height \(h_0 = 6.5\) ft, launch speed \(v_0 = 44\) ft/sec, elevation angle \(\alpha = 45^\circ\text{,}\) and constant gravitational acceleration value \(g = 32 \, \text{ft/sec}^2\text{.}\) The horizontal and vertical initial velocity components are:
    \begin{equation*} v_{0x} = 44\cos(45^\circ) = 44\left(\frac{\sqrt{2}}{2}\right) = 22\sqrt{2} \, \text{ft/sec} \end{equation*}
    \begin{equation*} v_{0y} = 44\sin(45^\circ) = 44\left(\frac{\sqrt{2}}{2}\right) = 22\sqrt{2} \, \text{ft/sec} \end{equation*}
    🔗
    Set up the general equations for horizontal and vertical position tracking over time \(t\text{:}\)
    \begin{equation*} x(t) = (v_0\cos\alpha)t = 22\sqrt{2}t \end{equation*}
    \begin{equation*} y(t) = h_0 + (v_0\sin\alpha)t - \frac{1}{2}gt^2 = 6.5 + 22\sqrt{2}t - 16t^2 \end{equation*}
    🔗
    Evaluate both position tracking equations at the target parameter time \(t = 2\) seconds:
    \begin{equation*} x(2) = 22\sqrt{2}(2) = 44\sqrt{2} \approx 62.23 \, \text{feet} \end{equation*}
    \begin{equation*} y(2) = 6.5 + 22\sqrt{2}(2) - 16(2)^2 = 6.5 + 44\sqrt{2} - 64 \approx 6.5 + 62.23 - 64 = 4.73 \, \text{feet} \end{equation*}
    🔗
    Two seconds later, the shot is located \(44\sqrt{2}\) feet horizontally downrange from the thrower and at a vertical height of approximately \(4.73\) feet above the ground.

    7.

    Find the length of the curve:
    \begin{equation*} \mathbf{r}(t)=(2\cos t)\mathbf{i}+(2\sin t)\mathbf{j}+t^2\mathbf{k}, \quad 0 \le t \le \frac{\pi}{4} \end{equation*}
    🔗
    Solution.
    First, differentiate the vector function components to track velocity:
    \begin{equation*} \mathbf{r}'(t) = \langle -2\sin t, \, 2\cos t, \, 2t \rangle \end{equation*}
    Next, compute the scalar speed magnitude function using the identity \(\sin^2 t + \cos^2 t = 1\text{:}\)
    \begin{equation*} \|\mathbf{r}'(t)\| = \sqrt{(-2\sin t)^2 + (2\cos t)^2 + (2t)^2} = \sqrt{4\sin^2 t + 4\cos^2 t + 4t^2} = \sqrt{4(1) + 4t^2} = 2\sqrt{1+t^2} \end{equation*}
    🔗
    Set up the arc length integration array from \(a=0\) to \(b=\frac{\pi}{4}\text{:}\)
    \begin{equation*} L = \int_{0}^{\frac{\pi}{4}} 2\sqrt{1+t^2}\, dt \end{equation*}
    Apply the standard trigonometric integration table substitution form \(\int \sqrt{1+t^2}\,dt = \frac{t}{2}\sqrt{1+t^2} + \frac{1}{2}\ln|t + \sqrt{1+t^2}|\text{:}\)
    \begin{equation*} L = 2 \left[ \frac{t}{2}\sqrt{1+t^2} + \frac{1}{2}\ln\left(t + \sqrt{1+t^2}\right) \right]_{0}^{\frac{\pi}{4}} = \left[ t\sqrt{1+t^2} + \ln\left(t + \sqrt{1+t^2}\right) \right]_{0}^{\frac{\pi}{4}} \end{equation*}
    \begin{equation*} L = \frac{\pi}{4}\sqrt{1 + \frac{\pi^2}{16}} + \ln\left(\frac{\pi}{4} + \sqrt{1 + \frac{\pi^2}{16}}\right) - (0 + \ln(1)) \end{equation*}
    \begin{equation*} L = \frac{\pi\sqrt{16+\pi^2}}{16} + \ln\left(\frac{\pi + \sqrt{16+\pi^2}}{4}\right) \approx 0.816 \, \text{units} \end{equation*}
    🔗
    🔗
    🔗

    8.

    Find \(\mathbf{T}\text{,}\) \(\mathbf{N}\text{,}\) and \(\kappa\) at the given \(t\text{:}\)
    \begin{equation*} \mathbf{r}(t)=\frac{4}{9}(1+t)^{3/2}\mathbf{i}+\frac{4}{9}(1-t)^{3/2}\mathbf{j}+\frac{1}{3}t\mathbf{k}, \qquad t=0 \end{equation*}
    🔗
    Solution.
    Compute the derivative vector function to track velocity:
    \begin{equation*} \mathbf{r}'(t) = \left\langle \frac{4}{9} \cdot \frac{3}{2}(1+t)^{1/2}, \, \frac{4}{9} \cdot \frac{3}{2}(1-t)^{1/2}(-1), \, \frac{1}{3} \right\rangle = \left\langle \frac{2}{3}\sqrt{1+t}, \, -\frac{2}{3}\sqrt{1-t}, \, \frac{1}{3} \right\rangle \end{equation*}
    Evaluating velocity at our target parameter point \(t=0\) yields:
    \begin{equation*} \mathbf{r}'(0) = \left\langle \frac{2}{3}(1), \, -\frac{2}{3}(1), \, \frac{1}{3} \right\rangle = \left\langle \frac{2}{3}, \, -\frac{2}{3}, \, \frac{1}{3} \right\rangle \end{equation*}
    Compute the speed magnitude value:
    \begin{equation*} \|\mathbf{r}'(0)\| = \sqrt{\left(\frac{2}{3}\right)^2 + \left(-\frac{2}{3}\right)^2 + \left(\frac{1}{3}\right)^2} = \sqrt{\frac{4}{9} + \frac{4}{9} + \frac{1}{9}} = \sqrt{\frac{9}{9}} = 1 \end{equation*}
    Thus, the unit tangent vector at \(t=0\) matches the velocity vector directly:
    \begin{equation*} \mathbf{T}(0) = \left\langle \frac{2}{3}, \, -\frac{2}{3}, \, \frac{1}{3} \right\rangle \end{equation*}
    🔗
    To find \(\mathbf{N}(0)\text{,}\) look back at the general velocity profile magnitude function:
    \begin{equation*} \|\mathbf{r}'(t)\| = \sqrt{\frac{4}{9}(1+t) + \frac{4}{9}(1-t) + \frac{1}{9}} = \sqrt{\frac{4}{9} + \frac{4}{9}t + \frac{4}{9} - \frac{4}{9}t + \frac{1}{9}} = \sqrt{\frac{9}{9}} = 1 \end{equation*}
    Since speed is identically \(1\) for all values of \(t\text{,}\) the general unit tangent equation reduces simply to \(\mathbf{T}(t) = \mathbf{r}'(t)\text{.}\) Differentiating a second time gives \(\mathbf{T}'(t)\text{:}\)
    \begin{equation*} \mathbf{T}'(t) = \mathbf{r}''(t) = \left\langle \frac{1}{3\sqrt{1+t}}, \, \frac{1}{3\sqrt{1-t}}, \, 0 \right\rangle \end{equation*}
    Evaluate this vector layout at \(t=0\text{:}\)
    \begin{equation*} \mathbf{T}'(0) = \left\langle \frac{1}{3}, \, \frac{1}{3}, \, 0 \right\rangle \implies \|\mathbf{T}'(0)\| = \sqrt{\frac{1}{9} + \frac{1}{9} + 0} = \frac{\sqrt{2}}{3} \end{equation*}
    🔗
    Normalize this derivative vector to find the unit normal vector:
    \begin{equation*} \mathbf{N}(0) = \frac{\mathbf{T}'(0)}{\|\mathbf{T}'(0)\|} = \frac{3}{\sqrt{2}} \left\langle \frac{1}{3}, \, \frac{1}{3}, \, 0 \right\rangle = \left\langle \frac{1}{\sqrt{2}}, \, \frac{1}{\sqrt{2}}, \, 0 \right\rangle \end{equation*}
    Finally, calculate the curvature using the relationship \(\kappa(t) = \frac{\|\mathbf{T}'(t)\|}{\|\mathbf{r}'(t)\|}\text{:}\)
    \begin{equation*} \kappa(0) = \frac{\sqrt{2}/3}{1} = \frac{\sqrt{2}}{3} \end{equation*}
    🔗
    🔗
    🔗

    9.

    Find the parametric equations for the line that is tangent to the curve:
    \begin{equation*} \mathbf{r}(t)=e^t\mathbf{i}+(\sin t)\mathbf{j}+\ln(1-t)\mathbf{k}, \quad t=0 \end{equation*}
    🔗
    Solution.
    First, evaluate the initial point vector coordinates on the curve at \(t=0\text{:}\)
    \begin{equation*} \mathbf{r}(0) = \langle e^0, \, \sin(0), \, \ln(1-0) \rangle = \langle 1, \, 0, \, \ln(1) \rangle = \langle 1, 0, 0 \rangle \end{equation*}
    This gives us the base point coordinates \((x_0, y_0, z_0) = (1, 0, 0)\text{.}\)
    🔗
    Next, differentiate the vector function components with respect to \(t\) using the chain rule to locate the direction vector profile:
    \begin{equation*} \mathbf{r}'(t) = \left\langle e^t, \, \cos t, \, \frac{1}{1-t}(-1) \right\rangle = \left\langle e^t, \, \cos t, \, \frac{1}{t-1} \right\rangle \end{equation*}
    Evaluating the derivative vector at our target parameter point \(t=0\) gives the direction coefficients for the tangent line:
    \begin{equation*} \mathbf{r}'(0) = \left\langle e^0, \, \cos(0), \, \frac{1}{0-1} \right\rangle = \langle 1, \, 1, \, -1 \rangle \end{equation*}
    This gives the parallel direction vector components \(\langle a, b, c \rangle = \langle 1, 1, -1 \rangle\text{.}\)
    🔗
    Using the base point \((1,0,0)\) and direction components \(\langle 1,1,-1 \rangle\text{,}\) the parametric equations for the tangent line are:
    \begin{equation*} x = 1 + t, \qquad y = t, \qquad z = -t \end{equation*}
