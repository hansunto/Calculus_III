    1.

    Sketch the region of integration and evaluate the double integral:
    \begin{equation*} \int^{3/2}_0 \int^{\sqrt{9-4y^2}}_{-\sqrt{9-4y^2}} y \, dx \, dy. \end{equation*}
    🔗
    Solution.
    Step 1: Identify and Sketch the Region of Integration
    🔗
    The given bounds of integration describe a Type II region in Cartesian coordinates where:
    \begin{equation*} 0 \leq y \leq \frac{3}{2} \quad \text{and} \quad -\sqrt{9-4y^2} \leq x \leq \sqrt{9-4y^2}. \end{equation*}
    Squaring the inner variable bounds reveals the equation of the perimeter curve:
    \begin{equation*} x = \pm\sqrt{9-4y^2} \implies x^2 = 9 - 4y^2 \implies x^2 + 4y^2 = 9 \implies \frac{x^2}{9} + \frac{y^2}{9/4} = 1. \end{equation*}
    This is an ellipse centered at the origin with a horizontal semi-axis of \(a=3\) and a vertical semi-axis of \(b=3/2\text{.}\) Because \(y \geq 0\text{,}\) the region \(R\) is the upper half-ellipse.
    🔗
    Step 2: Evaluate the Double Integral
    🔗
    We integrate the innermost expression with respect to \(x\text{,}\) treating \(y\) as a constant factor:
    \begin{align*} \int^{\sqrt{9-4y^2}}_{-\sqrt{9-4y^2}} y \, dx \amp = y \Big[ x \Big]_{-\sqrt{9-4y^2}}^{\sqrt{9-4y^2}} = y \left( \sqrt{9-4y^2} - \left(-\sqrt{9-4y^2}\right) \right)\\ \amp = 2y\sqrt{9-4y^2}. \end{align*}
    Now substitute this into the outer variable layer and integrate with respect to \(y\text{:}\)
    \begin{equation*} \int^{3/2}_0 2y\sqrt{9-4y^2} \, dy. \end{equation*}
    We solve this using a simple \(u\)-substitution. Let:
    \begin{equation*} u = 9 - 4y^2 \implies du = -8y \, dy \implies -\frac{1}{4}\,du = 2y \, dy. \end{equation*}
    Updating limits of integration: when \(y = 0 \implies u = 9\text{,}\) and when \(y = \frac{3}{2} \implies u = 9 - 4\left(\frac{9}{4}\right) = 0\text{.}\)
    \begin{align*} \int_{9}^{0} -\frac{1}{4}u^{1/2} \, du \amp = \frac{1}{4} \int_{0}^{9} u^{1/2} \, du = \frac{1}{4} \left[ \frac{2}{3}u^{3/2} \right]_{0}^{9}\\ \amp = \frac{1}{6} \left( 9^{3/2} - 0 \right) = \frac{1}{6}(27) = \frac{9}{2}. \end{align*}
    The value of the upper half-ellipse integral is exactly \(\frac{9}{2}\text{.}\)

    2.

    Write an equivalent integral with the order of integration reversed. Then evaluate the integral:
    \begin{equation*} \int^4_0 \int^{(y-4)/2}_{-\sqrt{4-y}} \, dx \, dy. \end{equation*}
    🔗
    Solution.
    Step 1: Analyze and Sketch the Region of Integration
    🔗
    The original integral is given in Type II format (\(dxdy\)) with limits:
    \begin{equation*} 0 \leq y \leq 4 \quad \text{and} \quad -\sqrt{4-y} \leq x \leq \frac{y-4}{2}. \end{equation*}
    Let’s rewrite these inner boundary walls by isolating \(y\) in terms of \(x\text{:}\)
    • Left boundary: \(x = -\sqrt{4-y} \implies x^2 = 4-y \implies y = 4 - x^2\) (with \(x \leq 0\)).
      🔗
      🔗
    • Right boundary: \(x = \frac{y-4}{2} \implies 2x = y-4 \implies y = 2x + 4\text{.}\)
      🔗
      🔗
    To locate the intersection vertices, we set these two equations equal to each other:
    \begin{equation*} 4 - x^2 = 2x + 4 \implies x^2 + 2x = 0 \implies x(x+2) = 0 \implies x = -2, \, x = 0. \end{equation*}
    The intersection points are \((-2,0)\) and \((0,4)\text{.}\)
    🔗
    Step 2: Reverse the Order of Integration to Type I (\(dydx\))
    🔗
    Viewing the region vertically, the horizontal span across the \(x\)-axis runs continuously from \(x = -2\) to \(x = 0\text{.}\) For any fixed value of \(x\text{,}\) a vertical strip starts at the bottom straight line boundary \(y = 2x+4\) and sweeps upward to touch the upper parabolic curve boundary \(y = 4-x^2\text{.}\) The equivalent reversed iterated integral setup is:
    \begin{equation*} \int_{-2}^{0} \int_{2x+4}^{4-x^2} \, dy \, dx. \end{equation*}
    🔗
    Step 3: Evaluate the Integral
    🔗
    Evaluating the inner variable layer directly:
    \begin{equation*} \int_{2x+4}^{4-x^2} 1 \, dy = \Big[ y \Big]_{2x+4}^{4-x^2} = (4-x^2) - (2x+4) = -x^2 - 2x. \end{equation*}
    Now, integrate this remaining simple single variable polynomial expression across the outer boundaries:
    \begin{align*} \int_{-2}^{0} (-x^2 - 2x) \, dx \amp = \left[ -\frac{1}{3}x^3 - x^2 \right]_{-2}^{0}\\ \amp = 0 - \left( -\frac{1}{3}(-2)^3 - (-2)^2 \right)\\ \amp = -\left( \frac{8}{3} - 4 \right) = -\left( -\frac{4}{3} \right) = \frac{4}{3}. \end{align*}
    The value of the reversed double integral is exactly \(\frac{4}{3}\text{.}\)
    🔗
    🔗

    3.

    Evaluate the integral:
    \begin{equation*} \int^8_0 \int^2_{\sqrt{x}} \frac{1}{y^4+1} \, dy \, dx. \end{equation*}
    🔗
    Solution.
    Step 1: Analyze the Region and Reverse the Order
    🔗
    The inner integrand function \(\frac{1}{y^4+1}\) does not have an elementary antiderivative with respect to \(y\text{.}\) Therefore, we must switch the integration order from Type I (\(dydx\)) to Type II (\(dxdy\)). The original Cartesian bounds are:
    \begin{equation*} 0 \leq x \leq 8 \quad \text{and} \quad \sqrt{x} \leq y \leq 2. \end{equation*}
    Isolating \(x\) from the lower boundary curve: \(y = \sqrt{x} \implies x = y^3\text{.}\) The new bounds for the reversed integral are:
    \begin{equation*} 0 \leq y \leq 2 \quad \text{and} \quad 0 \leq x \leq y^2. \end{equation*}
    🔗
    Step 2: Set Up and Compute the Transformed Integral
    🔗
    Applying our reversed boundaries creates a solvable iterated integral:
    \begin{equation*} \int_{0}^{2} \int_{0}^{y^3} \frac{1}{y^4+1} \, dx \, dy. \end{equation*}
    Evaluating the inner integral with respect to \(x\text{,}\) treating the \(y\)-term as a constant factor:
    \begin{equation*} \int_{0}^{y^3} \frac{1}{y^4+1} \, dx = \left[ \frac{x}{y^4+1} \right]_{x=0}^{x=y^3} = \frac{y^3}{y^4+1}. \end{equation*}
    Now substitute this result into the outer structural layer to integrate with respect to \(y\text{:}\)
    \begin{equation*} \int_{0}^{2} \frac{y^3}{y^4+1} \, dy. \end{equation*}
    We can solve this easily using \(u\)-substitution. Let:
    \begin{equation*} u = y^4 + 1 \implies du = 4y^3 \, dy \implies \frac{1}{4}\,du = y^3 \, dy. \end{equation*}
    Updating our definite boundaries: when \(y = 0 \implies u = 1\text{,}\) and when \(y = 2 \implies u = 2^4 + 1 = 17\text{.}\)
    \begin{align*} \int_{1}^{17} \frac{1}{4} \cdot \frac{1}{u} \, du \amp = \left[ \frac{1}{4}\ln|u| \right]_{1}^{17}\\ \amp = \frac{1}{4}\ln(17) - \frac{1}{4}\ln(1) = \frac{\ln(17)}{4}. \end{align*}
    The exact evaluation of the reversed integration is \(\frac{\ln(17)}{4}\text{.}\)

    4.

    Find the volume under the paraboloid \(z=x^2+y^2\) above the triangle enclosed by the lines \(y=x\text{,}\) \(x=0\text{,}\) and \(x+y=2\) in the \(xy\)-plane.
    🔗
    Solution.
    Step 1: Identify and Sketch the Base Region \(R\)
    🔗
    The base footprint region \(R\) is a triangle bounded by the vertical \(y\)-axis (\(x=0\)), the diagonal line \(y=x\text{,}\) and the downward-sloping line \(x+y=2 \implies y = 2-x\text{.}\) We locate the upper corner vertex by intersecting the lines:
    \begin{equation*} x = 2-x \implies 2x = 2 \implies x = 1 \implies y = 1. \end{equation*}
    The three vertices of the triangle are \((0,0)\text{,}\) \((0,2)\text{,}\) and \((1,1)\text{.}\) Treating this as a Type I region, the vertical variable \(y\) is bounded below by \(y = x\) and above by \(y = 2-x\) on the horizontal interval \(0 \leq x \leq 1\text{.}\)
    🔗
    Step 2: Set Up and Evaluate the Volume Integral
    🔗
    According to the volume definition theorem, we integrate the ceiling function \(z = x^2 + y^2\) over the domain \(R\text{:}\)
    \begin{equation*} V = \iint_R (x^2+y^2)\,dA = \int_{0}^{1} \int_{x}^{2-x} (x^2+y^2) \, dy \, dx. \end{equation*}
    First, evaluate the inner integral with respect to \(y\text{,}\) holding \(x\) constant:
    \begin{align*} \int_{x}^{2-x} (x^2+y^2) \, dy \amp = \left[ x^2y + \frac{1}{3}y^3 \right]_{y=x}^{y=2-x}\\ \amp = \left( x^2(2-x) + \frac{1}{3}(2-x)^3 \right) - \left( x^2(x) + \frac{1}{3}x^3 \right)\\ \amp = 2x^2 - x^3 + \frac{1}{3}(8 - 12x + 12x^2 - 4x^3) - x^3 - \frac{1}{3}x^3\\ \amp = -\frac{10}{3}x^3 + 6x^2 - 4x + \frac{8}{3}. \end{align*}
    Next, substitute this polynomial into the outer layer and integrate with respect to \(x\text{:}\)
    \begin{align*} V \amp = \int_{0}^{1} \left( -\frac{10}{3}x^3 + 6x^2 - 4x + \frac{8}{3} \right) \, dx\\ \amp = \left[ -\frac{10}{12}x^4 + 2x^3 - 2x^2 + \frac{8}{3}x \right]_{0}^{1}\\ \amp = -\frac{5}{6} + 2 - 2 + \frac{8}{3} = \frac{11}{6}. \end{align*}
    The total volume of the solid region under the paraboloid ceiling is exactly \(\frac{11}{6}\text{.}\)

    5.

    Evaluate the triple integral:
    \begin{equation*} \int^\pi_0 \int^\pi_0 \int^\pi_0 \cos(x+y+z) \, dx \, dy \, dz. \end{equation*}
    🔗
    Solution.
    Step-by-Step Mathematical Evaluation:
    🔗
    Since all six boundaries of integration are constants, this problem evaluates a volume density over a perfect cube in 3D space. We integrate from the innermost variable layer outward.
    🔗
    Step 1: Inner integral with respect to \(x\text{:}\)
    \begin{align*} \int_{0}^{\pi} \cos(x+y+z) \, dx \amp = \Big[ \sin(x+y+z) \Big]_{x=0}^{x=\pi}\\ \amp = \sin(\pi+y+z) - \sin(y+z)\\ \amp = -\sin(y+z) - \sin(y+z) = -2\sin(y+z), \end{align*}
    where we applied the identity \(\sin(\pi+\phi) = -\sin(\phi)\text{.}\)
    🔗
    Step 2: Middle integral with respect to \(y\text{:}\)
    \begin{align*} \int_{0}^{\pi} -2\sin(y+z) \, dy \amp = \Big[ 2\cos(y+z) \Big]_{y=0}^{y=\pi}\\ \amp = 2\cos(\pi+z) - 2\cos(z)\\ \amp = 2(-\cos z) - 2\cos z = -4\cos z, \end{align*}
    where we applied the identity \(\cos(\pi+\phi) = -\cos(\phi)\text{.}\)
    🔗
    Step 3: Outer integral with respect to \(z\text{:}\)
    \begin{align*} \int_{0}^{\pi} -4\cos z \, dz \amp = \Big[ -4\sin z \Big]_{0}^{\pi}\\ \amp = -4\sin(\pi) - (-4\sin 0) = 0. \end{align*}
    The final computed value of the triple cosine integral over the cube is exactly \(0\text{.}\)

    6.

    Given the cylindrical coordinate integral:
    \begin{equation*} \int^{2\pi}_0 \int^{\sqrt{2}}_0 \int^{\sqrt{4-r^2}}_r 3 \, dz \, r \, dr \, d\theta, \qquad r \geq 0 \end{equation*}
    perform the following tasks:
    🔗
    1. Convert to rectangular coordinates with the order of integration \(dz \, dx \, dy\text{.}\)
      🔗
      🔗
    2. Convert to spherical coordinates.
      🔗
      🔗
    3. Evaluate one of the integrals.
      🔗
      🔗
    Solution.
    Step 1: Analyze the Geometry of the Solid Region \(Q\)
    🔗
    Reading the cylindrical bounds from inside out reveals:
    \begin{equation*} 0 \leq \theta \leq 2\pi, \quad 0 \leq r \leq \sqrt{2}, \quad r \leq z \leq \sqrt{4-r^2}. \end{equation*}
    The upper ceiling \(z = \sqrt{4-r^2} \implies x^2+y^2+z^2 = 4\) describes a sphere of radius \(2\text{.}\) The lower floor boundary is the cone \(z = r \implies z = \sqrt{x^2+y^2}\text{.}\) The intersection of the cone and the sphere occurs where \(r^2 = 4-r^2 \implies r = \sqrt{2}\text{.}\) Thus, the solid \(Q\) is an ice-cream-cone-shaped region inside a sphere carved out by an interior cone.
    🔗
    Step 2: Complete the Target Conversions
    🔗
    1. The base shadow of the intersection ring on the \(xy\)-plane is a circle of radius \(\sqrt{2}\text{.}\) In the order \(dz \, dx \, dy\text{,}\) the rectangular formulation is:
      \begin{equation*} \int^{\sqrt{2}}_{-\sqrt{2}} \int^{\sqrt{2-y^2}}_{-\sqrt{2-y^2}} \int^{\sqrt{4-x^2-y^2}}_{\sqrt{x^2+y^2}} 3 \, dz \, dx \, dy. \end{equation*}
      🔗
      🔗
    2. The radius \(\rho\) spans from \(0\) out to the spherical cap \(\rho = 2\text{.}\) The cone \(z = r\) makes an angle of \(\phi = \frac{\pi}{4}\) with the positive vertical \(z\)-axis. Incorporating \(dV = \rho^2\sin\phi \, d\rho d\phi d\theta\text{:}\)
      \begin{equation*} \int^{2\pi}_0 \int^{\pi/4}_0 \int^{2}_0 3\rho^2\sin\phi \, d\rho \, d\phi \, d\theta. \end{equation*}
      🔗
      🔗
    3. Evaluating the spherical integral by separating the independent variables:
      \begin{align*} V \amp = 3 \left( \int^{2\pi}_0 \, d\theta \right) \left( \int^{\pi/4}_0 \sin\phi \, d\phi \right) \left( \int^{2}_0 \rho^2 \, d\rho \right)\\ \amp = 3 \cdot \Big[ \theta \Big]_{0}^{2\pi} \cdot \Big[ -\cos\phi \Big]_{0}^{\pi/4} \cdot \left[ \frac{1}{3}\rho^3 \right]_{0}^{2}\\ \amp = 3 \cdot (2\pi) \cdot \left( 1 - \frac{\sqrt{2}}{2} \right) \cdot \left( \frac{8}{3} \right) = 8\pi(2 - \sqrt{2}). \end{align*}
      🔗
      🔗
    🔗
    🔗

    7.

    Given the Cartesian triple integral:
    \begin{equation*} \int^1_{-1} \int^{\sqrt{1-x^2}}_{-\sqrt{1-x^2}} \int^1_{\sqrt{x^2+y^2}} \, dz \, dy \, dx \end{equation*}
    perform the following tasks:
    🔗
    1. Convert the expression into spherical coordinates.
      🔗
      🔗
    2. Evaluate the new transformed integral.
      🔗
      🔗
    Solution.
    Step 1: Identify Boundaries and Structural Slices
    🔗
    The outer boundaries track a complete unit circle base shadow disk in the \(xy\)-plane with radius \(r=1\text{.}\) The interior vertical boundaries are bounded below by the cone \(z = \sqrt{x^2+y^2} \implies z = r\) and capped above by the flat horizontal plane ceiling \(z = 1\text{.}\)
    🔗
    Mapping these two surfaces to spherical coordinates:
    • The lower boundary cone \(z = r \implies \phi = \frac{\pi}{4}\text{.}\)
      🔗
      🔗
    • The upper boundary plane \(z = 1 \implies \rho\cos\phi = 1 \implies \rho = \sec\phi\text{.}\)
      🔗
      🔗
    Thus, for any angle \(\phi\) between \(0\) and \(\frac{\pi}{4}\text{,}\) the radius variable \(\rho\) shoots out from the origin until it hits the flat ceiling at \(\rho = \sec\phi\text{.}\) The spherical limits are:
    \begin{equation*} 0 \leq \theta \leq 2\pi, \quad 0 \leq \phi \leq \frac{\pi}{4}, \quad 0 \leq \rho \leq \sec\phi. \end{equation*}
    🔗
    Step 2: Complete the Tasks
    🔗
    1. Substituting the spherical differential volume element (\(dzdydx = \rho^2\sin\phi \, d\rho d\phi d\theta\)) yields the formulation:
      \begin{equation*} \int^{2\pi}_0 \int^{\pi/4}_0 \int^{\sec\phi}_0 \rho^2\sin\phi \, d\rho \, d\phi \, d\theta. \end{equation*}
      🔗
      🔗
    2. Evaluating the inner radial integral layer with respect to \(\rho\text{:}\)
      \begin{equation*} \int^{\sec\phi}_0 \rho^2\sin\phi \, d\rho = \sin\phi \left[ \frac{1}{3}\rho^3 \right]_{0}^{\sec\phi} = \frac{1}{3}\sin\phi\sec^3\phi = \frac{1}{3}\frac{\sin\phi}{\cos^3\phi}. \end{equation*}
      Next, substitute this into the middle layer and integrate with respect to \(\phi\) using \(u = \cos\phi \implies du = -\sin\phi \, d\phi\text{:}\)
      \begin{equation*} \int^{\pi/4}_0 \frac{1}{3}\frac{\sin\phi}{\cos^3\phi} \, d\phi = \frac{1}{3} \int_{1/\sqrt{2}}^{1} u^{-3} \, du = \frac{1}{3} \left[ -\frac{1}{2}u^{-2} \right]_{1/\sqrt{2}}^{1} = \frac{1}{6}. \end{equation*}
      Finally, evaluate across the decoupled outer angular revolution layer:
      \begin{equation*} V = \int_{0}^{2\pi} \frac{1}{6} \, d\theta = \frac{1}{6}(2\pi) = \frac{\pi}{3} . \end{equation*}
      The evaluated value of the triple integral is exactly \(\frac{\pi}{3}\text{.}\)
      🔗
      🔗
    🔗

    8.

    Evaluate the double integral:
    \begin{equation*} \int^1_0 \int^{1-x}_0 \sqrt{x+y}(y-2x)^2 \, dy \, dx. \end{equation*}
    🔗
    Solution.
    Step 1: Choose a Transformation and Calculate the Jacobian
    🔗
    The structure of the integrand suggests a change of variables transformation. Let’s define the new tracking variables:
    \begin{equation*} u = x + y \quad \text{and} \quad v = y - 2x. \end{equation*}
    Solving this linear system for \(x\) and \(y\) to find the inverse transformation equations:
    \begin{equation*} x = \frac{1}{3}u - \frac{1}{3}v \quad \text{and} \quad y = \frac{2}{3}u + \frac{1}{3}v. \end{equation*}
    Now compute the Jacobian determinant \(J(u,v)\) of this transformation:
    \begin{equation*} \frac{\partial(x,y)}{\partial(u,v)} = \begin{vmatrix} \frac{1}{3} \amp -\frac{1}{3} \\ \frac{2}{3} \amp \frac{1}{3} \end{vmatrix} = \left(\frac{1}{3}\right)\left(\frac{1}{3}\right) - \left(-\frac{1}{3}\right)\left(\frac{2}{3}\right) = \frac{1}{9} + \frac{2}{9} = \frac{1}{3}. \end{equation*}
    Thus, the area differential element transforms as: \(dydx = \left|\frac{1}{3}\right|\,dudv = \frac{1}{3}\,dudv\text{.}\)
    🔗
    Step 2: Map the Triangular Domain Boundaries
    🔗
    The original region \(R\) is a triangle bounded by \(x=0, y=0\text{,}\) and \(x+y=1\text{.}\) Translating these boundary equations into the \(uv\)-plane:
    • The boundary line \(x+y=1 \implies u = 1\text{.}\)
      🔗
      🔗
    • The line \(x=0 \implies \frac{1}{3}u - \frac{1}{3}v = 0 \implies v = u\text{.}\)
      🔗
      🔗
    • The line \(y=0 \implies \frac{2}{3}u + \frac{1}{3}v = 0 \implies v = -2u\text{.}\)
      🔗
      🔗
    This maps the triangle into a new triangular region \(S\) in the \(uv\)-plane described by \(0 \leq u \leq 1\) and \(-2u \leq v \leq u\text{.}\)
    🔗
    Step 3: Setup and Evaluate the New Integral
    🔗
    Substituting the new tracking terms into the change of variables formula:
    \begin{equation*} V = \int_{0}^{1} \int_{-2u}^{u} \sqrt{u} \cdot v^2 \cdot \left( \frac{1}{3} \, dv \, du \right) = \frac{1}{3} \int_{0}^{1} \sqrt{u} \left( \int_{-2u}^{u} v^2 \, dv \right) du. \end{equation*}
    Evaluating the inner layer with respect to \(v\text{:}\)
    \begin{equation*} \int_{-2u}^{u} v^2 \, dv = \left[ \frac{1}{3}v^3 \right]_{-2u}^{u} = \frac{1}{3}\left( u^3 - (-2u)^3 \right) = \frac{1}{3}(u^3 + 8u^3) = 3u^3. \end{equation*}
    Now substitute this into the outer single layer and evaluate with respect to \(u\text{:}\)
    \begin{align*} V \amp = \frac{1}{3} \int_{0}^{1} u^{1/2} \cdot \Big( 3u^3 \Big) \, du = \int_{0}^{1} u^{7/2} \, du\\ \amp = \left[ \frac{2}{9}u^{9/2} \right]_{0}^{1} = \frac{2}{9}. \end{align*}
    The value of the transformed double integral is exactly \(\frac{2}{9}\text{.}\)
