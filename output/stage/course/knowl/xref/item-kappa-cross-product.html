Item 5.
For a curve in space defined by a 3D vector function \(\mathbf{r}(t)\text{,}\) the curvature can be computed directly without finding the unit tangent vector via the cross product:
\begin{equation*} \kappa = \frac{|\mathbf{r}'(t)\times\mathbf{r}''(t)|}{|\mathbf{r}'(t)|^3} \end{equation*}
