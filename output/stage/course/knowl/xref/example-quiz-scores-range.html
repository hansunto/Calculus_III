Example 4.65.

Using the quiz scores from above, find the range for Section A, Section B, and Section C.
🔗
Solution.
The range is calculated by subtracting the minimum score from the maximum score in each data set:
🔗
  • For Section A, the maximum value is \(5\) and the minimum value is \(5\text{.}\) The range is \(5 - 5 = 0\text{.}\)
    🔗
    🔗
  • For Section B, the maximum value is \(10\) and the minimum value is \(0\text{.}\) The range is \(10 - 0 = 10\text{.}\)
    🔗
    🔗
  • For Section C, the maximum value is \(6\) and the minimum value is \(4\text{.}\) The range is \(6 - 4 = 2\text{.}\)
