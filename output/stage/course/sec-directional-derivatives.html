    Section 3.6 Directional Derivatives

    In this section, we will explore the concept of directional derivatives, which generalize the idea of partial derivatives to arbitrary directions in space. Directional derivatives provide a way to measure how a function changes as we move in a specific direction from a given point.
    🔗

    Definition 3.68.

    Let \(\mathbf{u} = u_1\mathbf{i} + u_2\mathbf{j}\) be a unit vector. The directional derivative of \(f\) at \(P(x,y)\) in the direction of \(\mathbf{u}\), denoted by \(D_{\mathbf{u}} f(x,y)\text{,}\) is defined by the limit:
    \begin{equation*} D_{\mathbf{u}} f(x,y) = \lim_{s \to 0}\frac{f(x+su_1,y+su_2)-f(x,y)}{s} \end{equation*}
    🔗
    🔗

    Theorem 3.69. Directional Derivative Computation Formula.

    If \(f\) is a differentiable function of two variables and \(\mathbf{u} = u_1\mathbf{i} + u_2\mathbf{j}\) is a unit vector, then the directional derivative can be computed as:
    \begin{equation*} D_{\mathbf{u}} f(x,y) = f_x(x,y)u_1 + f_y(x,y)u_2 \end{equation*}
    🔗
    🔗

    Example 3.70.

    Find the directional derivative \(D_{\mathbf{u}} f(x,y)\) if
    \begin{equation*} f(x,y)=x^3-3xy+4y^2 \end{equation*}
    and \(\mathbf{u}\) is the unit vector given by the angle \(\theta=\pi/6\text{.}\) What is the value of \(D_{\mathbf{u}} f(1,2)\text{?}\)
    🔗
    Solution.
    First, we find the horizontal and vertical components of our directional unit vector \(\mathbf{u}\) using the given angle \(\theta = \pi/6\text{:}\)
    \begin{equation*} u_1 = \cos\left(\frac{\pi}{6}\right) = \frac{\sqrt{3}}{2} \quad \text{and} \quad u_2 = \sin\left(\frac{\pi}{6}\right) = \frac{1}{2} \end{equation*}
    This gives the unit direction vector \(\mathbf{u} = \frac{\sqrt{3}}{2}\mathbf{i} + \frac{1}{2}\mathbf{j}\text{.}\)
    🔗
    Next, we calculate the general first-order partial derivatives of the function \(f(x,y)\text{:}\)
    \begin{equation*} f_x(x,y) = \frac{\partial}{\partial x}(x^3-3xy+4y^2) = 3x^2 - 3y \end{equation*}
    \begin{equation*} f_y(x,y) = \frac{\partial}{\partial y}(x^3-3xy+4y^2) = -3x + 8y \end{equation*}
    🔗
    Applying the Theorem for Directional Derivatives, we substitute these components to find the general formula for \(D_{\mathbf{u}} f(x,y)\text{:}\)
    \begin{align*} D_{\mathbf{u}} f(x,y) \amp = f_x(x,y)u_1 + f_y(x,y)u_2\\ \amp = (3x^2 - 3y)\left(\frac{\sqrt{3}}{2}\right) + (-3x + 8y)\left(\frac{1}{2}\right)\\ \amp = \frac{\sqrt{3}}{2}(3x^2 - 3y) - \frac{3}{2}x + 4y \end{align*}
    🔗
    To find \(D_{\mathbf{u}} f(1,2)\text{,}\) we evaluate our partial derivatives at the specific point \((x,y) = (1,2)\text{:}\)
    \begin{equation*} f_x(1,2) = 3(1)^2 - 3(2) = 3 - 6 = -3 \end{equation*}
    \begin{equation*} f_y(1,2) = -3(1) + 8(2) = -3 + 16 = 13 \end{equation*}
    Substituting these numerical values back into our formula yields:
    \begin{align*} D_{\mathbf{u}} f(1,2) \amp = (-3)\left(\frac{\sqrt{3}}{2}\right) + (13)\left(\frac{1}{2}\right)\\ \amp = \frac{13 - 3\sqrt{3}}{2} \approx 3.9019 \end{align*}
    🔗
    🔗
    🔗
    By applying the properties of vector geometry, the directional derivative formula can be elegantly rewritten as the dot product of two distinct vector paths:
    \begin{align*} D_{\mathbf{u}} f(x,y) \amp = f_x(x,y)u_1 + f_y(x,y)u_2\\ \amp = \langle f_x(x,y), f_y(x,y) \rangle \cdot \langle u_1, u_2 \rangle\\ \amp = \langle f_x(x,y), f_y(x,y) \rangle \cdot \mathbf{u} \end{align*}
    🔗

    Definition 3.71. The Gradient Vector.

    Let \(f\) be a function of two variables. The gradient of \(f\) is the vector function denoted by \(\nabla f\) and defined by:
    \begin{equation*} \nabla f(x,y) = \langle f_x(x,y), f_y(x,y) \rangle = f_x(x,y)\mathbf{i} + f_y(x,y)\mathbf{j} \end{equation*}
    🔗
    🔗
    With this notation for the gradient vector, we can rewrite the expression for the directional derivative as a clean vector dot product:
    \begin{equation*} D_{\mathbf{u}} f(x,y) = \nabla f(x,y) \cdot \mathbf{u} \end{equation*}
    🔗

    Example 3.72.

    If \(f(x,y)=\sin(x)+e^{xy}\text{,}\) find the gradient vector \(\nabla f(x,y)\text{.}\)
    🔗
    Solution.
    To find the gradient vector, we first compute the first-order partial derivatives of \(f(x,y)\) with respect to \(x\) and \(y\text{:}\)
    🔗
    For the \(x\)-derivative, we treat \(y\) as a constant and apply the single-variable chain rule to the exponential term:
    \begin{equation*} f_x(x,y) = \frac{\partial}{\partial x}\left[\sin(x) + e^{xy}\right] = \cos(x) + y e^{xy} \end{equation*}
    🔗
    For the \(y\)-derivative, we treat \(x\) as a constant:
    \begin{equation*} f_y(x,y) = \frac{\partial}{\partial y}\left[\sin(x) + e^{xy}\right] = 0 + x e^{xy} = x e^{xy} \end{equation*}
    🔗
    Now, we assemble these component functions into the final gradient vector expression using both component-bracket and standard basis vector notations:
    \begin{equation*} \nabla f(x,y) = \langle \cos(x) + y e^{xy}, \, x e^{xy} \rangle = \left(\cos(x) + y e^{xy}\right)\mathbf{i} + \left(x e^{xy}\right)\mathbf{j} \end{equation*}
    🔗
    🔗
    🔗

    Checkpoint 3.73.

    Find the directional derivative of the function \(f(x,y)=x^2y^3-4y\) at the point \((2,-1)\) in the direction of the vector \(\mathbf{v}=2\mathbf{i}+5\mathbf{j}\text{.}\)
    🔗
    Solution.
    To compute the directional derivative, the direction vector must be a unit vector. We first find the magnitude of the given vector \(\mathbf{v} = \langle 2, 5 \rangle\text{:}\)
    \begin{equation*} \|\mathbf{v}\| = \sqrt{2^2 + 5^2} = \sqrt{4 + 25} = \sqrt{29} \end{equation*}
    Normalizing \(\mathbf{v}\) gives the required direction unit vector \(\mathbf{u}\text{:}\)
    \begin{equation*} \mathbf{u} = \frac{\mathbf{v}}{\|\mathbf{v}\|} = \left\langle \frac{2}{\sqrt{29}}, \, \frac{5}{\sqrt{29}} \right\rangle \end{equation*}
    🔗
    Next, we calculate the general first-order partial derivatives of \(f(x,y)\text{:}\)
    \begin{equation*} f_x(x,y) = \frac{\partial}{\partial x}(x^2y^3-4y) = 2xy^3 \end{equation*}
    \begin{equation*} f_y(x,y) = \frac{\partial}{\partial y}(x^2y^3-4y) = 3x^2y^2 - 4 \end{equation*}
    🔗
    Evaluating these partial derivatives at the given point \((2,-1)\) yields the components of the gradient vector \(\nabla f(2,-1)\text{:}\)
    \begin{equation*} f_x(2,-1) = 2(2)(-1)^3 = -4 \end{equation*}
    \begin{equation*} f_y(2,-1) = 3(2)^2(-1)^2 - 4 = 12(1) - 4 = 8 \end{equation*}
    \begin{equation*} \nabla f(2,-1) = \langle -4, 8 \rangle \end{equation*}
    🔗
    Finally, we compute the directional derivative by taking the dot product of the gradient vector and our direction unit vector:
    \begin{align*} D_{\mathbf{u}} f(2,-1) \amp = \nabla f(2,-1) \cdot \mathbf{u}\\ \amp = \langle -4, 8 \rangle \cdot \left\langle \frac{2}{\sqrt{29}}, \, \frac{5}{\sqrt{29}} \right\rangle\\ \amp = \frac{-4(2)}{\sqrt{29}} + \frac{8(5)}{\sqrt{29}}\\ \amp = \frac{-8 + 40}{\sqrt{29}} = \frac{32}{\sqrt{29}} \approx 5.9423 \end{align*}
    🔗
    🔗
    🔗

    Checkpoint 3.74.

    Let \(f(x,y)=x^2-4xy\text{.}\) Use the gradient to find the directional derivative of \(f\) at \(P(1,2)\) in the direction from \(P(1,2)\) to \(Q(2,5)\text{.}\)
    🔗
    Solution.
    First, we find the displacement vector \(\mathbf{v}\) pointing from the initial point \(P\) to the destination point \(Q\) by subtracting their respective coordinates:
    \begin{equation*} \mathbf{v} = \overrightarrow{PQ} = \langle 2-1, \, 5-2 \rangle = \langle 1, 3 \rangle \end{equation*}
    🔗
    We find the magnitude of \(\mathbf{v}\) and normalize it to obtain our unit direction vector \(\mathbf{u}\text{:}\)
    \begin{equation*} \|\mathbf{v}\| = \sqrt{1^2 + 3^2} = \sqrt{1 + 9} = \sqrt{10} \end{equation*}
    \begin{equation*} \mathbf{u} = \frac{\mathbf{v}}{\|\mathbf{v}\|} = \left\langle \frac{1}{\sqrt{10}}, \, \frac{3}{\sqrt{10}} \right\rangle \end{equation*}
    🔗
    Next, we calculate the general gradient vector \(\nabla f(x,y)\) by computing the first-order partial derivatives of \(f(x,y) = x^2-4xy\text{:}\)
    \begin{equation*} f_x(x,y) = 2x - 4y \quad \text{and} \quad f_y(x,y) = -4x \end{equation*}
    \begin{equation*} \nabla f(x,y) = \langle 2x - 4y, \, -4x \rangle \end{equation*}
    🔗
    Evaluating the gradient vector specifically at the point \(P(1,2)\) yields:
    \begin{equation*} \nabla f(1,2) = \langle 2(1) - 4(2), \, -4(1) \rangle = \langle 2 - 8, \, -4 \rangle = \langle -6, -4 \rangle \end{equation*}
    🔗
    Finally, we compute the directional derivative by evaluating the dot product \(\nabla f(1,2) \cdot \mathbf{u}\text{:}\)
    \begin{align*} D_{\mathbf{u}} f(1,2) \amp = \langle -6, -4 \rangle \cdot \left\langle \frac{1}{\sqrt{10}}, \, \frac{3}{\sqrt{10}} \right\rangle\\ \amp = \frac{-6(1)}{\sqrt{10}} + \frac{-4(3)}{\sqrt{10}}\\ \amp = \frac{-6 - 12}{\sqrt{10}} = \frac{-18}{\sqrt{10}} = -\frac{9\sqrt{10}}{5} \approx -5.6921 \end{align*}
    🔗
    🔗
    🔗

    Theorem 3.75. Gradient Theorem.

    Let \(f\) be a function of two variables that is differentiable at the point \(P(x,y)\text{.}\)
    1. The maximum value of the directional derivative \(D_{\mathbf{u}} f(x,y)\) at \(P(x,y)\) is \(\|\nabla f(x,y)\|\text{.}\)
      🔗
      🔗
    2. The maximum rate of increase of \(f(x,y)\) at \(P(x,y)\) occurs in the exact direction of the gradient vector \(\nabla f(x,y)\text{.}\)
      🔗
      🔗
    🔗
    🔗

    Corollary 3.76. Minimization via the Gradient Vector.

    Let \(f\) be a function of two variables that is differentiable at the point \(P(x,y)\text{.}\)
    1. The minimum value of the directional derivative \(D_{\mathbf{u}} f(x,y)\) at \(P(x,y)\) is \(-\|\nabla f(x,y)\|\text{.}\)
      🔗
      🔗
    2. The maximum rate of decrease of \(f(x,y)\) at \(P(x,y)\) occurs in the opposite direction of the gradient vector, \(-\nabla f(x,y)\text{.}\)
      🔗
      🔗
    🔗
    🔗

    Example 3.77.

    Let \(f(x,y)=2+x^2+\frac{1}{4}y^2\text{.}\) Find the direction in which \(f(x,y)\) increases most rapidly at the point \(P(1,2)\text{,}\) and find the maximum rate of increase of \(f\) at \(P\text{.}\)
    🔗
    Solution.
    According to the Gradient Theorem, a differentiable function increases most rapidly in the direction of its gradient vector, and the maximum rate of change is equal to the magnitude of that gradient.
    🔗
    First, we find the general first-order partial derivatives of \(f(x,y) = 2+x^2+\frac{1}{4}y^2\text{:}\)
    \begin{equation*} f_x(x,y) = \frac{\partial}{\partial x}\left(2+x^2+\frac{1}{4}y^2\right) = 2x \end{equation*}
    \begin{equation*} f_y(x,y) = \frac{\partial}{\partial y}\left(2+x^2+\frac{1}{4}y^2\right) = \frac{1}{2}y \end{equation*}
    This gives our general gradient vector: \(\nabla f(x,y) = \langle 2x, \, \frac{1}{2}y \rangle\text{.}\)
    🔗
    Next, we evaluate the gradient vector specifically at the point \(P(1,2)\text{:}\)
    \begin{equation*} \nabla f(1,2) = \left\langle 2(1), \, \frac{1}{2}(2) \right\rangle = \langle 2, 1 \rangle \end{equation*}
    Therefore, the direction in which \(f\) increases most rapidly at \(P(1,2)\) is given by the vector \(\langle 2, 1 \rangle\) (or normalized as the unit direction vector \(\mathbf{u} = \langle \frac{2}{\sqrt{5}}, \frac{1}{\sqrt{5}} \rangle\)).
    🔗
    Finally, we calculate the maximum rate of increase by taking the norm of this gradient vector:
    \begin{equation*} \|\nabla f(1,2)\| = \sqrt{2^2 + 1^2} = \sqrt{4 + 1} = \sqrt{5} \approx 2.2361 \end{equation*}
    🔗
    Thus, the maximum rate of increase of the function at \(P(1,2)\) is exactly \(\sqrt{5}\text{.}\)
    🔗
    🔗
    🔗

    Checkpoint 3.78.

    If \(f(x,y)=xe^{xy}\text{,}\) find the rate of change of \(f\) at the point \(P(2,0)\) in the direction from \(P(2,0)\) to \(Q(1/2,2)\text{.}\)
    🔗
    Solution.
    The rate of change in a specific direction is given by the directional derivative. First, we find the displacement vector \(\mathbf{v}\) pointing from \(P\) to \(Q\) by subtracting their coordinates:
    \begin{equation*} \mathbf{v} = \overrightarrow{PQ} = \left\langle \frac{1}{2}-2, \, 2-0 \right\rangle = \left\langle -\frac{3}{2}, \, 2 \right\rangle \end{equation*}
    🔗
    To compute a directional derivative, we must normalize \(\mathbf{v}\) into a unit vector \(\mathbf{u}\text{.}\) We find its magnitude:
    \begin{equation*} \|\mathbf{v}\| = \sqrt{\left(-\frac{3}{2}\right)^2 + 2^2} = \sqrt{\frac{9}{4} + 4} = \sqrt{\frac{25}{4}} = \frac{5}{2} \end{equation*}
    Dividing \(\mathbf{v}\) by its magnitude gives the direction unit vector \(\mathbf{u}\text{:}\)
    \begin{equation*} \mathbf{u} = \frac{\mathbf{v}}{\|\mathbf{v}\|} = \frac{2}{5} \left\langle -\frac{3}{2}, \, 2 \right\rangle = \left\langle -\frac{3}{5}, \, \frac{4}{5} \right\rangle \end{equation*}
    🔗
    Next, we calculate the general first-order partial derivatives of \(f(x,y) = xe^{xy}\) using the single-variable product and chain rules:
    \begin{equation*} f_x(x,y) = 1 \cdot e^{xy} + x \cdot (ye^{xy}) = e^{xy}(1 + xy) \end{equation*}
    \begin{equation*} f_y(x,y) = x \cdot (xe^{xy}) = x^2e^{xy} \end{equation*}
    🔗
    Evaluating these partial derivatives at the baseline point \(P(2,0)\) yields the components of the gradient vector \(\nabla f(2,0)\text{:}\)
    \begin{equation*} f_x(2,0) = e^{2(0)}(1 + 2(0)) = 1(1) = 1 \end{equation*}
    \begin{equation*} f_y(2,0) = (2)^2 e^{2(0)} = 4(1) = 4 \end{equation*}
    \begin{equation*} \nabla f(2,0) = \langle 1, 4 \rangle \end{equation*}
    🔗
    Finally, we compute the directional derivative by evaluating the dot product of the gradient vector and our direction unit vector:
    \begin{align*} D_{\mathbf{u}} f(2,0) \amp = \nabla f(2,0) \cdot \mathbf{u}\\ \amp = \langle 1, 4 \rangle \cdot \left\langle -\frac{3}{5}, \, \frac{4}{5} \right\rangle\\ \amp = 1\left(-\frac{3}{5}\right) + 4\left(\frac{4}{5}\right)\\ \amp = -\frac{3}{5} + \frac{16}{5} = \frac{13}{5} = 2.6 \end{align*}
    🔗
    🔗
    🔗

    Checkpoint 3.79.

    For the function \(f(x,y)=xe^{xy}\) at the point \(P(2,0)\text{,}\) in what direction does \(f\) have the maximum rate of change? What is this maximum rate of change?
    🔗
    Solution.
    According to the Gradient Theorem, a differentiable function increases most rapidly in the exact direction of its gradient vector, and the maximum rate of change is equal to the magnitude of that vector.
    🔗
    From our calculations in the previous exercise, the gradient vector evaluated at the point \(P(2,0)\) is:
    \begin{equation*} \nabla f(2,0) = \langle 1, 4 \rangle \end{equation*}
    🔗
    Therefore, \(f\) has its maximum rate of change in the direction of the vector \(\langle 1, 4 \rangle\text{.}\) Expressed as a normalized unit vector, this direction is:
    \begin{equation*} \mathbf{u}_{\text{max}} = \frac{\nabla f(2,0)}{\|\nabla f(2,0)\|} = \left\langle \frac{1}{\sqrt{17}}, \, \frac{4}{\sqrt{17}} \right\rangle \end{equation*}
    🔗
    The maximum rate of change is the magnitude of this gradient vector:
    \begin{equation*} \|\nabla f(2,0)\| = \sqrt{1^2 + 4^2} = \sqrt{1 + 16} = \sqrt{17} \approx 4.1231 \end{equation*}
    🔗
    🔗
    🔗

    Definition 3.80. Directional Derivative with Three Variables.

    Let \(\mathbf{u} = u_1\mathbf{i} + u_2\mathbf{j} + u_3\mathbf{k}\) be a 3D unit vector. The directional derivative of \(f\) at \(P(x,y,z)\) in the direction of \(\mathbf{u}\), denoted by \(D_{\mathbf{u}} f(x,y,z)\text{,}\) is defined by the limit:
    \begin{equation*} D_{\mathbf{u}} f(x,y,z) = \lim_{s \to 0}\frac{f(x+su_1,y+su_2,z+su_3)-f(x,y,z)}{s} \end{equation*}
    🔗
    🔗

    Theorem 3.81. Directional Derivative Formula for Three Variables.

    If \(f\) is a differentiable function of three variables and \(\mathbf{u} = u_1\mathbf{i} + u_2\mathbf{j} + u_3\mathbf{k}\) is a unit vector, then the directional derivative can be computed as:
    \begin{equation*} D_{\mathbf{u}} f(x,y,z) = f_x(x,y,z)u_1 + f_y(x,y,z)u_2 + f_z(x,y,z)u_3 \end{equation*}
    🔗
    🔗
    The gradient of a function of three variables is the vector function denoted by \(\nabla f\) and defined by:
    \begin{equation*} \nabla f(x,y,z) = \langle f_x(x,y,z), f_y(x,y,z), f_z(x,y,z) \rangle = f_x(x,y,z)\mathbf{i} + f_y(x,y,z)\mathbf{j} + f_z(x,y,z)\mathbf{k} \end{equation*}
    With this vector framework, the directional derivative can be written compactly as the dot product \(D_{\mathbf{u}} f(x,y,z) = \nabla f(x,y,z) \cdot \mathbf{u}\text{.}\)
    🔗

    Example 3.82.

    If \(f(x,y,z)=x \sin(yz)\text{,}\) find the gradient vector \(\nabla f(x,y,z)\text{.}\) Then, find the directional derivative of \(f\) at the point \((1,3,0)\) in the direction of the vector \(\mathbf{v}=\mathbf{i}+2\mathbf{j}-\mathbf{k}\text{.}\)
    🔗
    Solution.
    First, we compute the three first-order partial derivatives of \(f(x,y,z) = x \sin(yz)\) by treating the remaining variables as constants and applying single-variable chain rules:
    \begin{equation*} f_x(x,y,z) = \frac{\partial}{\partial x}[x \sin(yz)] = \sin(yz) \end{equation*}
    \begin{equation*} f_y(x,y,z) = \frac{\partial}{\partial y}[x \sin(yz)] = x \cdot \cos(yz) \cdot z = xz\cos(yz) \end{equation*}
    \begin{equation*} f_z(x,y,z) = \frac{\partial}{\partial z}[x \sin(yz)] = x \cdot \cos(yz) \cdot y = xy\cos(yz) \end{equation*}
    🔗
    Assembling these components gives the general gradient vector formula:
    \begin{equation*} \nabla f(x,y,z) = \langle \sin(yz), \, xz\cos(yz), \, xy\cos(yz) \rangle \end{equation*}
    🔗
    Next, we evaluate the gradient vector specifically at the given point \((1,3,0)\text{:}\)
    \begin{equation*} f_x(1,3,0) = \sin(3 \cdot 0) = \sin(0) = 0 \end{equation*}
    \begin{equation*} f_y(1,3,0) = (1)(0)\cos(3 \cdot 0) = 0 \end{equation*}
    \begin{equation*} f_z(1,3,0) = (1)(3)\cos(3 \cdot 0) = 3\cos(0) = 3(1) = 3 \end{equation*}
    This yields the evaluated vector: \(\nabla f(1,3,0) = \langle 0, 0, 3 \rangle\text{.}\)
    🔗
    The direction vector \(\mathbf{v} = \langle 1, 2, -1 \rangle\) is not a unit vector. We must normalize it before computing the directional derivative. We calculate its magnitude:
    \begin{equation*} \|\mathbf{v}\| = \sqrt{1^2 + 2^2 + (-1)^2} = \sqrt{1 + 4 + 1} = \sqrt{6} \end{equation*}
    Dividing \(\mathbf{v}\) by its magnitude gives the direction unit vector \(\mathbf{u}\text{:}\)
    \begin{equation*} \mathbf{u} = \frac{\mathbf{v}}{\|\mathbf{v}\|} = \left\langle \frac{1}{\sqrt{6}}, \, \frac{2}{\sqrt{6}}, \, -\frac{1}{\sqrt{6}} \right\rangle \end{equation*}
    🔗
    Finally, we calculate the directional derivative by taking the dot product of the evaluated gradient vector and our direction unit vector:
    \begin{align*} D_{\mathbf{u}} f(1,3,0) \amp = \nabla f(1,3,0) \cdot \mathbf{u}\\ \amp = \langle 0, 0, 3 \rangle \cdot \left\langle \frac{1}{\sqrt{6}}, \, \frac{2}{\sqrt{6}}, \, -\frac{1}{\sqrt{6}} \right\rangle\\ \amp = 0\left(\frac{1}{\sqrt{6}}\right) + 0\left(\frac{2}{\sqrt{6}}\right) + 3\left(-\frac{1}{\sqrt{6}}\right)\\ \amp = -\frac{3}{\sqrt{6}} = -\frac{\sqrt{6}}{2} \approx -1.2247 \end{align*}
    🔗
    🔗

    Checkpoint 3.83.

    Suppose that the temperature at a point \((x,y,z)\) in space is given by:
    \begin{equation*} T(x,y,z)=\frac{80}{1+x^2+2y^2+3z^2} \end{equation*}
    where \(T\) is measured in degrees Celsius and \(x, y, z\) are measured in meters. In which direction does the temperature increase fastest at the point \((1,1,-2)\text{?}\) What is the maximum rate of increase?
    🔗
    Solution.
    According to the Gradient Theorem, a differentiable function increases most rapidly in the exact direction of its gradient vector, and the maximum rate of change is equal to the magnitude of that gradient vector.
    🔗
    We rewrite the temperature function using a negative exponent to simplify differentiation: \(T(x,y,z) = 80(1+x^2+2y^2+3z^2)^{-1}\text{.}\) Next, we calculate the three first-order partial derivatives of \(T\) using the single-variable chain rule:
    \begin{equation*} T_x = -80(1+x^2+2y^2+3z^2)^{-2}(2x) = \frac{-160x}{(1+x^2+2y^2+3z^2)^2} \end{equation*}
    \begin{equation*} T_y = -80(1+x^2+2y^2+3z^2)^{-2}(4y) = \frac{-320y}{(1+x^2+2y^2+3z^2)^2} \end{equation*}
    \begin{equation*} T_z = -80(1+x^2+2y^2+3z^2)^{-2}(6z) = \frac{-480z}{(1+x^2+2y^2+3z^2)^2} \end{equation*}
    🔗
    Before evaluating the partials at the point \((1,1,-2)\text{,}\) we find the value of the shared denominator expression:
    \begin{equation*} 1 + x^2 + 2y^2 + 3z^2 = 1 + (1)^2 + 2(1)^2 + 3(-2)^2 = 1 + 1 + 2 + 12 = 16 \end{equation*}
    Substituting \(x=1\text{,}\) \(y=1\text{,}\) and \(z=-2\) into our partial derivatives yields:
    \begin{equation*} T_x(1,1,-2) = \frac{-160(1)}{(16)^2} = \frac{-160}{256} = -\frac{5}{8} \end{equation*}
    \begin{equation*} T_y(1,1,-2) = \frac{-320(1)}{(16)^2} = \frac{-320}{256} = -\frac{5}{4} \end{equation*}
    \begin{equation*} T_z(1,1,-2) = \frac{-480(-2)}{(16)^2} = \frac{960}{256} = \frac{15}{4} \end{equation*}
    🔗
    This gives our evaluated gradient vector:
    \begin{equation*} \nabla T(1,1,-2) = \left\langle -\frac{5}{8}, \, -\frac{5}{4}, \, \frac{15}{4} \right\rangle = \frac{5}{8}\langle -1, \, -2, \, 6 \rangle \end{equation*}
    Therefore, the temperature increases fastest in the direction of the vector \(\langle -1, -2, 6 \rangle\) (or its corresponding normalized direction unit vector \(\mathbf{u} = \frac{1}{\sqrt{41}}\langle -1, -2, 6 \rangle\)).
    🔗
    The maximum rate of increase is the magnitude of this gradient vector:
    \begin{align*} \|\nabla T(1,1,-2)\| \amp = \sqrt{\left(-\frac{5}{8}\right)^2 + \left(-\frac{5}{4}\right)^2 + \left(\frac{15}{4}\right)^2}\\ \amp = \sqrt{\frac{25}{64} + \frac{25}{16} + \frac{225}{16}} = \sqrt{\frac{25 + 100 + 900}{64}}\\ \amp = \sqrt{\frac{1025}{64}} = \frac{5\sqrt{41}}{8} \approx 4.0019 \text{ }^{\circ}\text{C/m} \end{align*}
    Thus, the maximum rate of temperature increase at the point \((1,1,-2)\) is exactly \(\frac{5\sqrt{41}}{8} \text{ }^{\circ}\text{C/m}\text{.}\)

    Checkpoint 3.84.

    A simple electrical circuit consists of a resistor \(R\) and an electromotive force \(V\text{.}\) At a certain instant, \(V\) is \(80\text{ volts}\) and is increasing at a rate of \(5\text{ volts/min}\text{,}\) while \(R\) is \(40\text{ ohms}\) and is decreasing at a rate of \(2\text{ ohms/min}\text{.}\) Use Ohm’s law, \(I=\frac{V}{R}\text{,}\) to find the rate at which the current \(I\) is changing.
    🔗
    Solution.
    Ohm’s law defines current \(I\) as a function of two variables—voltage \(V\) and resistance \(R\text{:}\) \(I(V,R) = \frac{V}{R}\text{.}\) Since both \(V\) and \(R\) change over time, they implicitly depend on a common time parameter \(t\text{.}\) By applying the Case I Multivariate Chain Rule, the rate of change of the current with respect to time is:
    \begin{equation*} \frac{dI}{dt} = \frac{\partial I}{\partial V}\frac{dV}{dt} + \frac{\partial I}{\partial R}\frac{dR}{dt} \end{equation*}
    🔗
    First, we compute the first-order partial derivatives of \(I\) with respect to \(V\) and \(R\text{:}\)
    \begin{equation*} \frac{\partial I}{\partial V} = \frac{\partial}{\partial V}\left[\frac{V}{R}\right] = \frac{1}{R} \quad \text{and} \quad \frac{\partial I}{\partial R} = \frac{\partial}{\partial R}\left[V R^{-1}\right] = -\frac{V}{R^2} \end{equation*}
    🔗
    From the statement of the problem, we isolate our baseline parameters and independent rate shifts at this specific instant (noting that a decreasing rate requires a negative value):
    \begin{equation*} V = 80, \quad R = 40, \quad \frac{dV}{dt} = 5, \quad \frac{dR}{dt} = -2 \end{equation*}
    🔗
    Substituting these numerical values into our expanded chain rule expression yields:
    \begin{align*} \frac{dI}{dt} \amp = \left(\frac{1}{R}\right)\frac{dV}{dt} + \left(-\frac{V}{R^2}\right)\frac{dR}{dt}\\ \amp = \left(\frac{1}{40}\right)(5) + \left(-\frac{80}{40^2}\right)(-2)\\ \amp = \frac{5}{40} + \left(-\frac{80}{1600}\right)(-2)\\ \amp = \frac{1}{8} + \left(-\frac{1}{20}\right)(-2) = \frac{1}{8} + \frac{1}{10}\\ \amp = \frac{5 + 4}{40} = \frac{9}{40} = 0.225\text{ amperes/min} \end{align*}
    🔗
    Thus, at this instant, the current \(I\) is increasing at a rate of exactly \(0.225\text{ amperes per minute}\text{.}\)
