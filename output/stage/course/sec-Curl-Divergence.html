    Section 5.4 Curl and Divergence

    In this section, we will introduce two important concepts in vector calculus: curl and divergence. These concepts are used to describe the behavior of vector fields in three-dimensional space.
    🔗

    Definition 5.45. The Curl of a Vector Field.

    If \(\mathbf{F} = P\mathbf{i} + Q\mathbf{j} + R\mathbf{k}\) is a vector field on \(\mathbb{R}^3\) and the partial derivatives of \(P\text{,}\) \(Q\text{,}\) and \(R\) all exist, then the curl of \(\mathbf{F}\) is the vector field defined by:
    \begin{equation*} \operatorname{curl} \mathbf{F} = \left(\frac{\partial R}{\partial y} - \frac{\partial Q}{\partial z}\right) \mathbf{i} + \left(\frac{\partial P}{\partial z} - \frac{\partial R}{\partial x}\right) \mathbf{j} + \left(\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y}\right) \mathbf{k}. \end{equation*}
    In terms of the del gradient operator, this can be written as a formal cross product:
    \begin{equation*} \operatorname{curl} \mathbf{F} = \nabla \times \mathbf{F}. \end{equation*}
    🔗
    🔗

    Example 5.46.

    If \(\mathbf{F}(x,y,z) = xz\mathbf{i} + xyz\mathbf{j} - y^2\mathbf{k}\text{,}\) find \(\operatorname{curl}\mathbf{F}\text{.}\)
    🔗
    Solution.
    Step 1: Set Up the Formal Cross Product Matrix
    🔗
    We express the curl as a symbolic 3-by-3 determinant using the component fields \(P = xz\text{,}\) \(Q = xyz\text{,}\) and \(R = -y^2\text{:}\)
    \begin{equation*} \operatorname{curl} \mathbf{F} = \nabla \times \mathbf{F} = \begin{vmatrix} \mathbf{i} \amp \mathbf{j} \amp \mathbf{k} \\ \frac{\partial}{\partial x} \amp \frac{\partial}{\partial y} \amp \frac{\partial}{\partial z} \\ xz \amp xyz \amp -y^2 \end{vmatrix} \end{equation*}
    🔗
    Step 2: Expand the Determinant by Coactors
    🔗
    Expanding along the first row yields three 2-by-2 component derivative blocks:
    \begin{align*} \operatorname{curl} \mathbf{F} \amp = \begin{vmatrix} \frac{\partial}{\partial y} \amp \frac{\partial}{\partial z} \\ xyz \amp -y^2 \end{vmatrix}\mathbf{i} - \begin{vmatrix} \frac{\partial}{\partial x} \amp \frac{\partial}{\partial z} \\ xz \amp -y^2 \end{vmatrix}\mathbf{j} + \begin{vmatrix} \frac{\partial}{\partial x} \amp \frac{\partial}{\partial y} \\ xz \amp xyz \end{vmatrix}\mathbf{k}\\ \amp = \left( \frac{\partial}{\partial y}(-y^2) - \frac{\partial}{\partial z}(xyz) \right)\mathbf{i} - \left( \frac{\partial}{\partial x}(-y^2) - \frac{\partial}{\partial z}(xz) \right)\mathbf{j} + \left( \frac{\partial}{\partial x}(xyz) - \frac{\partial}{\partial y}(xz) \right)\mathbf{k} \end{align*}
    🔗
    Step 3: Evaluate Partial Derivatives and Simplify
    🔗
    Computing each bracketed operation yields:
    • For the \(\mathbf{i}\) component:
      \begin{equation*} \frac{\partial}{\partial y}(-y^2) - \frac{\partial}{\partial z}(xyz) = -2y - xy. \end{equation*}
      🔗
      🔗
    • For the \(\mathbf{j}\) component (including the outer subtraction sign):
      \begin{equation*} -\left( \frac{\partial}{\partial x}(-y^2) - \frac{\partial}{\partial z}(xz) \right) = -\left( 0 - x \right) = x. \end{equation*}
      🔗
      🔗
    • For the \(\mathbf{k}\) component:
      \begin{equation*} \frac{\partial}{\partial x}(xyz) - \frac{\partial}{\partial y}(xz) = yz - 0 = yz. \end{equation*}
      🔗
      🔗
    🔗
    Step 4: Conclusion
    🔗
    Combining the evaluated terms gives the final vector field:
    \begin{equation*} \operatorname{curl} \mathbf{F} = (-2y - xy)\mathbf{i} + x\mathbf{j} + yz\mathbf{k} = \langle -2y - xy, \ x, \ yz \rangle. \end{equation*}
    🔗
    🔗
    🔗

    Theorem 5.47. The Curl of a Gradient Field.

    If \(f\) is a function of three variables that has continuous second-order partial derivatives, then:
    \begin{equation*} \operatorname{curl}(\nabla f) = \mathbf{0}. \end{equation*}
    Since a conservative vector field is defined as one for which there exists a potential scalar function such that \(\mathbf{F}=\nabla f\text{,}\) it follows that if \(\mathbf{F}\) is a conservative vector field, then:
    \begin{equation*} \operatorname{curl} \mathbf{F} = \mathbf{0}. \end{equation*}
    🔗
    🔗

    Example 5.48.

    Show that the vector field \(\mathbf{F}(x,y,z) = xz\mathbf{i} + xyz\mathbf{j} - y^2\mathbf{k}\) is not conservative.
    🔗
    Solution.
    Step 1: Compute the Curl of the Field
    🔗
    According to Theorem 5.47, if a vector field is conservative, its curl must equal the zero vector. Let’s calculate \(\operatorname{curl} \mathbf{F}\) using a symbolic cross product:
    \begin{equation*} \operatorname{curl} \mathbf{F} = \nabla \times \mathbf{F} = \begin{vmatrix} \mathbf{i} \amp \mathbf{j} \amp \mathbf{k} \\ \frac{\partial}{\partial x} \amp \frac{\partial}{\partial y} \amp \frac{\partial}{\partial z} \\ xz \amp xyz \amp -y^2 \end{vmatrix} \end{equation*}
    Expanding along the first row yields:
    \begin{align*} \operatorname{curl} \mathbf{F} \amp = \left( \frac{\partial}{\partial y}(-y^2) - \frac{\partial}{\partial z}(xyz) \right)\mathbf{i} - \left( \frac{\partial}{\partial x}(-y^2) - \frac{\partial}{\partial z}(xz) \right)\mathbf{j} + \left( \frac{\partial}{\partial x}(xyz) - \frac{\partial}{\partial y}(xz) \right)\mathbf{k}\\ \amp = (-2y - xy)\mathbf{i} - (0 - x)\mathbf{j} + (yz - 0)\mathbf{k}\\ \amp = \langle -2y - xy, \ x, \ yz \rangle. \end{align*}
    🔗
    Step 2: Conclusion
    🔗
    Because \(\operatorname{curl} \mathbf{F} = \langle -2y - xy, \ x, \ yz \rangle \neq \langle 0, 0, 0 \rangle\) throughout \(\mathbb{R}^3\text{,}\) the field fails the zero-curl condition. Therefore, \(\mathbf{F}\) is not conservative.
    🔗
    🔗
    🔗

    Example 5.49.

    Show that the vector field \(\mathbf{F}(x,y,z) = y^2z^3\mathbf{i} + 2xyz^3\mathbf{j} + 3xy^2z^2\mathbf{k}\) is conservative.
    🔗
    Solution.
    Step 1: Compute the Curl of the Field
    🔗
    We match the given parameters to the component fields \(P = y^2z^3\text{,}\) \(Q = 2xyz^3\text{,}\) and \(R = 3xy^2z^2\text{.}\) We evaluate their symbolic determinant:
    \begin{equation*} \operatorname{curl} \mathbf{F} = \nabla \times \mathbf{F} = \begin{vmatrix} \mathbf{i} \amp \mathbf{j} \amp \mathbf{k} \\ \frac{\partial}{\partial x} \amp \frac{\partial}{\partial y} \amp \frac{\partial}{\partial z} \\ y^2z^3 \amp 2xyz^3 \amp 3xy^2z^2 \end{vmatrix} \end{equation*}
    🔗
    Step 2: Evaluate Component Derivatives
    🔗
    Expanding by row minors and taking partial derivatives:
    • For the \(\mathbf{i}\) component:
      \begin{equation*} \frac{\partial R}{\partial y} - \frac{\partial Q}{\partial z} = \frac{\partial}{\partial y}(3xy^2z^2) - \frac{\partial}{\partial z}(2xyz^3) = 6xyz^2 - 6xyz^2 = 0. \end{equation*}
      🔗
      🔗
    • For the \(\mathbf{j}\) component:
      \begin{equation*} -\left( \frac{\partial R}{\partial x} - \frac{\partial P}{\partial z} \right) = -\left( \frac{\partial}{\partial x}(3xy^2z^2) - \frac{\partial}{\partial z}(y^2z^3) \right) = -\left( 3y^2z^2 - 3y^2z^2 \right) = 0. \end{equation*}
      🔗
      🔗
    • For the \(\mathbf{k}\) component:
      \begin{equation*} \frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y} = \frac{\partial}{\partial x}(2xyz^3) - \frac{\partial}{\partial y}(y^2z^3) = 2yz^3 - 2yz^3 = 0. \end{equation*}
      🔗
      🔗
    🔗
    Step 3: Conclusion
    🔗
    Combining our calculated vector paths yields:
    \begin{equation*} \operatorname{curl} \mathbf{F} = 0\mathbf{i} + 0\mathbf{j} + 0\mathbf{k} = \langle 0, 0, 0 \rangle = \mathbf{0}. \end{equation*}
    Because \(\operatorname{curl} \mathbf{F} = \mathbf{0}\) continuously everywhere on the simply-connected domain \(\mathbb{R}^3\text{,}\) the test is satisfied. Therefore, the vector field \(\mathbf{F}\) is conservative.
    🔗
    🔗
    🔗

    Definition 5.50. The Divergence of a Vector Field.

    If \(\mathbf{F} = P\mathbf{i} + Q\mathbf{j} + R\mathbf{k}\) is a vector field on \(\mathbb{R}^3\) and the partial derivatives \(\frac{\partial P}{\partial x}\text{,}\) \(\frac{\partial Q}{\partial y}\text{,}\) and \(\frac{\partial R}{\partial z}\) all exist, then the divergence of \(\mathbf{F}\) is the scalar field defined by:
    \begin{equation*} \operatorname{div} \mathbf{F} = \frac{\partial P}{\partial x} + \frac{\partial Q}{\partial y} + \frac{\partial R}{\partial z}. \end{equation*}
    In terms of the del gradient operator, this can be written as a formal dot product:
    \begin{equation*} \operatorname{div} \mathbf{F} = \nabla \cdot \mathbf{F}. \end{equation*}
    🔗
    🔗

    Example 5.51.

    If \(\mathbf{F}(x,y,z) = xz\mathbf{i} + xyz\mathbf{j} - y^2\mathbf{k}\text{,}\) find \(\operatorname{div}\mathbf{F}\text{.}\)
    🔗
    Solution.
    Step 1: Identify Component Functions
    🔗
    We match the given vector field to the standard component form \(\mathbf{F} = P\mathbf{i} + Q\mathbf{j} + R\mathbf{k}\text{:}\)
    \begin{equation*} P(x,y,z) = xz, \quad Q(x,y,z) = xyz, \quad R(x,y,z) = -y^2. \end{equation*}
    🔗
    Step 2: Compute Partial Derivatives
    🔗
    We find the corresponding first-order partial derivative for each independent component:
    • Differentiate \(P\) with respect to \(x\text{:}\)
      \begin{equation*} \frac{\partial P}{\partial x} = \frac{\partial}{\partial x}(xz) = z. \end{equation*}
      🔗
      🔗
    • Differentiate \(Q\) with respect to \(y\text{:}\)
      \begin{equation*} \frac{\partial Q}{\partial y} = \frac{\partial}{\partial y}(xyz) = xz. \end{equation*}
      🔗
      🔗
    • Differentiate \(R\) with respect to \(z\text{:}\)
      \begin{equation*} \frac{\partial R}{\partial z} = \frac{\partial}{\partial z}(-y^2) = 0. \end{equation*}
      🔗
      🔗
    🔗
    Step 3: Combine and Simplify
    🔗
    According to the divergence definition, we sum the three partial derivative terms together:
    \begin{align*} \operatorname{div} \mathbf{F} \amp = \frac{\partial P}{\partial x} + \frac{\partial Q}{\partial y} + \frac{\partial R}{\partial z}\\ \amp = z + xz + 0 = z + xz. \end{align*}
    The divergence of the vector field is the scalar function \(z + xz\text{.}\)
    🔗
    🔗
    🔗

    Theorem 5.52. The Divergence of a Curl.

    If \(\mathbf{F} = P\mathbf{i} + Q\mathbf{j} + R\mathbf{k}\) is a vector field on \(\mathbb{R}^3\) and its component functions \(P\text{,}\) \(Q\text{,}\) and \(R\) have continuous second-order partial derivatives, then:
    \begin{equation*} \operatorname{div}(\operatorname{curl} \mathbf{F}) = \nabla \cdot (\nabla \times \mathbf{F}) = 0. \end{equation*}
    🔗
    🔗

    Example 5.53.

    Show that the vector field \(\mathbf{F}(x,y,z) = xz\mathbf{i} + xyz\mathbf{j} - y^2\mathbf{k}\) cannot be written as the curl of another vector field; that is, \(\mathbf{F} \neq \operatorname{curl} \mathbf{G}\) for any vector field \(\mathbf{G}\text{.}\)
    🔗
    Solution.
    Step 1: Apply the Divergence of a Curl Theorem
    🔗
    According to Theorem 5.52, if a vector field \(\mathbf{F}\) can be expressed as the curl of some vector field \(\mathbf{G}\) (meaning \(\mathbf{F} = \operatorname{curl} \mathbf{G}\)), then taking its divergence must automatically yield zero:
    \begin{equation*} \operatorname{div} \mathbf{F} = \operatorname{div}(\operatorname{curl} \mathbf{G}) = 0. \end{equation*}
    Therefore, to show that \(\mathbf{F}\) cannot be written as a curl, we only need to compute \(\operatorname{div} \mathbf{F}\) and verify that it is not identically zero.
    🔗
    Step 2: Calculate the Divergence of \(\mathbf{F}\)
    🔗
    We extract the component fields from the problem statement: \(P = xz\text{,}\) \(Q = xyz\text{,}\) and \(R = -y^2\text{.}\) We apply the standard divergence sum formula:
    \begin{align*} \operatorname{div} \mathbf{F} \amp = \frac{\partial P}{\partial x} + \frac{\partial Q}{\partial y} + \frac{\partial R}{\partial z}\\ \amp = \frac{\partial}{\partial x}(xz) + \frac{\partial}{\partial y}(xyz) + \frac{\partial}{\partial z}(-y^2)\\ \amp = z + xz + 0 = z(1 + x). \end{align*}
    🔗
    Step 3: Conclusion
    🔗
    The result shows that \(\operatorname{div} \mathbf{F} = z(1+x)\text{,}\) which is not identically zero everywhere in \(\mathbb{R}^3\) (for instance, at the point \((1,1,1)\text{,}\) \(\operatorname{div}\mathbf{F} = 2\)).
    🔗
    Because \(\operatorname{div} \mathbf{F} \neq 0\text{,}\) it fails the fundamental identity requirement of a curl field. Therefore, \(\mathbf{F}\) cannot be written as the curl of another vector field (\(\mathbf{F} \neq \operatorname{curl} \mathbf{G}\)).
