    Section 5.2 Line Integrals

    A line integral is a type of integral where the function to be integrated is evaluated along a curve or path in space. It generalizes the concept of integration to functions defined on curves, allowing us to compute quantities such as work done by a force field along a path.
    🔗

    Definition 5.7. Line Integral along a Curve.

    If \(f\) is defined on a smooth curve \(C\) given by parametric equations
    \begin{equation*} x=x(t), \quad y=y(y), \quad a \leq t \leq b, \end{equation*}
    or by the vector equation \(\mathbf{r}(t) = x(t)\mathbf{i} + y(t)\mathbf{j}\text{,}\) then the line integral of \(f\) along \(C\) is:
    \begin{equation*} \int_C f(x,y)\,ds = \lim_{n \rightarrow \infty} \sum^{n}_{i=1} f(x_i, y_i)\Delta s_i, \end{equation*}
    provided this limit exists.
    🔗
    🔗
    Since the differential of arc length along \(C\) is related to the parameter \(t\) by \(ds = \sqrt{\left(\frac{dx}{dt}\right)^2 + \left(\frac{dy}{dt}\right)^2}\,dt\text{,}\) we arrive at the following formula for evaluation:
    🔗

    Theorem 5.8. Evaluation of Line Integrals.

    \begin{equation*} \int_C f(x,y)\,ds = \int^b_a f(x(t), y(t)) \sqrt{\left(\frac{dx}{dt}\right)^2 + \left(\frac{dy}{dt}\right)^2}\,dt. \end{equation*}
    🔗
    🔗

    Example 5.9.

    Evaluate \(\int_C (2+x^2y)\,ds\text{,}\) where \(C\) is the upper half of the unit circle \(x^2+y^2=1\text{.}\)
    🔗
    Solution.
    Step 1: Parameterize the Path curve \(C\)
    🔗
    The upper half of the unit circle can be parameterized using standard polar angles where:
    \begin{equation*} x(t) = \cos(t), \quad y(t) = \sin(t), \quad 0 \leq t \leq \pi. \end{equation*}
    Next, we calculate the derivatives and the arc length differential element \(ds\text{:}\)
    \begin{equation*} \frac{dx}{dt} = -\sin(t), \quad \frac{dy}{dt} = \cos(t) \end{equation*}
    \begin{equation*} ds = \sqrt{\left(\frac{dx}{dt}\right)^2 + \left(\frac{dy}{dt}\right)^2}\,dt = \sqrt{(-\sin t)^2 + (\cos t)^2}\,dt = 1\,dt. \end{equation*}
    🔗
    Step 2: Evaluate the Iterated Integral
    🔗
    Substituting the parametric expressions into our line integral layout formula:
    \begin{align*} \int_C (2+x^2y)\,ds \amp = \int_{0}^{\pi} \left( 2 + (\cos(t))^2\sin(t) \right) \cdot (1)\,dt\\ \amp = \int_{0}^{\pi} 2\,dt + \int_{0}^{\pi} \cos^2(t)\sin(t)\,dt\\ \amp = \Big[ 2t \Big]_{0}^{\pi} + \left[ -\frac{1}{3}\cos^3(t) \right]_{0}^{\pi}\\ \amp = (2\pi - 0) + \left( -\frac{1}{3}(-1)^3 - \left(-\frac{1}{3}(1)^3\right) \right)\\ \amp = 2\pi + \left( \frac{1}{3} + \frac{1}{3} \right) = 2\pi + \frac{2}{3}. \end{align*}
    The exact value of the line integral along the semicircle path is \(2\pi + \frac{2}{3}\text{.}\)
    🔗
    🔗
    🔗

    Example 5.11.

    Evaluate \(\int_C (2x)\,ds\text{,}\) where \(C\) consists of the arc \(C_1\) of the parabola \(y=x^2\) from \((0,0)\) to \((1,1)\) followed by the vertical line segment \(C_2\) from \((1,1)\) to \((1,2)\text{.}\)
    🔗
    Solution.
    Step 1: Parameterize the Piecewise Paths and Sketch
    🔗
    Because \(C\) is a piecewise smooth curve, we split the line integral into two separate parts: \(\int_C = \int_{C_1} + \int_{C_2}\text{.}\)
    🔗
    • Path \(C_1\) (Parabola): Parameterizing using \(x=t\) gives:
      \begin{equation*} x(t) = t, \quad y(t) = t^2, \quad 0 \leq t \leq 1. \end{equation*}
      \begin{equation*} \frac{dx}{dt} = 1, \quad \frac{dy}{dt} = 2t \implies ds = \sqrt{1 + (2t)^2}\,dt = \sqrt{1+4t^2}\,dt. \end{equation*}
      🔗
      🔗
    • Path \(C_2\) (Vertical Line): Parameterizing using \(y=t\) along the vertical boundary line \(x=1\) gives:
      \begin{equation*} x(t) = 1, \quad y(t) = t, \quad 1 \leq t \leq 2. \end{equation*}
      \begin{equation*} \frac{dx}{dt} = 0, \quad \frac{dy}{dt} = 1 \implies ds = \sqrt{0^2 + 1^2}\,dt = 1\,dt. \end{equation*}
      🔗
      🔗
    Step 2: Evaluate the Component Integrals
    🔗
    We evaluate each integral expression independently:
    🔗
    • Evaluation along \(C_1\text{:}\)
      \begin{equation*} \int_{C_1} 2x\,ds = \int_{0}^{1} 2t\sqrt{1+4t^2}\,dt \end{equation*}
      Using \(u\)-substitution where \(u = 1+4t^2 \implies du = 8t\,dt \implies \frac{1}{4}\,du = 2t\,dt\text{.}\) Limits: when \(t=0 \implies u=1\text{,}\) and when \(t=1 \implies u=5\text{.}\)
      \begin{equation*} \int_{1}^{5} \frac{1}{4}u^{1/2}\,du = \left[ \frac{1}{4} \cdot \frac{2}{3}u^{3/2} \right]_{1}^{5} = \frac{1}{6}\left( 5^{3/2} - 1^{3/2} \right) = \frac{5\sqrt{5}-1}{6}. \end{equation*}
      🔗
      🔗
    • Evaluation along \(C_2\text{:}\)
      \begin{equation*} \int_{C_2} 2x\,ds = \int_{1}^{2} 2(1) \cdot (1)\,dt = \int_{1}^{2} 2\,dt = \Big[ 2t \Big]_{1}^{2} = 4 - 2 = 2. \end{equation*}
      🔗
      🔗
    Step 3: Combine Results
    🔗
    Adding the two individual components together gives the total value of the line integral:
    \begin{equation*} \int_C 2x\,ds = \frac{5\sqrt{5}-1}{6} + 2 = \frac{5\sqrt{5}-1 + 12}{6} = \frac{5\sqrt{5}+11}{6}. \end{equation*}
    The complete line integral value along the piecewise path is exactly \(\frac{5\sqrt{5}+11}{6}\text{.}\)
    🔗
    🔗
    🔗
    To distinguish the original line integral \(\int_C f(x,y)\,ds\) from \(\int_C f(x,y)\,dx\) and \(\int_C f(x,y)\,dy\text{,}\) we call the first the line integral with respect to arc length. If everything is expressed in terms of the parameter \(t\text{,}\) where \(x=x(t)\text{,}\) \(y=y(t)\text{,}\) \(dx=x'(t)\,dt\text{,}\) and \(dy=y'(t)\,dt\text{,}\) we have:
    \begin{equation*} \int_C f(x,y)\,dx = \int^b_a f(x(t),y(t))x'(t)\,dt \end{equation*}
    \begin{equation*} \int_C f(x,y)\,dy = \int^b_a f(x(t),y(t))y'(t)\,dt. \end{equation*}
    It often happens that line integrals with respect to \(x\) and \(y\) occur together. When this happens, we write:
    \begin{equation*} \int_C P(x,y)\,dx + \int_C Q(x,y)\,dy = \int_C P(x,y)\,dx + Q(x,y)\,dy. \end{equation*}
    🔗
    A vector representation of the straight line segment that starts from \(\mathbf{r}_0\) and ends at \(\mathbf{r}_1\) is given by:
    \begin{equation*} \mathbf{r}(t) = (1-t)\mathbf{r}_0 + t\mathbf{r}_1, \quad 0 \leq t \leq 1. \end{equation*}
    🔗

    Example 5.13.

    Evaluate \(\int_C y^2\,dx + x\,dy\text{,}\) where \(C\) is the line segment from \((-5,-3)\) to \((0,2)\text{.}\)
    🔗
    Solution.
    Step 1: Parameterize the Straight Line Segment \(C\)
    🔗
    Using the vector representation for a line segment with initial point \(\mathbf{r}_0 = \langle -5, -3 \rangle\) and terminal point \(\mathbf{r}_1 = \langle 0, 2 \rangle\text{:}\)
    \begin{align*} \mathbf{r}(t) \amp = (1-t)\langle -5, -3 \rangle + t\langle 0, 2 \rangle\\ \amp = \langle -5 + 5t, -3 + 3t \rangle + \langle 0, 2t \rangle = \langle 5t - 5, 5t - 3 \rangle \end{align*}
    This gives our parametric equations on the interval \(0 \leq t \leq 1\text{:}\)
    \begin{equation*} x(t) = 5t - 5 \implies dx = 5\,dt \end{equation*}
    \begin{equation*} y(t) = 5t - 3 \implies dy = 5\,dt. \end{equation*}
    🔗
    Step 2: Setup and Evaluate the Iterated Integral
    🔗
    Substituting our parametric definitions and differentials into the given coordinate line integral string:
    \begin{align*} \int_C y^2\,dx + x\,dy \amp = \int_{0}^{1} \left[ (5t-3)^2 \cdot (5) + (5t-5) \cdot (5) \right] \, dt\\ \amp = 5 \int_{0}^{1} \left[ (25t^2 - 30t + 9) + (5t - 5) \right] \, dt\\ \amp = 5 \int_{0}^{1} \left( 25t^2 - 25t + 4 \right) \, dt\\ \amp = 5 \left[ \frac{25}{3}t^3 - \frac{25}{2}t^2 + 4t \right]_{0}^{1}\\ \amp = 5 \left( \frac{25}{3}(1)^3 - \frac{25}{2}(1)^2 + 4(1) - 0 \right)\\ \amp = 5 \left( \frac{25}{3} - \frac{25}{2} + 4 \right). \end{align*}
    Finding a common denominator of \(6\) for the rational components:
    \begin{align*} \int_C y^2\,dx + x\,dy \amp = 5 \left( \frac{50}{6} - \frac{75}{6} + \frac{24}{6} \right)\\ \amp = 5 \left( -\frac{1}{6} \right) = -\frac{5}{6}. \end{align*}
    The value of the line integral with respect to the coordinate variables is exactly \(-\frac{5}{6}\text{.}\)
    🔗
    🔗
    🔗

    Example 5.15.

    Evaluate \(\int_C y^2\,dx + x\,dy\text{,}\) where \(C\) is the arc of the parabola \(x=4-y^2\) from \((-5,-3)\) to \((0,2)\text{.}\)
    🔗
    Solution.
    Step 1: Parameterize the Parabolic Path Curve \(C\)
    🔗
    Since the parabola is given in the form \(x = g(y)\text{,}\) it is most natural to use \(y\) itself as the parameter. Let:
    \begin{equation*} y = t \implies dy = dt \end{equation*}
    where the parameter \(t\) varies from the initial \(y\)-coordinate (\(-3\)) to the terminal \(y\)-coordinate (\(2\)): \(-3 \leq t \leq 2\text{.}\) Substituting this into the parabola equation gives the parametric description for \(x\) and its differential:
    \begin{equation*} x = 4 - t^2 \implies dx = -2t\,dt. \end{equation*}
    🔗
    Step 2: Setup and Evaluate the Iterated Integral
    🔗
    Substituting our parametric equations and differentials into the given line integral yields:
    \begin{align*} \int_C y^2\,dx + x\,dy \amp = \int_{-3}^{2} \left[ (t)^2 \cdot (-2t\,dt) + (4 - t^2) \cdot (dt) \right]\\ \amp = \int_{-3}^{2} \left( -2t^3 + 4 - t^2 \right) \, dt\\ \amp = \left[ -\frac{1}{2}t^4 + 4t - \frac{1}{3}t^3 \right]_{-3}^{2} \end{align*}
    🔗
    Now, evaluating at the upper and lower limits:
    \begin{align*} \text{Upper Limit } (t=2): \amp \quad -\frac{1}{2}(2)^4 + 4(2) - \frac{1}{3}(2)^3 = -8 + 8 - \frac{8}{3} = -\frac{8}{3}\\ \text{Lower Limit } (t=-3): \amp \quad -\frac{1}{2}(-3)^4 + 4(-3) - \frac{1}{3}(-3)^3 = -\frac{81}{2} - 12 - (-9) = -\frac{81}{2} - 3 = -\frac{87}{2} \end{align*}
    🔗
    Subtracting the lower limit evaluation from the upper limit evaluation gives:
    \begin{align*} V \amp = -\frac{8}{3} - \left( -\frac{87}{2} \right) = -\frac{16}{6} + \frac{261}{6}\\ \amp = \frac{245}{6}. \end{align*}
    The value of the line integral along the parabolic arc path is exactly \(\frac{245}{6}\text{.}\)
    🔗
    🔗
    🔗
    If \(-C\) denotes the curve consisting of the same points as \(C\) but with the opposite orientation, then we have:
    \begin{equation*} \int_{-C} f(x,y)\,dx = -\int_C f(x,y)\,dx, \quad \int_{-C} f(x,y)\,dy = -\int_C f(x,y)\,dy. \end{equation*}
    But if we integrate with respect to arc length, the value of the line integral does not change when we reverse the orientation of the curve:
    \begin{equation*} \int_{-C} f(x,y)\,ds = \int_C f(x,y)\,ds. \end{equation*}
    This occurs because the arc length differential segment \(\Delta s_i\) is always positive, whereas the coordinate differentials \(\Delta x_i\) and \(\Delta y_i\) change sign when we reverse the orientation of \(C\text{.}\)
    🔗

    Definition 5.17. Line Integrals in Space.

    If \(f\) is defined on a smooth space curve \(C\) given by parametric equations
    \begin{equation*} x=x(t), \quad y=y(t), \quad z=z(t), \quad a \leq t \leq b, \end{equation*}
    or by the vector equation \(\mathbf{r}(t) = x(t)\mathbf{i} + y(t)\mathbf{j} + z(t)\mathbf{k}\text{,}\) then the line integral of \(f\) along \(C\) is:
    \begin{equation*} \int_C f(x,y,z)\,ds = \lim_{n \rightarrow \infty} \sum^{n}_{i=1} f(x_i, y_i, z_i)\Delta s_i, \end{equation*}
    provided this limit exists.
    🔗
    🔗

    Theorem 5.18. Evaluation of Space Line Integrals.

    The line integral can be evaluated by expressing the differential element \(ds\) in terms of the parameter \(t\text{:}\)
    \begin{equation*} \int_C f(x,y,z)\,ds = \int^b_a f(x(t),y(t),z(t)) \sqrt{\left(\frac{dx}{dt}\right)^2+\left(\frac{dy}{dt}\right)^2+\left(\frac{dz}{dt}\right)^2}\,dt = \int^b_a f(\mathbf{r}(t))|\mathbf{r}'(t)|\,dt. \end{equation*}
    🔗
    🔗

    Example 5.19.

    Evaluate \(\int_C y \sin z \,ds\text{,}\) where \(C\) is the circular helix given by the equations \(x=\cos t, \ y=\sin t, \ z=t\text{,}\) for \(0 \leq t \leq 2\pi\text{.}\)
    🔗
    Solution.
    Step 1: Compute the Arc Length Differential Element \(ds\)
    🔗
    We find the derivatives of the parametric equations with respect to the parameter \(t\text{:}\)
    \begin{equation*} \frac{dx}{dt} = -\sin t, \quad \frac{dy}{dt} = \cos t, \quad \frac{z}{dt} = 1. \end{equation*}
    Substituting these into the radical magnitude formula for space curves gives:
    \begin{equation*} ds = \sqrt{(-\sin t)^2 + (\cos t)^2 + (1)^2}\,dt = \sqrt{\sin^2 t + \cos^2 t + 1}\,dt = \sqrt{1 + 1}\,dt = \sqrt{2}\,dt. \end{equation*}
    🔗
    Step 2: Set Up and Evaluate the Iterated Integral
    🔗
    We rewrite the integrand entirely in terms of \(t\) by substituting \(y = \sin t\text{,}\) \(z = t\text{,}\) and \(ds = \sqrt{2}\,dt\) across our bounds from \(0\) to \(2\pi\text{:}\)
    \begin{equation*} \int_C y \sin z \,ds = \int_{0}^{2\pi} (\sin t)(\sin t) \cdot \left(\sqrt{2}\,dt\right) = \sqrt{2} \int_{0}^{2\pi} \sin^2 t \,dt. \end{equation*}
    Using the trigonometric half-angle reduction identity \(\sin^2 t = \frac{1-\cos(2t)}{2}\text{:}\)
    \begin{align*} \sqrt{2} \int_{0}^{2\pi} \left( \frac{1-\cos(2t)}{2} \right) \,dt \amp = \frac{\sqrt{2}}{2} \Big[ t - \frac{1}{2}\sin(2t) \Big]_{0}^{2\pi}\\ \amp = \frac{\sqrt{2}}{2} \left[ \left(2\pi - \frac{1}{2}\sin(4\pi)\right) - \left(0 - \frac{1}{2}\sin(0)\right) \right]\\ \amp = \frac{\sqrt{2}}{2} \Big[ (2\pi - 0) - 0 \Big] = \frac{\sqrt{2}}{2}(2\pi) = \pi\sqrt{2}. \end{align*}
    The exact value of the line integral along the circular helix space curve is \(\pi\sqrt{2}\text{.}\)
    🔗
    🔗
    🔗

    Example 5.21.

    Evaluate \(\int_C y\,dx + z\,dy + x\,dz\text{,}\) where \(C\) consists of the line segment \(C_1\) from \((2,0,0)\) to \((3,4,5)\) followed by the vertical line segment \(C_2\) from \((3,4,5)\) to \((3,4,0)\text{.}\)
    🔗
    Solution.
    Because \(C\) is a piecewise smooth curve, we split the line integral into the sum of its two separate component tracks: \(\int_C = \int_{C_1} + \int_{C_2}\text{.}\)
    🔗
    Step 1: Parameterize Path \(C_1\) and Evaluate
    🔗
    The straight line segment \(C_1\) starts at \(\mathbf{r}_0 = \langle 2,0,0 \rangle\) and ends at \(\mathbf{r}_1 = \langle 3,4,5 \rangle\text{.}\) Using the standard vector parameterization formula \(\mathbf{r}(t) = (1-t)\mathbf{r}_0 + t\mathbf{r}_1\) for \(0 \leq t \leq 1\text{:}\)
    \begin{equation*} x(t) = 2 + t \implies dx = dt \end{equation*}
    \begin{equation*} y(t) = 4t \implies dy = 4\,dt \end{equation*}
    \begin{equation*} z(t) = 5t \implies dz = 5\,dt. \end{equation*}
    Substituting these parametric expressions into the first line integral gives:
    \begin{align*} \int_{C_1} y\,dx + z\,dy + x\,dz \amp = \int_{0}^{1} \left[ (4t)(dt) + (5t)(4\,dt) + (2+t)(5\,dt) \right]\\ \amp = \int_{0}^{1} \left( 4t + 20t + 10 + 5t \right) \, dt\\ \amp = \int_{0}^{1} \left( 29t + 10 \right) \, dt = \left[ \frac{29}{2}t^2 + 10t \right]_{0}^{1} = \frac{29}{2} + 10 = \frac{49}{2}. \end{align*}
    🔗
    Step 2: Parameterize Path \(C_2\) and Evaluate
    🔗
    The vertical line segment \(C_2\) goes straight downward from \((3,4,5)\) to \((3,4,0)\text{.}\) Along this path, \(x\) and \(y\) remain constant, which makes their differentials zero:
    \begin{equation*} x = 3 \implies dx = 0 \end{equation*}
    \begin{equation*} y = 4 \implies dy = 0 \end{equation*}
    The height variable \(z\) drops continuously from \(5\) down to \(0\text{.}\) Let’s use \(z = t\) as our parameter on the directional interval from \(t = 5\) to \(t = 0\text{,}\) where \(dz = dt\text{:}\)
    \begin{align*} \int_{C_2} y\,dx + z\,dy + x\,dz \amp = \int_{5}^{0} \left[ (4)(0) + (t)(0) + (3)(dt) \right]\\ \amp = \int_{5}^{0} 3 \, dt = \Big[ 3t \Big]_{5}^{0} = 3(0) - 3(5) = -15. \end{align*}
    🔗
    Step 3: Combine Component Integrals
    🔗
    Adding the results from the two paths together gives the final total evaluation:
    \begin{equation*} \int_C y\,dx + z\,dy + x\,dz = \frac{49}{2} + (-15) = \frac{49}{2} - \frac{30}{2} = \frac{19}{2}. \end{equation*}
    The complete line integral value along the piecewise space curve path is exactly \(\frac{19}{2}\) (or \(9.5\)).
    🔗
    🔗
    🔗

    Definition 5.23. Line Integral of a Vector Field.

    Let \(\mathbf{F}\) be a continuous vector field defined on a smooth curve \(C\) given by a vector function \(\mathbf{r}(t)\) for \(a \leq t \leq b\text{.}\) Then the line integral of \(\mathbf{F}\) along \(C\) is:
    \begin{equation*} \int_C \mathbf{F} \cdot d\mathbf{r} = \int^b_a \mathbf{F}(\mathbf{r}(t)) \cdot \mathbf{r}'(t)\,dt = \int_C \mathbf{F} \cdot \mathbf{T}\,ds, \end{equation*}
    where \(\mathbf{T}\) is the unit tangent vector at the point \((x,y,z)\) on \(C\text{.}\)
    🔗
    🔗

    Definition 5.24. Work Done by a Force Field.

    We define the work \(W\) done by a continuous force field \(\mathbf{F}\) as the limit of Riemann sums over a smooth curve \(C\text{:}\)
    \begin{equation*} W = \int_C \mathbf{F}(x,y,z) \cdot \mathbf{Y}(x,y,z)\,ds = \int_C \mathbf{F} \cdot \mathbf{T}\,ds. \end{equation*}
    If the curve \(C\) is given by the parametric vector equation \(\mathbf{r}(t) = x(t)\mathbf{i} + y(t)\mathbf{j} + z(t)\mathbf{k}\text{,}\) then the unit tangent vector profile simplifies to \(\mathbf{T}(t) = \frac{\mathbf{r}'(t)}{|\mathbf{r}'(t)|}\text{,}\) allowing us to rewrite the definition as an evaluation formula:
    \begin{equation*} W = \int^b_a \left[ \mathbf{F}(\mathbf{r}(t)) \cdot \frac{\mathbf{r}'(t)}{|\mathbf{r}'(t)|} \right] |\mathbf{r}'(t)|\,dt = \int^b_a \mathbf{F}(\mathbf{r}(t)) \cdot \mathbf{r}'(t)\,dt. \end{equation*}
    🔗
    🔗

    Example 5.25.

    Find the work done by the force field \(\mathbf{F}(x,y) = x^2\mathbf{i} - xy\mathbf{j}\) in moving a particle along the quarter-circle \(\mathbf{r}(t) = \cos t\mathbf{i} + \sin t\mathbf{j}\) for \(0 \leq t \leq \pi/2\text{.}\)
    🔗
    Solution.
    Step 1: Express the Vector Field and Velocity Vector in terms of \(t\)
    🔗
    From the parametric equations of the curve, we have \(x = \cos t\) and \(y = \sin t\text{.}\) Substituting these into the vector field equation yields:
    \begin{equation*} \mathbf{F}(\mathbf{r}(t)) = (\cos t)^2\mathbf{i} - (\cos t)(\sin t)\mathbf{j} = \langle \cos^2 t, -\cos t\sin t \rangle. \end{equation*}
    Next, we take the derivative of the path vector equation to compute the velocity vector:
    \begin{equation*} \mathbf{r}'(t) = -\sin t\mathbf{i} + \cos t\mathbf{j} = \langle -\sin t, \cos t \rangle. \end{equation*}
    🔗
    Step 2: Calculate the Inner Dot Product
    🔗
    We find the dot product of the field and the tangent vector:
    \begin{align*} \mathbf{F}(\mathbf{r}(t)) \cdot \mathbf{r}'(t) \amp = \langle \cos^2 t, -\cos t\sin t \rangle \cdot \langle -\sin t, \cos t \rangle\\ \amp = (\cos^2 t)(-\sin t) + (-\cos t\sin t)(\cos t)\\ \amp = -\cos^2 t\sin t - \cos^2 t\sin t = -2\cos^2 t\sin t. \end{align*}
    🔗
    Step 3: Evaluate the Work Integral
    🔗
    Using the work formula for a vector field, we evaluate the definite integral from \(t=0\) to \(t=\pi/2\text{:}\)
    \begin{equation*} W = \int_{0}^{\pi/2} \mathbf{F}(\mathbf{r}(t)) \cdot \mathbf{r}'(t)\,dt = \int_{0}^{\pi/2} -2\cos^2 t\sin t \, dt. \end{equation*}
    We solve this using a simple \(u\)-substitution. Let:
    \begin{equation*} u = \cos t \implies du = -\sin t \, dt \implies -du = \sin t \, dt. \end{equation*}
    Updating limits: when \(t = 0 \implies u = \cos(0) = 1\text{,}\) and when \(t = \pi/2 \implies u = \cos(\pi/2) = 0\text{.}\)
    \begin{align*} W \amp = \int_{1}^{0} -2u^2 (-du) = \int_{1}^{0} 2u^2 \, du = -\int_{0}^{1} 2u^2 \, du\\ \amp = -\left[ \frac{2}{3}u^3 \right]_{0}^{1} = -\left( \frac{2}{3}(1)^3 - 0 \right) = -\frac{2}{3}. \end{align*}
    The total work done by the force field along the quarter-circle is exactly \(-\frac{2}{3}\text{.}\)

    Example 5.26.

    Evaluate \(\int_C \mathbf{F} \cdot d\mathbf{r}\text{,}\) where \(\mathbf{F}(x,y,z) = xy\mathbf{i} + yz\mathbf{j} + zx\mathbf{k}\) and \(C\) is the twisted cubic given by:
    \begin{equation*} x = t, \quad y = t^2, \quad z = t^3, \quad 0 \leq t \leq 1. \end{equation*}
    🔗
    Solution.
    Step 1: Express the Field and Differentials in terms of \(t\)
    🔗
    We translate the position and component paths into the parameter space:
    \begin{equation*} x = t \implies dx = dt \end{equation*}
    \begin{equation*} y = t^2 \implies dy = 2t\,dt \end{equation*}
    \begin{equation*} z = t^3 \implies dz = 3t^2\,dt. \end{equation*}
    Substituting these variables into our component definitions for the field vector gives:
    \begin{equation*} \mathbf{F}(\mathbf{r}(t)) = \langle (t)(t^2), (t^2)(t^3), (t^3)(t) \rangle = \langle t^3, t^5, t^4 \rangle. \end{equation*}
    The tangent vector is: \(\mathbf{r}'(t) = \langle 1, 2t, 3t^2 \rangle\text{.}\)
    🔗
    Step 2: Setup and Evaluate the Vector Line Integral
    🔗
    We find the dot product \(\mathbf{F}(\mathbf{r}(t)) \cdot \mathbf{r}'(t)\) and substitute it into the work form expression:
    \begin{align*} \mathbf{F}(\mathbf{r}(t)) \cdot \mathbf{r}'(t) \amp = \langle t^3, t^5, t^4 \rangle \cdot \langle 1, 2t, 3t^2 \rangle\\ \amp = (t^3)(1) + (t^5)(2t) + (t^4)(3t^2)\\ \amp = t^3 + 2t^6 + 3t^6 = t^3 + 5t^6. \end{align*}
    Now evaluate the integral over the parameter limits from \(0\) to \(1\text{:}\)
    \begin{align*} \int_C \mathbf{F} \cdot d\mathbf{r} \amp = \int_{0}^{1} \left( t^3 + 5t^6 \right) \, dt\\ \amp = \left[ \frac{1}{4}t^4 + \frac{5}{7}t^7 \right]_{0}^{1}\\ \amp = \left( \frac{1}{4}(1)^4 + \frac{5}{7}(1)^7 \right) - 0\\ \amp = \frac{1}{4} + \frac{5}{7} = \frac{7 + 20}{28} = \frac{27}{28}. \end{align*}
    The value of the line integral along the twisted cubic curve is exactly \(\frac{27}{28}\text{.}\)

    Suppose the vector field \(\mathbf{F}\) on \(\mathbb{R}^3\) is given in component form by the equation \(\mathbf{F} = P\mathbf{i} + Q\mathbf{j} + R\mathbf{k}\text{.}\) We use the definition of the line integral of \(\mathbf{F}\) along the curve \(C\) to expand the expression into coordinate components:
    🔗
    \begin{align*} \int_C \mathbf{F} \cdot d\mathbf{r} \amp = \int^b_a \mathbf{F}(\mathbf{r}(t)) \cdot \mathbf{r}'(t)\,dt\\ \amp = \int^b_a (P\mathbf{i} + Q\mathbf{j} + R\mathbf{k}) \cdot (x'(t)\mathbf{i} + y'(t)\mathbf{j} + z'(t)\mathbf{k})\,dt\\ \amp = \int^b_a \left[ Px'(t) + Qy'(t) + Rz'(t) \right]\,dt. \end{align*}

    Theorem 5.27. Equivalence of Vector and Coordinate Form.

    The last integral matches the definition of a line integral with respect to coordinate variables. Therefore, we establish the following equivalence:
    \begin{equation*} \int_C \mathbf{F} \cdot d\mathbf{r} = \int_C P\,dx + Q\,dy + R\,dz, \quad \text{where } \mathbf{F} = P\mathbf{i} + Q\mathbf{j} + R\mathbf{k}. \end{equation*}
    🔗
    🔗

    Proof.
    Let the smooth space curve \(C\) be parameterized by the vector function \(\mathbf{r}(t) = \langle x(t), y(t), z(t) \rangle\) for \(a \leq t \leq b\text{.}\) The derivative of \(\mathbf{r}(t)\) with respect to the parameter \(t\) provides the tangent velocity vector:
    \begin{equation*} \mathbf{r}'(t) = \frac{d\mathbf{r}}{dt} = \left\langle \frac{dx}{dt}, \frac{dy}{dt}, \frac{dz}{dt} \right\rangle = x'(t)\mathbf{i} + y'(t)\mathbf{j} + z'(t)\mathbf{k}. \end{equation*}
    🔗
    By definition, the line integral of a continuous vector field \(\mathbf{F} = P\mathbf{i} + Q\mathbf{j} + R\mathbf{k}\) along \(C\) is evaluated as:
    \begin{equation*} \int_C \mathbf{F} \cdot d\mathbf{r} = \int^b_a \mathbf{F}(\mathbf{r}(t)) \cdot \mathbf{r}'(t)\,dt. \end{equation*}
    Substituting the component forms of both \(\mathbf{F}\) and \(\mathbf{r}'(t)\) into the dot product yields:
    \begin{align*} \int_C \mathbf{F} \cdot d\mathbf{r} \amp = \int^b_a \langle P, Q, R \rangle \cdot \langle x'(t), y'(t), z'(t) \rangle\,dt\\ \amp = \int^b_a \left[ P(x(t),y(t),z(t))x'(t) + Q(x(t),y(t),z(t))y'(t) + R(x(t),y(t),z(t))z'(t) \right]\,dt. \end{align*}
    🔗
    We can distribute the integral across the addition operations. Separating this into three distinct single integrals gives:
    \begin{equation*} = \int^b_a P(x(t),y(t),z(t))x'(t)\,dt + \int^b_a Q(x(t),y(t),z(t))y'(t)\,dt + \int^b_a R(x(t),y(t),z(t))z'(t)\,dt. \end{equation*}
    🔗
    Recall from the definition of line integrals with respect to coordinate variables that the differentials \(dx\text{,}\) \(dy\text{,}\) and \(dz\) are defined parametrically as:
    \begin{equation*} dx = x'(t)\,dt, \quad dy = y'(t)\,dt, \quad dz = z'(t)\,dt. \end{equation*}
    Replacing these differential terms back into the single integrals transforms the parameter space representations into their corresponding coordinate line integrals along \(C\text{:}\)
    \begin{equation*} = \int_C P(x,y,z)\,dx + \int_C Q(x,y,z)\,dy + \int_C R(x,y,z)\,dz. \end{equation*}
    🔗
    Combining these three adjacent coordinate integrals under a single integral sign produces the standard shorthand notation:
    \begin{equation*} \int_C \mathbf{F} \cdot d\mathbf{r} = \int_C P\,dx + Q\,dy + R\,dz. \end{equation*}
