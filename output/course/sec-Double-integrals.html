    Section 4.1 Double Integrals

    In this section, we will explore how double integrals can be used to calculate areas and volumes of regions in two-dimensional and three-dimensional space.
    🔗
    We consider a function \(f(x,y)\text{,}\) with two variables, defined on a rectangular region \(R\text{,}\)
    🔗
    \begin{equation*} R : \quad a \leq x \leq b, \quad c \leq y \leq d. \end{equation*}
    Subdivide \(R\) into small rectangles by using parallel lines to the \(x\)- and \(y\)-axis. The lines divide \(R\) into \(n\) rectangular pieces, where the number of such pieces \(n\) gets large as the width and height of each piece gets small. These rectangles form a partition of \(R\text{.}\) A small rectangular piece of width \(\Delta x\) and height \(\Delta y\) has area \(\Delta A=\Delta x \Delta y\text{.}\) If we number the small pieces partitioning \(R\) in some order, then their areas are given by numbers \(\Delta A_1, \Delta A_2, \dots, \Delta A_n\text{,}\) where \(\Delta A_k\) is the area of the \(k\)th small rectangle.
    🔗

    Definition 4.1.

    1. Let \(f(x,y)\) be defined on a region \(R\text{,}\) and let \(P=\{R_k\}\) be an inner partition of \(R\text{.}\) A Riemann sum of \(f\) for \(P\) is any sum of the form \(\displaystyle \sum_{k} f(x_k,y_k)\Delta A_k\text{,}\) where \((x_k,y_k)\) is a point in \(R_k\text{,}\) and \(\Delta A_k\) is the area of \(R_k\text{.}\)
      🔗
      🔗
    2. The statement
      \begin{equation*} \lim_{|P|\to 0} \sum_{k} f(x_k,y_k)\Delta A_k=L \end{equation*}
      means that for every \(\epsilon \gt 0\) there is a \(\delta \gt 0\) such that if \(P=\{R_k\}\) is an inner partition of \(R\) with \(\|P\| \lt \delta\text{,}\) then
      \begin{equation*} \left| \sum_{k} f(x_k,y_k)\Delta A_k-L \right| \lt \epsilon \end{equation*}
      for every choice of \((x_k,y_k)\) in \(R_k\text{.}\)
      🔗
      🔗
    3. The double integral of \(f\) over \(R\), denoted by \(\iint_R f(x,y)\,dA\text{,}\) is
      \begin{equation*} \iint_R f(x,y)\,dA = \lim_{|P|\to 0} \sum_{k} f(x_k,y_k)\Delta A_k, \end{equation*}
      if the limit exists.
      🔗
      🔗
    4. Let \(f\) be a continuous function of two variables such that \(f(x,y) \geq 0\) for every \((x,y)\) in a region \(R\text{.}\) The volume \(V\) of the solid that lies under the graph of \(z=f(x,y)\) over \(R\) is
      \begin{equation*} V = \iint_R f(x,y)\,dA. \end{equation*}
      🔗
      🔗
    🔗

    Example 4.2.

    Calculate the volume under the plane \(z=4-x-y\) over the rectangular region \(R: 0 \leq x \leq 2, \quad 0 \leq y \leq 1\) in the \(xy\)-plane.
    🔗
    Solution.
    The volume can be written as
    \begin{equation*} \int^{x=2}_{x=0} A(x)\,dx, \end{equation*}
    where \(A(x)\) is the cross-sectional area at \(x\text{.}\) For each value of \(x\text{,}\) we may calculate \(A(x)\) as the integral
    \begin{equation*} A(x) = \int^{y=1}_{y=0} (4-x-y)\,dy, \end{equation*}
    that is the area under the curve \(z=4-x-y\) in the plane of the cross-section at \(x\text{.}\)
    \begin{equation*} \text{Volume} = \int^{x=2}_{x=0} A(x)\,dx = \int^{x=2}_{x=0} \left( \int^{y=1}_{y=0} (4-x-y)\,dy \right) dx. \end{equation*}
    🔗
    🔗
    🔗

    Note 4.4.

    The expression, \(\displaystyle \int^{x=2}_{x=0}\int^{y=1}_{y=0} (4-x-y)\,dydx\text{,}\) is called an iterated or repeated integral. The volume is obtained by integrating \(4-x-y\) with respect to \(y\) from \(y=0\) to \(y=1\text{,}\) holding \(x\) fixed, and then integrating the resulting expression in \(x\) with respect to \(x\) from \(x=0\) to \(x=2\text{.}\)
    🔗
    🔗

    Theorem 4.5. Fubini’s Theorem (First Form).

    If \(f(x,y)\) is continuous throughout the rectangular region \(R : a \leq x \leq b, \ c \leq y \leq d,\) then
    \begin{equation*} \iint_R f(x,y)\,dA = \int^d_c \int^b_a f(x,y)\,dxdy = \int^b_a \int^d_c f(x,y)\,dydx. \end{equation*}
    🔗
    🔗

    Example 4.6. Evaluating a Double Integral.

    Calculate \(\iint_R f(x,y)\,dA\) for
    \begin{equation*} f(x,y) = 1-6x^2y \quad \text{for} \quad R : 0 \leq x \leq 2, \ -1 \leq y \leq 1. \end{equation*}
    🔗
    Solution.
    We can integrate in either order using Fubini’s Theorem. Let’s integrate with respect to \(x\) first, and then with respect to \(y\text{:}\)
    🔗
    \begin{align*} \iint_R (1-6x^2y)\,dA \amp = \int_{-1}^{1} \int_{0}^{2} (1-6x^2y)\,dxdy\\ \amp = \int_{-1}^{1} \left[ x - 2x^3y \right]_{x=0}^{x=2} \,dy\\ \amp = \int_{-1}^{1} \left( (2 - 2(8)y) - (0 - 0) \right) \,dy\\ \amp = \int_{-1}^{1} (2 - 16y)\,dy\\ \amp = \left[ 2y - 8y^2 \right]_{-1}^{1}\\ \amp = (2(1) - 8(1)^2) - (2(-1) - 8(-1)^2)\\ \amp = (2 - 8) - (-2 - 8)\\ \amp = -6 - (-10) = 4 \end{align*}
    Alternatively, integrating with respect to \(y\) first yields the same result:
    🔗
    \begin{align*} \int_{0}^{2} \int_{-1}^{1} (1-6x^2y)\,dydx \amp = \int_{0}^{2} \left[ y - 3x^2y^2 \right]_{y=-1}^{y=1} \,dx\\ \amp = \int_{0}^{2} \left( (1 - 3x^2(1)^2) - (-1 - 3x^2(-1)^2) \right) \,dx\\ \amp = \int_{0}^{2} (1 - 3x^2 + 1 + 3x^2)\,dx\\ \amp = \int_{0}^{2} 2\,dx\\ \amp = \left[ 2x \right]_{0}^{2} = 4 \end{align*}
    🔗
    🔗

    Example 4.8.

    Calculate \(\iint_R f(x,y)\,dA\) for
    \begin{equation*} f(x,y) = 2y\cos x \quad \text{for} \quad R : \pi/6 \leq x \leq y^2, \ 1 \leq y \leq 3. \end{equation*}
    🔗
    Solution.
    Because the limits for \(x\) depend on \(y\text{,}\) we must set this up as a Type II region and integrate with respect to \(x\) first:
    🔗
    \begin{align*} \iint_R 2y\cos x\,dA \amp = \int_{1}^{3} \int_{\pi/6}^{y^2} 2y\cos x \,dxdy\\ \amp = \int_{1}^{3} \left[ 2y\sin x \right]_{x=\pi/6}^{x=y^2} \,dy\\ \amp = \int_{1}^{3} \left( 2y\sin(y^2) - 2y\sin\left(\frac{\pi}{6}\right) \right) \,dy \end{align*}
    Since \(\sin(\pi/6) = \frac{1}{2}\text{,}\) the expression simplifies to:
    🔗
    \begin{align*} \amp = \int_{1}^{3} \left( 2y\sin(y^2) - y \right) \,dy \end{align*}
    We integrate each term. For the first term, we can use \(u\)-substitution with \(u = y^2\) and \(du = 2y\,dy\text{:}\)
    🔗
    \begin{align*} \amp = \left[ -\cos(y^2) - \frac{1}{2}y^2 \right]_{1}^{3}\\ \amp = \left( -\cos(9) - \frac{1}{2}(9) \right) - \left( -\cos(1) - \frac{1}{2}(1) \right)\\ \amp = \cos(1) - \cos(9) - \frac{9}{2} + \frac{1}{2}\\ \amp = \cos(1) - \cos(9) - 4 \end{align*}
    Thus, the exact value of the double integral is \(\cos(1) - \cos(9) - 4\text{.}\)
    🔗
    🔗
    🔗

    Theorem 4.10. Fubini’s Theorem (Stronger Form).

    Let \(f(x,y)\) be continuous on a region \(R\text{.}\)
    🔗
    1. If \(R\) is defined by \(a \leq x \leq b, \ g_1(x) \leq y \leq g_2(x),\) with \(g_1\) and \(g_2\) continuous on \([a,b],\) then
      \begin{equation*} \iint_R f(x,y)\,dA = \int^b_a \int^{g_2(x)}_{g_1(x)} f(x,y)\,dydx. \end{equation*}
      🔗
      🔗
    2. If \(R\) is defined by \(c \leq y \leq d, \ h_1(y) \leq x \leq h_2(y),\) with \(h_1\) and \(h_2\) continuous on \([c,d],\) then
      \begin{equation*} \iint_R f(x,y)\,dA = \int^d_c \int^{h_2(y)}_{h_1(y)} f(x,y)\,dxdy. \end{equation*}
      🔗
      🔗
    🔗

    Example 4.11.

    Let \(R\) be the region in the \(xy\)-plane bounded by the graphs of \(y=x^2\) and \(y=2x\text{.}\) Evaluate \(\iint_R (x^3+4y)\,dA\text{.}\)
    🔗
    Solution.
    First, we find the points of intersection of the bounding curves by setting them equal to each other:
    \begin{equation*} x^2 = 2x \implies x^2 - 2x = 0 \implies x(x-2) = 0 \end{equation*}
    This gives intersection points at \(x = 0\) and \(x = 2\text{.}\) On the interval \([0,2]\text{,}\) the line \(y = 2x\) lies above the parabola \(y = x^2\text{.}\) Thus, our region is described by \(0 \leq x \leq 2\) and \(x^2 \leq y \leq 2x\text{.}\)
    🔗
    Using Fubini’s stronger form, we integrate with respect to \(y\) first:
    🔗
    \begin{align*} \iint_R (x^3+4y)\,dA \amp = \int_{0}^{2} \int_{x^2}^{2x} (x^3+4y)\,dydx\\ \amp = \int_{0}^{2} \left[ x^3y + 2y^2 \right]_{y=x^2}^{y=2x} \,dx\\ \amp = \int_{0}^{2} \left( \left(x^3(2x) + 2(2x)^2\right) - \left(x^3(x^2) + 2(x^2)^2\right) \right) \,dx\\ \amp = \int_{0}^{2} \left( 2x^4 + 8x^2 - x^5 - 2x^4 \right) \,dx\\ \amp = \int_{0}^{2} \left( 8x^2 - x^5 \right) \,dx\\ \amp = \left[ \frac{8}{3}x^3 - \frac{1}{6}x^6 \right]_{0}^{2}\\ \amp = \left( \frac{8}{3}(8) - \frac{1}{6}(64) \right) - (0 - 0)\\ \amp = \frac{64}{3} - \frac{32}{3} = \frac{32}{3} \end{align*}
    The value of the double integral over the bounded region is \(\frac{32}{3}\text{.}\)
    🔗
    🔗

    Checkpoint 4.13.

    Given \(\displaystyle \int^4_0\int^2_{\sqrt{y}}y\cos(x^5) \, dxdy\text{,}\) reverse the order of integration and evaluate the resulting integral.
    🔗
    Solution.
    Step 1: Analyze the Original Domain of Integration
    🔗
    The original limits correspond to a Type II integration order (\(dxdy\)). The boundaries for the region \(R\) are given by:
    \begin{equation*} 0 \leq y \leq 4 \quad \text{and} \quad \sqrt{y} \leq x \leq 2 \end{equation*}
    The inner lower boundary curve \(x = \sqrt{y}\) is equivalent to the parabola \(y = x^2\) for \(x \geq 0\text{.}\) The upper boundary for \(x\) is the vertical line \(x = 2\text{.}\) These boundaries intersect at the points \((0,0)\) and \((2,4)\text{.}\)
    🔗
    Step 2: Reverse the Order of Integration to Type I (\(dydx\))
    🔗
    To transition to Type I format, we look at the region sweeping from left to right along the constant boundaries of the \(x\)-axis. The region spans from \(x = 0\) to \(x = 2\text{.}\) For any fixed value of \(x\text{,}\) the vertical strip starts at the bottom boundary line \(y = 0\) and moves up to touch the upper parabolic curve boundary \(y = x^2\text{.}\)
    🔗
    The rewritten boundary inequalities become:
    \begin{equation*} 0 \leq x \leq 2 \quad \text{and} \quad 0 \leq y \leq x^2 \end{equation*}
    🔗
    Step 3: Evaluate the New Iterated Integral
    🔗
    Setting up the reversed iterated integral gives:
    \begin{equation*} \int_{0}^{2} \int_{0}^{x^2} y\cos(x^5) \, dy \, dx \end{equation*}
    First, we integrate the inner expression with respect to \(y\text{,}\) treating \(\cos(x^5)\) as a constant factor:
    🔗
    \begin{align*} \int_{0}^{2} \left[ \frac{1}{2}y^2 \cos(x^5) \right]_{y=0}^{y=x^2} \, dx \amp = \int_{0}^{2} \left( \frac{1}{2}(x^2)^2 \cos(x^5) - 0 \right) \, dx\\ \amp = \int_{0}^{2} \frac{1}{2}x^4 \cos(x^5) \, dx \end{align*}
    Now, we evaluate the outer single integral with respect to \(x\text{.}\) We can use \(u\)-substitution by setting:
    \begin{equation*} u = x^5 \implies du = 5x^4 \, dx \implies \frac{1}{5}\,du = x^4 \, dx \end{equation*}
    We transform the integration boundaries accordingly:
    🔗
    • When \(x = 0\text{:}\) \(u = 0^5 = 0\)
      🔗
      🔗
    • When \(x = 2\text{:}\) \(u = 2^5 = 32\)
      🔗
      🔗
    Substituting these components back into the integral equation yields:
    🔗
    \begin{align*} \int_{0}^{32} \frac{1}{2} \cdot \left(\frac{1}{5} \cos(u)\right) \, du \amp = \frac{1}{10} \int_{0}^{32} \cos(u) \, du\\ \amp = \frac{1}{10} \Big[ \sin(u) \Big]_{0}^{32}\\ \amp = \frac{1}{10} \big( \sin(32) - \sin(0) \big)\\ \amp = \frac{\sin(32)}{10} \end{align*}
    Thus, the exact value of the reversed integration evaluation is \(\frac{\sin(32)}{10}\text{.}\)
    🔗
    🔗

    Example 4.16. Finding Volume.

    Find the volume of a prism whose base is the triangle in the \(xy\)-plane bounded by the \(x\)-axis and the lines \(y=x\) and \(x=1\text{,}\) and whose top lies in the plane
    \begin{equation*} z = f(x,y) = 3-x-y. \end{equation*}
    🔗
    Solution.
    Step 1: Identify the Base Region \(R\)
    🔗
    The triangular base region \(R\) in the \(xy\)-plane is bounded by the lines \(y = 0\) (the \(x\)-axis), \(y = x\text{,}\) and \(x = 1\text{.}\) These lines intersect at the coordinates \((0,0)\text{,}\) \((1,0)\text{,}\) and \((1,1)\text{.}\) Treating this as a Type I region, the limits of integration are:
    \begin{equation*} 0 \leq x \leq 1 \quad \text{and} \quad 0 \leq y \leq x \end{equation*}
    🔗
    Step 2: Set Up and Evaluate the Volume Integral
    🔗
    The volume \(V\) under the surface \(z = 3-x-y\) over the region \(R\) is calculated using a double integral:
    \begin{equation*} V = \iint_R (3-x-y) \, dA = \int_{0}^{1} \int_{0}^{x} (3-x-y) \, dy \, dx \end{equation*}
    First, we compute the inner integral with respect to \(y\text{,}\) holding \(x\) constant:
    🔗
    \begin{align*} \int_{0}^{x} (3-x-y) \, dy \amp = \left[ 3y - xy - \frac{1}{2}y^2 \right]_{y=0}^{y=x}\\ \amp = \left( 3(x) - x(x) - \frac{1}{2}(x)^2 \right) - 0\\ \amp = 3x - x^2 - \frac{1}{2}x^2\\ \amp = 3x - \frac{3}{2}x^2 \end{align*}
    Next, we substitute this result into the outer integral and integrate with respect to \(x\text{:}\)
    🔗
    \begin{align*} V \amp = \int_{0}^{1} \left( 3x - \frac{3}{2}x^2 \right) \, dx\\ \amp = \left[ \frac{3}{2}x^2 - \frac{1}{2}x^3 \right]_{0}^{1}\\ \amp = \left( \frac{3}{2}(1)^2 - \frac{1}{2}(1)^3 \right) - 0\\ \amp = \frac{3}{2} - \frac{1}{2} = 1 \end{align*}
    The total volume of the prism is exactly \(1\text{.}\)
    🔗
    🔗
    🔗

    Checkpoint 4.18.

    1. Evaluate the double integral over the given region
      \begin{equation*} \iint_R e^{x-y}\,dA, \qquad R: 0\leq x \leq \ln 2, \ 0\leq y\leq \ln 2 \end{equation*}
      🔗
      🔗
    2. Find the volume of the region bounded above by the paraboloid \(z=x^2+y^2\) and below by the square \(R:-1\leq x\leq 1, \ -1\leq y \leq 1\text{.}\)
      🔗
      🔗
    Solution.
    1. Solution to Part 1:
      🔗
      Since the limits of integration for both variables are constants, this is a simple rectangular region. We can use Fubini’s Theorem to write it as an iterated integral:
      \begin{equation*} \iint_R e^{x-y}\,dA = \int_{0}^{\ln 2} \int_{0}^{\ln 2} e^x e^{-y} \,dx\,dy \end{equation*}
      Because the integrand factors into a function of \(x\) multiplied by a function of \(y\text{,}\) we can split the double integral into the product of two separate single integrals:
      \begin{equation*} = \left( \int_{0}^{\ln 2} e^x \,dx \right) \left( \int_{0}^{\ln 2} e^{-y} \,dy \right) \end{equation*}
      🔗
      Evaluating each single integral independently:
      \begin{equation*} \int_{0}^{\ln 2} e^x \,dx = \left[ e^x \right]_{0}^{\ln 2} = e^{\ln 2} - e^0 = 2 - 1 = 1 \end{equation*}
      \begin{equation*} \int_{0}^{\ln 2} e^{-y} \,dy = \left[ -e^{-y} \right]_{0}^{\ln 2} = -e^{-\ln 2} - (-e^0) = -\frac{1}{2} + 1 = \frac{1}{2} \end{equation*}
      🔗
      Multiplying the two results together gives:
      \begin{equation*} \iint_R e^{x-y}\,dA = 1 \cdot \frac{1}{2} = \frac{1}{2} \end{equation*}
      🔗
      🔗
    2. Solution to Part 2:
      🔗
      The volume \(V\) under the surface \(z = x^2 + y^2\) over the square region \(R\) is given by the double integral:
      \begin{equation*} V = \iint_R (x^2 + y^2)\,dA = \int_{-1}^{1} \int_{-1}^{1} (x^2 + y^2) \,dy\,dx \end{equation*}
      🔗
      First, we compute the inner integral with respect to \(y\text{:}\)
      \begin{align*} \int_{-1}^{1} (x^2 + y^2) \,dy \amp = \left[ x^2y + \frac{1}{3}y^3 \right]_{y=-1}^{y=1}\\ \amp = \left( x^2(1) + \frac{1}{3}(1)^3 \right) - \left( x^2(-1) + \frac{1}{3}(-1)^3 \right)\\ \amp = \left( x^2 + \frac{1}{3} \right) - \left( -x^2 - \frac{1}{3} \right)\\ \amp = 2x^2 + \frac{2}{3} \end{align*}
      🔗
      Now, we integrate this result with respect to \(x\) across the outer boundaries:
      \begin{align*} V \amp = \int_{-1}^{1} \left( 2x^2 + \frac{2}{3} \right) \,dx\\ \amp = \left[ \frac{2}{3}x^3 + \frac{2}{3}x \right]_{-1}^{1}\\ \amp = \left( \frac{2}{3}(1)^3 + \frac{2}{3}(1) \right) - \left( \frac{2}{3}(-1)^3 + \frac{2}{3}(-1) \right)\\ \amp = \left( \frac{2}{3} + \frac{2}{3} \right) - \left( -\frac{2}{3} - \frac{2}{3} \right)\\ \amp = \frac{4}{3} - \left( -\frac{4}{3} \right) = \frac{8}{3} \end{align*}
      The total volume bounded by the paraboloid over the square base is \(\frac{8}{3}\text{.}\)
