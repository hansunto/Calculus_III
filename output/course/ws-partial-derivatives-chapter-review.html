    Worksheet Worksheet 03: Partial Derivatives

    1.

    Show that the following limit does not exist:
    \begin{equation*} \lim_{\substack{(x,y)\to (0,0) \\ y \neq x^2}} \frac{y}{x^2-y} \end{equation*}
    🔗
    Solution.
    Let \(f(x,y) = \frac{y}{x^2-y}\text{.}\) We test the behavior of this function along different paths approaching the origin \((0,0)\text{.}\)
    🔗
    First, let us approach the origin along the linear path \(y = mx\) where \(m \neq 0\text{:}\)
    \begin{equation*} \lim_{x \to 0} \frac{mx}{x^2 - mx} = \lim_{x \to 0} \frac{m}{x - m} = \frac{m}{-m} = -1 \end{equation*}
    Along every straight line of non-zero slope, the function approaches \(-1\text{.}\) If we approach along the line \(y = 0\) (the \(x\)-axis), the function is identically \(0\text{,}\) giving a path limit of \(0\text{.}\)
    🔗
    Since approaching along \(y = 0\) yields \(0\text{,}\) while approaching along \(y = mx\) yields \(-1\text{,}\) the limits along two distinct paths are different. By the two-path rule, the limit does not exist.
    🔗
    🔗
    🔗

    2.

    Find the limit:
    \begin{equation*} \lim_{(x,y)\to (0,0)} \frac{x^2-y^2}{x^2+y^2} \end{equation*}
    🔗
    Solution.
    Let \(f(x,y) = \frac{x^2-y^2}{x^2+y^2}\text{.}\) We test the limit along the coordinate axes:
    • Along the \(x\)-axis (\(y=0\) for \(x \neq 0\)): \(f(x,0) = \frac{x^2-0}{x^2+0} = 1\text{.}\) Thus, the limit along this path is \(1\text{.}\)
      🔗
      🔗
    • Along the \(y\)-axis (\(x=0\) for \(y \neq 0\)): \(f(0,y) = \frac{0-y^2}{0+y^2} = -1\text{.}\) Thus, the limit along this path is \(-1\text{.}\)
      🔗
      🔗
    Since the function approaches two different values along two different geometric paths, the limit does not exist.
    🔗
    🔗
    🔗

    3.

    Find the extreme values of \(f(x,y,z)=x-y+z\) on the unit sphere \(x^2+y^2+z^2=1\text{.}\)
    🔗
    Solution.
    We use the method of Lagrange Multipliers with constraint function \(g(x,y,z) = x^2+y^2+z^2 - 1 = 0\text{.}\) We set \(\nabla f = \lambda \nabla g\text{:}\)
    \begin{equation*} \langle 1, -1, 1 \rangle = \lambda \langle 2x, 2y, 2z \rangle \end{equation*}
    This gives the system: \(1 = 2\lambda x\text{,}\) \(-1 = 2\lambda y\text{,}\) and \(1 = 2\lambda z\text{.}\) Isolating the variables, we find:
    \begin{equation*} x = \frac{1}{2\lambda}, \quad y = -\frac{1}{2\lambda}, \quad z = \frac{1}{2\lambda} \end{equation*}
    Substituting these into the constraint sphere equation yields:
    \begin{equation*} \left(\frac{1}{2\lambda}\right)^2 + \left(-\frac{1}{2\lambda}\right)^2 + \left(\frac{1}{2\lambda}\right)^2 = 1 \implies \frac{3}{4\lambda^2} = 1 \implies \lambda = \pm \frac{\sqrt{3}}{2} \end{equation*}
    This produces two critical points: \(\left(\frac{1}{\sqrt{3}}, -\frac{1}{\sqrt{3}}, \frac{1}{\sqrt{3}}\right)\) and \(\left(-\frac{1}{\sqrt{3}}, \frac{1}{\sqrt{3}}, -\frac{1}{\sqrt{3}}\right)\text{.}\) Evaluating \(f\text{:}\)
    \begin{equation*} f_{\text{max}} = \frac{1}{\sqrt{3}} - \left(-\frac{1}{\sqrt{3}}\right) + \frac{1}{\sqrt{3}} = \frac{3}{\sqrt{3}} = \sqrt{3} \end{equation*}
    \begin{equation*} f_{\text{min}} = -\frac{1}{\sqrt{3}} - \frac{1}{\sqrt{3}} - \frac{1}{\sqrt{3}} = -\sqrt{3} \end{equation*}

    4.

    Find \(\frac{\partial w}{\partial r}\) and \(\frac{\partial w}{\partial s}\) when \(r=\pi\) and \(s=0\text{,}\) if:
    \begin{equation*} w=\sin(2x-y), \quad x=r+\sin(s), \quad y=rs \end{equation*}
    🔗
    Solution.
    When \(r=\pi\) and \(s=0\text{,}\) we find the intermediate coordinate states: \(x = \pi + \sin(0) = \pi\) and \(y = \pi(0) = 0\text{.}\) The first partial derivatives of \(w\) are:
    \begin{equation*} \frac{\partial w}{\partial x} = 2\cos(2x-y) = 2\cos(2\pi) = 2, \quad \frac{\partial w}{\partial y} = -\cos(2x-y) = -\cos(2\pi) = -1 \end{equation*}
    Using the Multivariate Chain Rule for the parameter \(r\text{:}\)
    \begin{equation*} \frac{\partial w}{\partial r} = \frac{\partial w}{\partial x}\frac{\partial x}{\partial r} + \frac{\partial w}{\partial y}\frac{\partial y}{\partial r} = (2)(1) + (-1)(s) = 2 - s \end{equation*}
    At \(s=0\text{,}\) \(\frac{\partial w}{\partial r} = 2\text{.}\)
    🔗
    Using the Chain Rule for the parameter \(s\text{:}\)
    \begin{equation*} \frac{\partial w}{\partial s} = \frac{\partial w}{\partial x}\frac{\partial x}{\partial s} + \frac{\partial w}{\partial y}\frac{\partial y}{\partial s} = (2)(\cos s) + (-1)(r) = 2\cos s - r \end{equation*}
    At \(r=\pi\) and \(s=0\text{,}\) \(\frac{\partial w}{\partial s} = 2\cos(0) - \pi = 2 - \pi\text{.}\)
    🔗

    5.

    Find the value of the derivative of \(f(x,y,z)=xy+yz+xz\) with respect to \(t\) along the parameter path \(x=\cos(t)\text{,}\) \(y=\sin(t)\text{,}\) and \(z=\cos(2t)\) at \(t=1\text{.}\)
    🔗
    Solution.
    By applying the Multivariate Chain Rule Case I, the total derivative with respect to time is:
    \begin{equation*} \frac{df}{dt} = (y+z)(-\sin t) + (x+z)(\cos t) + (y+x)(-2\sin(2t)) \end{equation*}
    Substituting \(x=\cos(1)\text{,}\) \(y=\sin(1)\text{,}\) and \(z=\cos(2)\) directly into this expression evaluated at \(t=1\) yields:
    \begin{align*} \frac{df}{dt} \amp = -\sin(1)[\sin(1) + \cos(2)] + \cos(1)[\cos(1) + \cos(2)] - 2\sin(2)[\sin(1) + \cos(1)]\\ \amp = \cos^2(1) - \sin^2(1) + \cos(2)[\cos(1) - \sin(1)] - 2\sin(2)[\sin(1) + \cos(1)]\\ \amp = \cos(2) + \cos(2)[\cos(1) - \sin(1)] - 2\sin(2)[\sin(1) + \cos(1)] \end{align*}
    Expressed as a clean single decimal value approximation, \(\frac{df}{dt} \approx -3.0741\text{.}\)
    🔗
    🔗

    6.

    Find the directions in which \(f(x,y,z)=\ln(2x+3y+6z)\) increases and decreases most rapidly at the point \(P_0(-1,-1,1)\text{.}\) Then, find the directional derivative of \(f\) at \(P_0\) in the direction of the vector \(\mathbf{v}=2\mathbf{i}+3\mathbf{j}+6\mathbf{k}\text{.}\)
    🔗
    Solution.
    We find the first-order partial derivatives forming the components of the gradient vector:
    \begin{equation*} \nabla f = \left\langle \frac{2}{2x+3y+6z}, \, \frac{3}{2x+3y+6z}, \, \frac{6}{2x+3y+6z} \right\rangle \end{equation*}
    At \(P_0(-1,-1,1)\text{,}\) the inner sum denominator expression evaluates to \(2(-1)+3(-1)+6(1) = 1\text{.}\) Thus, the gradient vector is \(\nabla f(-1,-1,1) = \langle 2, 3, 6 \rangle\text{.}\)
    🔗
    • The function increases most rapidly in the direction of the gradient: \(\langle 2, 3, 6 \rangle\text{.}\)
      🔗
      🔗
    • The function decreases most rapidly in the opposite direction: \(\langle -2, -3, -6 \rangle\text{.}\)
      🔗
      🔗
    The direction vector \(\mathbf{v} = \langle 2,3,6 \rangle\) perfectly matches the direction of the gradient. Normalizing \(\mathbf{v}\) via its magnitude \(\|\mathbf{v}\| = \sqrt{2^2+3^2+6^2} = \sqrt{49}=7\) gives the unit vector \(\mathbf{u} = \frac{1}{7}\langle 2,3,6 \rangle\text{.}\) The directional derivative is the magnitude of the gradient vector:
    \begin{equation*} D_{\mathbf{u}} f(-1,-1,1) = \|\nabla f\| = \sqrt{2^2+3^2+6^2} = 7 \end{equation*}
    🔗
    🔗
    🔗

    7.

    Find the directional derivative of \(f(x,y,z)=xyz\) in the direction of the velocity vector of the helix:
    \begin{equation*} \mathbf{r}(t)=(\cos 3t)\mathbf{i}+(\sin 3t)\mathbf{j}+3t\mathbf{k} \quad \text{at} \quad t=\frac{\pi}{3} \end{equation*}
    🔗
    Solution.
    First, we compute the velocity vector \(\mathbf{v}(t) = \mathbf{r}'(t) = \langle -3\sin(3t), 3\cos(3t), 3 \rangle\text{.}\) Evaluating this at \(t = \pi/3\text{:}\)
    \begin{equation*} \mathbf{v}(\pi/3) = \langle -3\sin(\pi), 3\cos(\pi), 3 \rangle = \langle 0, -3, 3 \rangle \end{equation*}
    Normalizing this direction vector yields our unit vector \(\mathbf{u}\text{:}\)
    \begin{equation*} \|\mathbf{v}\| = \sqrt{0^2 + (-3)^2 + 3^2} = \sqrt{18} = 3\sqrt{2} \implies \mathbf{u} = \left\langle 0, -\frac{1}{\sqrt{2}}, \frac{1}{\sqrt{2}} \right\rangle \end{equation*}
    The coordinate point on the helix at \(t = \pi/3\) is \(x = \cos(\pi) = -1\text{,}\) \(y = \sin(\pi) = 0\text{,}\) and \(z = 3(\pi/3) = \pi\text{.}\) The gradient of \(f(x,y,z)=xyz\) is \(\nabla f = \langle yz, xz, xy \rangle\text{.}\) Evaluating the gradient at \((-1,0,\pi)\) yields \(\nabla f = \langle 0, -\pi, 0 \rangle\text{.}\)
    🔗
    Finally, the directional derivative is computed via the vector dot product:
    \begin{equation*} D_{\mathbf{u}} f = \langle 0, -\pi, 0 \rangle \cdot \left\langle 0, -\frac{1}{\sqrt{2}}, \frac{1}{\sqrt{2}} \right\rangle = \frac{\pi}{\sqrt{2}} = \frac{\pi\sqrt{2}}{2} \approx 2.2214 \end{equation*}
    🔗
    🔗
    🔗

    8.

    Find an equation for the plane tangent to the level surface \(x^2-y-5z=0\) at the point \(P_0(2,-1,1)\text{.}\) Additionally, find parametric equations for the normal line to the surface passing through \(P_0\text{.}\)
    🔗
    Solution.
    Let \(F(x,y,z) = x^2-y-5z\text{.}\) We find the components of the normal vector by evaluating the gradient \(\nabla F = \langle 2x, -1, -5 \rangle\) at \(P_0(2,-1,1)\text{:}\)
    \begin{equation*} \nabla F(2,-1,1) = \langle 2(2), -1, -5 \rangle = \langle 4, -1, -5 \rangle \end{equation*}
    Using the point-normal equation, the equation of the tangent plane is:
    \begin{equation*} 4(x-2) - 1(y - (-1)) - 5(z-1) = 0 \implies 4x - y - 5z = 4 \end{equation*}
    🔗
    The normal line passes through \((2,-1,1)\) and moves parallel to the direction of the normal vector \(\langle 4,-1,-5 \rangle\text{.}\) Thus, its parametric equations are:
    \begin{equation*} x = 2 + 4t, \quad y = -1 - t, \quad z = 1 - 5t \quad \text{for } t \in \mathbb{R} \end{equation*}
    🔗
    🔗
    🔗

    9.

    Find an equation for the plane tangent to the explicit surface \(z=\ln(x^2+y^2)\) at the point \((0,1,0)\text{.}\)
    🔗
    Solution.
    We rewrite the equation as a level surface function: \(F(x,y,z) = \ln(x^2+y^2) - z = 0\text{.}\) We find the gradient vector by computing the first-order partial derivatives:
    \begin{equation*} F_x = \frac{2x}{x^2+y^2}, \quad F_y = \frac{2y}{x^2+y^2}, \quad F_z = -1 \end{equation*}
    Evaluating these partial derivatives at the given point \((0,1,0)\) yields:
    \begin{equation*} F_x(0,1,0) = \frac{0}{1} = 0, \quad F_y(0,1,0) = \frac{2(1)}{1} = 2, \quad F_z(0,1,0) = -1 \end{equation*}
    This gives the normal vector \(\langle 0, 2, -1 \rangle\text{.}\) Using the point-normal equation, the equation of the tangent plane is:
    \begin{equation*} 0(x-0) + 2(y-1) - 1(z-0) = 0 \implies 2y - z = 2 \end{equation*}
    🔗
    🔗
    🔗

    10.

    Find an equation for the line tangent and normal to the level curve \(y-\sin(x)=1\) at the point \(P_0(\pi,1)\text{.}\)
    🔗
    Solution.
    Let our level curve function be defined as \(f(x,y) = y - \sin(x)\text{.}\) The gradient vector is orthogonal to the level curve and is given by:
    \begin{equation*} \nabla f = \langle -\cos x, \, 1 \rangle \implies \nabla f(\pi,1) = \langle -\cos(\pi), \, 1 \rangle = \langle 1, 1 \rangle \end{equation*}
    The gradient vector \(\langle 1,1 \rangle\) serves as the normal vector to the tangent line. Thus, the equation of the tangent line passing through \((\pi,1)\) is:
    \begin{equation*} 1(x-\pi) + 1(y-1) = 0 \implies x + y = \pi + 1 \end{equation*}
    🔗
    The normal line moves parallel to the gradient vector direction \(\langle 1,1 \rangle\text{.}\) Thus, its linear equation is found by using an orthogonal slope direction:
    \begin{equation*} 1(x-\pi) - 1(y-1) = 0 \implies x - y = \pi - 1 \end{equation*}
    🔗
    🔗
    🔗

    11.

    Find parametric equations for the line that is tangent to the curve of intersection of the surfaces \(x^2+2y+2z=4\) and \(y=1\) at the point \((1,1,1/2)\text{.}\)
    🔗
    Solution.
    The direction of the line tangent to the intersection curve of two surfaces is perpendicular to the normal vectors of both surfaces simultaneously. We can find this direction vector \(\mathbf{v}\) by computing the cross product of their gradients. Let \(F(x,y,z) = x^2+2y+2z-4\) and \(G(x,y,z) = y-1\text{:}\)
    \begin{equation*} \nabla F = \langle 2x, 2, 2 \rangle \implies \nabla F(1,1,1/2) = \langle 2, 2, 2 \rangle \end{equation*}
    \begin{equation*} \nabla G = \langle 0, 1, 0 \rangle \implies \nabla G(1,1,1/2) = \langle 0, 1, 0 \rangle \end{equation*}
    Computing the vector cross product:
    \begin{equation*} \mathbf{v} = \nabla F \times \nabla G = \begin{vmatrix} \mathbf{i} \amp \mathbf{j} \amp \mathbf{k} \\ 2 \amp 2 \amp 2 \\ 0 \amp 1 \amp 0 \end{vmatrix} = -2\mathbf{i} + 0\mathbf{j} + 2\mathbf{k} = \langle -2, 0, 2 \rangle \end{equation*}
    We can scale this parallel vector to \(\langle -1, 0, 1 \rangle\text{.}\) The tangent line passes through \((1,1,1/2)\) in this direction, yielding the parametric equations:
    \begin{equation*} x = 1 - t, \quad y = 1, \quad z = \frac{1}{2} + t \quad \text{for } t \in \mathbb{R} \end{equation*}
    🔗
    🔗
    🔗

    12.

    Find the local maximum values, local minimum values, and saddle points of the function:
    \begin{equation*} f(x,y)=x^2-xy+y^2+2x+2y-4 \end{equation*}
    Find the function’s value at each of these points.
    🔗
    Solution.
    We first find the critical points by calculating the first-order partial derivatives and setting them equal to zero:
    \begin{equation*} f_x = 2x - y + 2 = 0 \implies y = 2x + 2 \end{equation*}
    \begin{equation*} f_y = -x + 2y + 2 = 0 \end{equation*}
    Substituting \(y = 2x+2\) into the second equation:
    \begin{equation*} -x + 2(2x+2) + 2 = 0 \implies 3x + 6 = 0 \implies x = -2 \end{equation*}
    Using \(x = -2\text{,}\) we find \(y = 2(-2)+2 = -2\text{.}\) This gives a single unique critical point at \((-2,-2)\text{.}\)
    🔗
    To classify this point using the Second Derivatives Test, we compute the second partial derivatives:
    \begin{equation*} f_{xx} = 2, \quad f_{yy} = 2, \quad f_{xy} = -1 \end{equation*}
    We assemble these into the discriminant formula:
    \begin{equation*} D = f_{xx}f_{yy} - (f_{xy})^2 = (2)(2) - (-1)^2 = 4 - 1 = 3 \end{equation*}
    Since \(D > 0\) and \(f_{xx} = 2 > 0\text{,}\) the function has a local minimum at \((-2,-2)\text{.}\) Evaluating the function at this point yields:
    \begin{equation*} f(-2,-2) = (-2)^2 - (-2)(-2) + (-2)^2 + 2(-2) + 2(-2) - 4 = 4 - 4 + 4 - 4 - 4 - 4 = -6 \end{equation*}
    🔗
    🔗
    🔗

    13.

    Find the absolute maximum and absolute minimum values of \(f(x,y)=x^2+xy+y^2-3x+3y\) on the closed triangular region in the first quadrant bounded by the lines \(x=0\text{,}\) \(y=0\text{,}\) and \(x+y=4\text{.}\)
    🔗
    Solution.
    First, we check for critical points in the interior of the triangle:
    \begin{equation*} f_x = 2x + y - 3 = 0 \quad \text{and} \quad f_y = x + 2y + 3 = 0 \end{equation*}
    Solving this linear system yields \(x = 3, y = -3\text{.}\) However, the point \((3,-3)\) lies outside the first quadrant region (\(y \geq 0\)). Thus, there are no valid interior critical points.
    🔗
    Next, we test the three boundary lines of the triangular region:
    1. Along the boundary \(x = 0\) (for \(0 \leq y \leq 4\)): The function simplifies to \(f(0,y) = y^2 + 3y\text{.}\) Its derivative with respect to \(y\) is \(2y+3=0 \implies y=-1.5\) (out of bounds). Evaluating the endpoints: \(f(0,0) = 0\) and \(f(0,4) = 4^2+3(4) = 28\text{.}\)
      🔗
      🔗
    2. Along the boundary \(y = 0\) (for \(0 \leq x \leq 4\)): The function simplifies to \(f(x,0) = x^2 - 3x\text{.}\) Its derivative with respect to \(x\) is \(2x-3=0 \implies x=1.5\text{.}\) Evaluating this critical point and the remaining endpoint: \(f(1.5,0) = -2.25\) and \(f(4,0) = 4^2-3(4) = 4\text{.}\)
      🔗
      🔗
    3. Along the slanted boundary \(y = 4-x\) (for \(0 \leq x \leq 4\)): Substituting this condition gives:
      \begin{equation*} f(x,4-x) = x^2 + x(4-x) + (4-x)^2 - 3x + 3(4-x) = x^2 - 11x + 28 \end{equation*}
      Its derivative is \(2x-11=0 \implies x=5.5\) (out of bounds).
      🔗
      🔗
    Comparing all valid candidates, the absolute maximum value is \(28\) at \((0,4)\text{,}\) and the absolute minimum value is \(-2.25\) at \((1.5,0)\text{.}\)
    🔗
    🔗
    🔗

    14.

    Find the second-order partial derivatives of the function \(f(x,y)=e^x+y\sin(x)\text{.}\)
    🔗
    Solution.
    We begin by computing the first-order partial derivatives of \(f\) with respect to \(x\) and \(y\text{:}\)
    \begin{equation*} f_x = \frac{\partial}{\partial x}(e^x + y\sin x) = e^x + y\cos x \end{equation*}
    \begin{equation*} f_y = \frac{\partial}{\partial y}(e^x + y\sin x) = \sin x \end{equation*}
    🔗
    Next, we differentiate these expressions a second time to find the four second-order partial derivatives:
    \begin{equation*} f_{xx} = \frac{\partial}{\partial x}(e^x + y\cos x) = e^x - y\sin x \end{equation*}
    \begin{equation*} f_{xy} = \frac{\partial}{\partial y}(e^x + y\cos x) = \cos x \end{equation*}
    \begin{equation*} f_{yx} = \frac{\partial}{\partial x}(\sin x) = \cos x \end{equation*}
    \begin{equation*} f_{yy} = \frac{\partial}{\partial y}(\sin x) = 0 \end{equation*}
    Notice that the mixed partial derivatives are equal (\(f_{xy} = f_{yx} = \cos x\)), as guaranteed by Clairaut’s Theorem.
    🔗
    🔗

    15.

    Find the absolute max and min values of \(f(x,y)=x^3+y^3+3x^2-3y^2\) on the square region enclosed by the lines \(x=\pm1\) and \(y=\pm1\text{.}\)
    🔗
    Solution.
    First, we find the critical points in the interior of the region by setting the partial derivatives to zero:
    🔗
    \begin{align*} f_x(x,y) \amp = 3x^2 + 6x = 3x(x+2) = 0 \implies x = 0, \, x = -2\\ f_y(x,y) \amp = 3y^2 - 6y = 3y(y-2) = 0 \implies y = 0, \, y = 2 \end{align*}
    Since our region is bounded by \([-1,1] \times [-1,1]\text{,}\) the values \(x = -2\) and \(y = 2\) lie outside the square. The only interior critical point is \((0,0)\text{.}\) Evaluating the function here yields:
    \begin{equation*} f(0,0) = 0 \end{equation*}
    🔗
    Next, we check the four boundaries of the square region:
    🔗
    • On the right boundary (\(x = 1, -1 \le y \le 1\)): \(g_1(y) = f(1,y) = y^3 - 3y^2 + 4\text{.}\) Setting \(g_1'(y) = 3y^2 - 6y = 0\) gives \(y = 0\text{.}\) We check the critical point and the edge endpoints:
      \begin{equation*} f(1,0) = 4, \quad f(1,-1) = 0, \quad f(1,1) = 2 \end{equation*}
      🔗
      🔗
    • On the left boundary (\(x = -1, -1 \le y \le 1\)): \(g_2(y) = f(-1,y) = y^3 - 3y^2 + 2\text{.}\) The critical point is \(y = 0\text{.}\) Checking these locations gives:
      \begin{equation*} f(-1,0) = 2, \quad f(-1,-1) = -2, \quad f(-1,1) = 0 \end{equation*}
      🔗
      🔗

    16.

    If \(w=\ln(x^2+y^2+2z)\text{,}\) \(x=r+s\text{,}\) \(y=r-s\text{,}\) \(z=2rs\text{,}\) find \(w_r\) and \(w_s\) by the Chain Rule. Then check your answer another way.
    🔗
    Solution.
    Method 1: Chain Rule
    🔗
    First, we find the partial derivatives of \(w\) with respect to its intermediate variables \(x\text{,}\) \(y\text{,}\) and \(z\text{:}\)
    🔗
    \begin{equation*} w_x = \frac{2x}{x^2+y^2+2z}, \quad w_y = \frac{2y}{x^2+y^2+2z}, \quad w_z = \frac{2}{x^2+y^2+2z} \end{equation*}
    Now, we calculate the partial derivatives of the inner components with respect to \(r\) and \(s\text{:}\)
    🔗
    \begin{equation*} x_r = 1, \quad y_r = 1, \quad z_r = 2s \qquad \text{and} \qquad x_s = 1, \quad y_s = -1, \quad z_s = 2r \end{equation*}
    Applying the Chain Rule for \(w_r\text{:}\)
    🔗
    \begin{align*} w_r \amp = w_x x_r + w_y y_r + w_z z_r\\ \amp = \frac{2x(1) + 2y(1) + 2(2s)}{x^2+y^2+2z} = \frac{2(r+s) + 2(r-s) + 4s}{(r+s)^2 + (r-s)^2 + 2(2rs)}\\ \amp = \frac{2r + 2s + 2r - 2s + 4s}{(r^2+2rs+s^2) + (r^2-2rs+s^2) + 4rs} = \frac{4r + 4s}{2r^2 + 4rs + 2s^2}\\ \amp = \frac{4(r+s)}{2(r+s)^2} = \frac{2}{r+s} \end{align*}
    Applying the Chain Rule for \(w_s\text{:}\)
    🔗
    \begin{align*} w_s \amp = w_x x_s + w_y y_s + w_z z_s\\ \amp = \frac{2x(1) + 2y(-1) + 2(2r)}{x^2+y^2+2z} = \frac{2(r+s) - 2(r-s) + 4r}{2(r+s)^2}\\ \amp = \frac{2r + 2s - 2r + 2s + 4r}{2(r+s)^2} = \frac{4r + 4s}{2(r+s)^2} = \frac{2}{r+s} \end{align*}
    Method 2: Direct Substitution
    🔗
    We can check our work by substituting \(x\text{,}\) \(y\text{,}\) and \(z\) into the expression for \(w\) before differentiating:
    🔗
    \begin{align*} x^2 + y^2 + 2z \amp = (r+s)^2 + (r-s)^2 + 2(2rs)\\ \amp = (r^2 + 2rs + s^2) + (r^2 - 2rs + s^2) + 4rs\\ \amp = 2r^2 + 4rs + 2s^2 = 2(r+s)^2 \end{align*}
    Thus, the function can be rewritten directly as:
    \begin{equation*} w = \ln(2(r+s)^2) = \ln(2) + 2\ln(r+s) \end{equation*}
    🔗
    Differentiating directly with respect to \(r\) and \(s\) yields:
    \begin{equation*} w_r = \frac{2}{r+s} \qquad \text{and} \qquad w_s = \frac{2}{r+s} \end{equation*}
    Both methods yield identical results.
