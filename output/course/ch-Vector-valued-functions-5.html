    Section 2.3 Integrals of Vector Functions

    In this section, we will discuss the integrals of vector functions. A vector function is a function that takes one or more variables and returns a vector. The integral of a vector function can be computed component-wise, meaning that we can integrate each component of the vector function separately.
    🔗

    Definition 2.33. Indefinite Integral.

    If \(\mathbf{R}(t)\) is an antiderivative of \(\mathbf{r}(t)\text{,}\) then:
    \begin{equation*} \int\mathbf{r}(t)\, dt=\mathbf{R}(t)+\mathbf{C} \end{equation*}
    🔗
    🔗

    Definition 2.34.

    Let \(\mathbf{r}(t)=x(t)\mathbf{i}+y(t)\mathbf{j}+z(t)\mathbf{k}=\langle x(t),\ y(t),\ z(t) \rangle\) where \(x\text{,}\) \(y\text{,}\) and \(z\) are integrable on \([a,b]\text{.}\) Then we say \(\mathbf{r}(t)\) is integrable on \([a,b]\) and:
    \begin{equation*} \int^b_a\mathbf{r}(t)\, dt= \left[\int^b_ax(t)\, dt\right]\mathbf{i}+\left[\int^b_ay(t)\, dt\right] \mathbf{j} +\left[\int^b_az(t)\, dt \right]\mathbf{k} \end{equation*}
    If \(\mathbf{R}'(t)=\mathbf{r}(t)\text{,}\) then \(\mathbf{R}(t)\) is an antiderivative of \(\mathbf{r}(t)\text{.}\)
    🔗
    🔗

    Example 2.35.

    Evaluate:
    \begin{equation*} \int^\pi_0((\cos t)\mathbf{i}+\mathbf{j}-t^2 \mathbf{k})\, dt \end{equation*}
    🔗
    Solution.
    We integrate each component function independently over the interval \([0, \pi]\) by applying the Fundamental Theorem of Calculus component-wise:
    \begin{align*} \int^\pi_0 ((\cos t)\mathbf{i}+\mathbf{j}-t^2 \mathbf{k})\, dt \amp = \left[ \int^\pi_0 \cos t\, dt \right]\mathbf{i} + \left[ \int^\pi_0 1\, dt \right]\mathbf{j} - \left[ \int^\pi_0 t^2\, dt \right]\mathbf{k}\\ \amp = \left[ \sin t \right]^\pi_0 \mathbf{i} + \left[ t \right]^\pi_0 \mathbf{j} - \left[ \frac{t^3}{3} \right]^\pi_0 \mathbf{k}\\ \amp = (\sin\pi - \sin 0)\mathbf{i} + (\pi - 0)\mathbf{j} - \left( \frac{\pi^3}{3} - 0 \right)\mathbf{k}\\ \amp = 0\mathbf{i} + \pi\mathbf{j} - \frac{\pi^3}{3}\mathbf{k}\\ \amp = \pi\mathbf{j} - \frac{\pi^3}{3}\mathbf{k} = \left\langle 0, \pi, -\frac{\pi^3}{3} \right\rangle \end{align*}
    🔗
    🔗
    🔗

    Theorem 2.36.

    If \(\mathbf{R}(t)\) is an antiderivative of \(\mathbf{r}(t)\) on \([a,b]\text{,}\) then:
    \begin{equation*} \int^b_a\mathbf{r}(t)\, dt = \left.\mathbf{R}(t)\right|^b_a = \mathbf{R}(b)-\mathbf{R}(a) \end{equation*}
    🔗
    🔗

    Example 2.37.

    If \(\mathbf{r}(t)=12t^3\mathbf{i}+4e^{2t}\mathbf{j}+(t+1)^{-1}\mathbf{k}\text{,}\) find \(\int^2_0 \mathbf{r}(t) \, dt\text{.}\)
    🔗
    Solution.
    We find an antiderivative vector function \(\mathbf{R}(t)\) by integrating each scalar component function:
    \begin{align*} \mathbf{R}(t) \amp = \left( \int 12t^3\, dt \right)\mathbf{i} + \left( \int 4e^{2t}\, dt \right)\mathbf{j} + \left( \int \frac{1}{t+1}\, dt \right)\mathbf{k}\\ \amp = 3t^4\mathbf{i} + 2e^{2t}\mathbf{j} + \ln|t+1|\mathbf{k} \end{align*}
    Now, apply the Fundamental Theorem of Calculus by evaluating \(\mathbf{R}(t)\) from \(t=0\) to \(t=2\text{:}\)
    \begin{align*} \int^2_0 \mathbf{r}(t) \, dt \amp = \mathbf{R}(2) - \mathbf{R}(0)\\ \amp = \left( 3(2)^4\mathbf{i} + 2e^{2(2)}\mathbf{j} + \ln|2+1|\mathbf{k} \right) - \left( 3(0)^4\mathbf{i} + 2e^{2(0)}\mathbf{j} + \ln|0+1|\mathbf{k} \right)\\ \amp = \left( 48\mathbf{i} + 2e^4\mathbf{j} + \ln(3)\mathbf{k} \right) - \left( 0\mathbf{i} + 2\mathbf{j} + 0\mathbf{k} \right)\\ \amp = 48\mathbf{i} + (2e^4 - 2)\mathbf{j} + \ln(3)\mathbf{k}\\ \amp = \left\langle 48, 2e^4 - 2, \ln(3) \right\rangle \end{align*}
    🔗
    🔗
    🔗

    Example 2.38.

    A particle traveling in a straight line is located at the point \((1,-1,2)\) and has speed \(2\) at time \(t=0\text{.}\) The particle moves toward the point \((3,0,3)\) with constant acceleration \(\mathbf{a}(t) = 2\mathbf{i}+\mathbf{j}+\mathbf{k}\text{.}\) Find its position vector \(\mathbf{r}(t)\) at time \(t\text{.}\)
    🔗
    Solution.
    First, we establish our initial conditions at \(t=0\text{.}\) The initial position vector is given directly by the starting coordinates:
    \begin{equation*} \mathbf{r}(0) = \langle 1, -1, 2 \rangle \end{equation*}
    To find the initial velocity vector \(\mathbf{v}(0)\text{,}\) we know the particle moves from \(P(1,-1,2)\) toward \(Q(3,0,3)\text{.}\) The direction vector is:
    \begin{equation*} \vec{PQ} = \langle 3-1, 0-(-1), 3-2 \rangle = \langle 2, 1, 1 \rangle \end{equation*}
    We normalize this direction vector to get a unit direction vector:
    \begin{equation*} \mathbf{u} = \frac{\langle 2, 1, 1 \rangle}{\sqrt{2^2 + 1^2 + 1^2}} = \frac{1}{\sqrt{6}}\langle 2, 1, 1 \rangle \end{equation*}
    Multiplying the initial speed of \(2\) by this unit direction vector yields our initial velocity vector:
    \begin{equation*} \mathbf{v}(0) = 2\mathbf{u} = \frac{2}{\sqrt{6}}\langle 2, 1, 1 \rangle = \left\langle \frac{4}{\sqrt{6}}, \frac{2}{\sqrt{6}}, \frac{2}{\sqrt{6}} \right\rangle \end{equation*}
    🔗
    Next, we find the general velocity vector \(\mathbf{v}(t)\) by integrating the constant acceleration vector:
    \begin{equation*} \mathbf{v}(t) = \int \mathbf{a}(t)\, dt = \int \langle 2, 1, 1 \rangle\, dt = \langle 2t, t, t \rangle + \mathbf{C}_1 \end{equation*}
    Using our initial velocity condition at \(t=0\) reveals \(\mathbf{C}_1 = \mathbf{v}(0)\text{,}\) so:
    \begin{equation*} \mathbf{v}(t) = \left\langle 2t + \frac{4}{\sqrt{6}}, t + \frac{2}{\sqrt{6}}, t + \frac{2}{\sqrt{6}} \right\rangle \end{equation*}
    🔗
    Finally, we integrate the velocity vector function to find the position vector function \(\mathbf{r}(t)\text{:}\)
    \begin{equation*} \mathbf{r}(t) = \int \mathbf{v}(t)\, dt = \left\langle t^2 + \frac{4}{\sqrt{6}}t, \frac{1}{2}t^2 + \frac{2}{\sqrt{6}}t, \frac{1}{2}t^2 + \frac{2}{\sqrt{6}}t \right\rangle + \mathbf{C}_2 \end{equation*}
    Using our initial position condition at \(t=0\) reveals \(\mathbf{C}_2 = \mathbf{r}(0) = \langle 1, -1, 2 \rangle\text{.}\) Combining these gives the final position vector function:
    \begin{equation*} \mathbf{r}(t) = \left\langle t^2 + \frac{4}{\sqrt{6}}t + 1, \frac{1}{2}t^2 + \frac{2}{\sqrt{6}}t - 1, \frac{1}{2}t^2 + \frac{2}{\sqrt{6}}t + 2 \right\rangle \end{equation*}
