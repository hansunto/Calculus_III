    Section 2.1 Vector-valued Functions and space curves

    A vector-valued function is a function that takes one or more variables and returns a vector. It can be represented in the form:
    🔗
    \begin{equation*} \mathbf{r}(t) = \langle x(t), y(t), z(t) \rangle \end{equation*}
    where \(x(t)\text{,}\) \(y(t)\text{,}\) and \(z(t)\) are scalar functions of the variable \(t\text{.}\)
    🔗

    Definition 2.1.

    A vector-valued function is a function whose domain is a set of real numbers and whose range is a set of vectors. Let \(D\) be a set of real numbers. Then the three components of the vector-valued function \(\mathbf{r}(t)\) are uniquely determined:
    \begin{equation*} \mathbf{r}(t) = \langle x(t),y(t),z(t) \rangle = x(t)\mathbf{i}+y(t)\mathbf{j}+z(t)\mathbf{k} \end{equation*}
    where \(x\text{,}\) \(y\text{,}\) and \(z\) are scalar functions with the domain \(D\text{.}\)
    🔗
    🔗

    Example 2.2.

    If \(\mathbf{r}(t) = \langle t^3,\ln{(3-t)},\sqrt{t} \rangle\text{,}\) find the component functions.
    🔗
    Solution.
    By identifying the scalar functions multiplying each unit vector or filling each slot, the component functions are:
    \begin{equation*} x(t)=t^3, \quad y(t)=\ln{(3-t)}, \quad z(t)=\sqrt{t} \end{equation*}
    🔗
    🔗
    🔗

    Example 2.3.

    Let \(\mathbf{r}(t)=\langle t+1,t^2-4,t^2 \rangle\) for \(t\in\mathbb{R}\text{.}\)
    🔗
    1. Sketch \(\mathbf{r}(1)\) and \(\mathbf{r}(3)\text{.}\)
      🔗
      🔗
    2. Find the vectors \(\mathbf{r}(t)\) that are in a coordinate plane.
      🔗
      🔗
    Solution.
    1. The positions vectors are evaluated at the given parameter inputs:
      \begin{align*} \mathbf{r}(1) \amp = \langle 1+1, 1^2-4, 1^2 \rangle = \langle 2, -3, 1 \rangle\\ \mathbf{r}(3) \amp = \langle 3+1, 3^2-4, 3^2 \rangle = \langle 4, 5, 9 \rangle \end{align*}
      🔗
      🔗
    2. We analyze each coordinate plane by setting the appropriate component to zero:
      🔗
      1. \(xy\)-plane : The \(\mathbf{k}\)-component is zero.
        \begin{align*} t^2 = 0 \amp \implies t = 0\\ \amp \implies \mathbf{r}(0) = \langle 1,-4,0 \rangle \end{align*}
        🔗
        🔗
      2. \(yz\)-plane : The \(\mathbf{i}\)-component is zero.
        \begin{align*} t + 1 = 0 \amp \implies t = -1\\ \amp \implies \mathbf{r}(-1) = \langle 0,-3,1 \rangle \end{align*}
        🔗
        🔗
      3. \(xz\)-plane : The \(\mathbf{j}\)-component is zero.
        \begin{align*} t^2 - 4 = 0 \amp \implies t = 2 \text{ or } t = -2\\ \amp \implies \mathbf{r}(2) = \langle 3,0,4 \rangle\\ \amp \text{and } \mathbf{r}(-2) = \langle -1,0,4 \rangle \end{align*}
        🔗
        🔗
      🔗
    🔗
    🔗

    Subsection Limits and Continuity

    Definition 2.5.

    The limit of a vector function \(\mathbf{r}(t)=\langle x(t),y(t),z(t) \rangle\) is defined by taking the limits of its component functions:
    \begin{equation*} \lim_{t\to a}\mathbf{r}(t)=\langle \lim_{t\to a}x(t), \lim_{t\to a}y(t), \lim_{t\to a}z(t) \rangle \end{equation*}
    provided the limits of the component functions exist.
    🔗
    A vector function \(\mathbf{r}(t)\) is continuous at \(a\) if:
    \begin{equation*} \lim_{t\to a}\mathbf{r}(t)=\mathbf{r}(a) \end{equation*}
    Equivalently, \(\mathbf{r}(t)\) is continuous at \(a\) if and only if its component functions \(x\text{,}\) \(y\text{,}\) and \(z\) are continuous at \(a\text{.}\)
    🔗
    🔗

    Example 2.6.

    Find \(\lim_{t\to 0}\mathbf{r}(t)\text{,}\) where:
    \begin{equation*} \mathbf{r}(t)=\left\langle 1+t^3, te^{-t}, \frac{\sin t}{t} \right\rangle \end{equation*}
    🔗
    Solution.
    We evaluate the limit by computing the limit of each component function individually as \(t \to 0\text{:}\)
    \begin{align*} \lim_{t\to 0} (1+t^3) \amp = 1 + 0^3 = 1\\ \lim_{t\to 0} (te^{-t}) \amp = 0 \cdot e^{0} = 0\\ \lim_{t\to 0} \left(\frac{\sin t}{t}\right) \amp = 1 \end{align*}
    Combining these results into a single vector gives:
    \begin{equation*} \lim_{t\to 0}\mathbf{r}(t) = \langle 1, 0, 1 \rangle \end{equation*}
    🔗
    🔗
    🔗
    Suppose that \(f\text{,}\) \(g\text{,}\) and \(h\) are continuous on an interval \(I\text{.}\) Then the set \(C\) of all points \((x,y,z)\) in space, where:
    \begin{equation*} x=f(t),\quad y=g(t),\quad z=h(t) \end{equation*}
    and \(t\) varies in \(I\text{,}\) is called a space curve. The equations are called the parametric equations of C with a parameter \(t\text{.}\)
    🔗

    Example 2.7.

    Describe the curve defined by the vector function:
    \begin{equation*} \mathbf{r}(t)=\langle 1+t,2+5t,-1+6t \rangle \end{equation*}
    🔗
    Solution.
    The corresponding parametric equations are:
    \begin{equation*} x=1+t,\quad y=2+5t,\quad z=-1+6t \end{equation*}
    These are the parametric equations of a line passing through the point \((1,2,-1)\) and parallel to the vector \(\langle 1,5,6 \rangle\text{.}\)
    🔗
    Alternatively, we can observe that the function can be written as:
    \begin{equation*} \mathbf{r}(t)=\mathbf{r}_0+t\mathbf{v}, \quad \text{where } \mathbf{r}_0=\langle 1,2,-1 \rangle \text{ and } \mathbf{v}=\langle 1,5,6 \rangle \end{equation*}
    which is the vector equation of a line.
    🔗
    🔗
    🔗

    Example 2.8.

    Sketch the curve whose vector equation is:
    \begin{equation*} \mathbf{r}(t)=\cos t\mathbf{i}+\sin t\mathbf{j}+t\mathbf{k} \end{equation*}
    This curve is called a helix. This occurs in the model of DNA.
    🔗
    Solution.
    As the parameter \(t\) increases, the \(x\)- and \(y\)-components trace out a circle of radius \(1\) in the horizontal plane via \(x = \cos t\) and \(y = \sin t\text{.}\) Concurrently, the \(z\)-component increases linearly since \(z = t\text{.}\) The combination of this circular base rotation and vertical motion generates a spiral path that climbs upward along the surface of a cylinder.
    🔗
    🔗
    🔗
    Recall a vector equation for the line segment that joins the tip of the vector \(\mathbf{r}_0\) to the tip of the vector \(\mathbf{r}_1\text{:}\)
    \begin{equation*} \mathbf{r}(t)=(1-t)\mathbf{r}_0 +t\mathbf{r}_1\quad\quad 0\le t\le 1 \end{equation*}
    🔗

    Example 2.10.

    Find a vector equation and parametric equations for the line segment that joins the point \(P(1,3,-2)\) to the point \(Q(2,-1,3)\text{.}\)
    🔗
    Solution.
    Using the points as components for the position vectors \(\mathbf{r}_0 = \langle 1,3,-2 \rangle\) and \(\mathbf{r}_1 = \langle 2,-1,3 \rangle\text{,}\) the vector equation is:
    \begin{equation*} \mathbf{r}(t)=\langle 1+t, \ 3-4t, \ -2 +5t \rangle, \quad 0 \le t \le 1 \end{equation*}
    The corresponding parametric equations are:
    \begin{equation*} x=1+t,\quad y=3-4t, \quad z=-2+5t\quad\quad 0\le t\le 1 \end{equation*}
    🔗
    🔗
    🔗

    Example 2.11.

    Find a vector function that represents the curve of the intersection of the cylinder \(x^2+y^2=1\) and the plane \(y+z=2\text{.}\)
    🔗
    Solution.
    The projection of the intersection curve \(C\) onto the \(xy\)-plane is the unit circle \(x^2+y^2=1\) with \(z=0\text{.}\) We can parameterize this circular footprint as:
    \begin{equation*} x=\cos t, \quad y=\sin t \quad \quad 0 \le t \le 2\pi \end{equation*}
    To find the corresponding \(z\)-coordinate along the path, substitute the expression for \(y\) into the plane equation \(y+z=2\text{:}\)
    \begin{equation*} z = 2 - y = 2 - \sin t \end{equation*}
    Thus, the parametric equations for \(C\) are \(x=\cos t\text{,}\) \(y=\sin t\text{,}\) and \(z=2-\sin t\text{.}\) The corresponding vector function is:
    \begin{equation*} \mathbf{r}(t)=\cos t\mathbf{i}+\sin t\mathbf{j}+(2-\sin t)\mathbf{k} \quad \quad 0 \le t \le 2\pi \end{equation*}
    🔗
    🔗
    🔗

    Example 2.13.

    Sketch the curve whose vector equation is:
    \begin{equation*} \mathbf{r}(t)=t\mathbf{i}+ t^2\mathbf{j}+t^3\mathbf{k}\quad t \ge 0 \end{equation*}
    This curve is called a twisted cubic. You can view this curve as the intersection of the cylinders \(y=x^2\) and \(z=x^3\text{.}\)
    🔗
    Solution.
    As \(t\) increases from \(0\text{,}\) the path traces out a curve where the \(y\)-coordinate grows quadratically relative to \(x\text{,}\) and the \(z\)-coordinate grows cubically.
    🔗
    🔗
    🔗

    Checkpoint 2.15.

    Let \(\mathbf{r}(t)=2t\mathbf{i}+ (8-2t^2)\mathbf{j}\) for \(-2 \le t \le 2\text{.}\) Sketch the curve \(C\) defined by \(\mathbf{r}(t)\) and indicate its orientation.
    🔗
    Solution.
    Eliminating the parameter yields \(t = x/2\text{.}\) Substituting this into the \(y\)-component gives the Cartesian equation:
    \begin{equation*} y = 8 - 2\left(\frac{x}{2}\right)^2 = 8 - \frac{1}{2}x^2 \end{equation*}
    This is a downward-opening parabola. As \(t\) goes from \(-2\) to \(2\text{,}\) the curve is traced from left to right starting at \((-4,0)\text{,}\) passing through the vertex at \((0,8)\text{,}\) and ending at \((4,0)\text{.}\)
    🔗
    🔗
    🔗

    Checkpoint 2.17.

    Let \(\mathbf{r}(t)=\left(\frac{1}{2}t\cos t\right)\mathbf{i}+ \left(\frac{1}{2}t\sin t\right)\mathbf{j}\) for \(0 \le t \le 3\pi\text{.}\) Sketch the curve \(C\) defined by \(\mathbf{r}(t)\) and indicate its orientation.
    🔗
    Solution.
    In polar coordinates, this curve can be viewed as the spiral \(r = \frac{1}{2}\theta\text{,}\) where the parameter \(t\) acts as the angle \(\theta\text{.}\) As \(t\) increases from \(0\) to \(3\pi\text{,}\) the point winds counterclockwise outward from the origin, completing one and a half full revolutions.
