    Section 2.2 Derivatives of Vector Functions

    In this section, we will explore the concept of derivatives of vector functions. A vector function is a function that takes one or more variables and returns a vector. The derivative of a vector function provides information about how the vector changes with respect to its input variables.
    🔗
    We will cover the following topics:
    🔗
    • Definition of the derivative of a vector function
      🔗
    • Rules for differentiating vector functions
      🔗

    Subsection Differentiability

    A vector-valued function is differentiable at \(t = a\) if the derivative exists, which is defined as:
    🔗
    \begin{equation*} \mathbf{r}'(a) = \lim_{h \to 0} \frac{\mathbf{r}(a + h) - \mathbf{r}(a)}{h} \end{equation*}
    The derivative of a vector-valued function gives the tangent vector to the curve at that point.
    🔗

    Definition 2.19.

    The derivative \(\mathbf{r}'\) of a vector function \(\mathbf{r}\) is defined in the same manner as for real-valued functions:
    \begin{equation*} \frac{d\mathbf{r}}{dt} = \mathbf{r}'(t) = \lim_{h\to 0}\frac{\mathbf{r}(t+h) - \mathbf{r}(t)}{h} = \lim_{\Delta t\to 0}\frac{1}{\Delta t}(\mathbf{r}(t+\Delta t) - \mathbf{r}(t)) \end{equation*}
    🔗
    🔗

    Theorem 2.20.

    If \(\mathbf{r}(t)=x(t)\mathbf{i}+y(t)\mathbf{j}+z(t)\mathbf{k}=\langle x(t),\ y(t),\ z(t) \rangle\) where \(x\text{,}\) \(y\text{,}\) and \(z\) are differentiable functions, then:
    \begin{equation*} \mathbf{r}'(t)=x'(t)\mathbf{i}+y'(t)\mathbf{j}+z'(t)\mathbf{k}=\langle x'(t),\ y'(t),\ z'(t) \rangle \end{equation*}
    🔗
    🔗

    Proof.
    This rule is easily proven by applying the component-wise limits directly into the fundamental definition of the derivative of vector functions.
    🔗
    🔗
    If the points \(P\) and \(Q\) have position vectors \(\mathbf{r}(t)\) and \(\mathbf{r}(t+h)\text{,}\) then the displacement vector \(\vec{PQ}\) represents the vector \(\mathbf{r}(t+h) - \mathbf{r}(t)\text{.}\) As \(h \to 0\text{,}\) the vector \(\mathbf{r}(t+h) - \mathbf{r}(t)\) approaches a vector that lies along the target trajectory’s path.
    🔗
    The vector \(\mathbf{r}'(t)\) is called the tangent vector to the curve defined by \(\mathbf{r}(t)\) at \(P\text{,}\) provided that \(\mathbf{r}'(t)\) exists and \(\mathbf{r}'(t) \neq \mathbf{0}\text{.}\) The tangent line to \(C\) at \(P\) is defined to be the line through \(P\) parallel to the tangent vector \(\mathbf{r}'(t)\text{.}\) (In a 2D plane, the slope of the line through \(P\) parallel to \(\mathbf{r}'(t)\) evaluates to \(\frac{y'(t)}{x'(t)}\)).
    🔗

    The Unit Tangent Vector.

    We define the unit tangent vector as:
    \begin{equation*} \mathbf{T}(t)=\frac{\mathbf{r}'(t)}{|\mathbf{r}'(t)|} \end{equation*}
    🔗
    🔗

    Corollary 2.21.

    If \(\mathbf{r}'(t)\) exists, then we say \(\mathbf{r}(t)\) is differentiable and write:
    \begin{equation*} \mathbf{r}'(t)=D_t\mathbf{r}(t)=\frac{d}{dt}\mathbf{r}(t) \end{equation*}
    Similarly, the second derivative \(\mathbf{r}''(t)\) is defined as:
    \begin{equation*} \mathbf{r}''(t)=x''(t)\mathbf{i}+y''(t)\mathbf{j}+z''(t)\mathbf{k}=\langle x''(t),\ y''(t),\ z''(t) \rangle \end{equation*}
    🔗
    🔗

    Proof.
    Let \(\mathbf{r}(t) = \langle x(t), y(t), z(t) \rangle\text{.}\) By applying the component derivative theorem, the first derivative is computed component-by-component:
    \begin{equation*} \frac{d}{dt}\mathbf{r}(t) = \left\langle \frac{dx}{dt}, \frac{dy}{dt}, \frac{dz}{dt} \right\rangle = \langle x'(t), y'(t), z'(t) \rangle \end{equation*}
    🔗
    Since each component function \(x'(t)\text{,}\) \(y'(t)\text{,}\) and \(z'(t)\) is a standard real-valued scalar function, we can differentiate a second time using standard single-variable calculus rules:
    \begin{align*} \mathbf{r}''(t) \amp = \frac{d}{dt}\left[ \mathbf{r}'(t) \right]\\ \amp = \left\langle \frac{d}{dt}[x'(t)], \frac{d}{dt}[y'(t)], \frac{d}{dt}[z'(t)] \right\rangle\\ \amp = \langle x''(t), y''(t), z''(t) \rangle \end{align*}
    Expressing this result in terms of the standard unit basis vectors yields \(x''(t)\mathbf{i}+y''(t)\mathbf{j}+z''(t)\mathbf{k}\text{.}\)
    🔗
    🔗

    Example 2.22.

    Let \(\mathbf{r}(t)=\ln t\ \mathbf{i}+e^{-3t}\mathbf{j}+t^2\mathbf{k}\text{.}\) Find the derivative \(\mathbf{r}'(t)\) where \(t \gt 0\text{.}\)
    🔗
    Solution.
    We differentiate each scalar component function with respect to \(t\text{:}\)
    \begin{align*} \frac{d}{dt}(\ln t) \amp = \frac{1}{t}\\ \frac{d}{dt}(e^{-3t}) \amp = -3e^{-3t}\\ \frac{d}{dt}(t^2) \amp = 2t \end{align*}
    Combining these component results gives the derivative vector function:
    \begin{equation*} \mathbf{r}'(t) = \frac{1}{t}\mathbf{i} - 3e^{-3t}\mathbf{j} + 2t\mathbf{k} \end{equation*}
    🔗
    🔗
    🔗

    Example 2.23.

    Let \(\mathbf{r}(t)=(1+t^3)\mathbf{i}+te^{-t}\mathbf{j}+\sin 2t\mathbf{k}\text{.}\)
    🔗
    1. Find \(\mathbf{r}'(t)\) and \(\mathbf{r}''(t)\text{.}\)
      🔗
      🔗
    2. Find the unit tangent vector at the point where \(t=0\text{.}\)
      🔗
      🔗
    Solution.
    1. Differentiating component-by-component using the power rule, product rule, and chain rule yields the first derivative:
      \begin{equation*} \mathbf{r}'(t) = 3t^2\mathbf{i} + (e^{-t} - te^{-t})\mathbf{j} + 2\cos 2t\mathbf{k} \end{equation*}
      Differentiating a second time gives the second derivative:
      \begin{equation*} \mathbf{r}''(t) = 6t\mathbf{i} + (-e^{-t} - e^{-t} + te^{-t})\mathbf{j} - 4\sin 2t\mathbf{k} = 6t\mathbf{i} + (te^{-t} - 2e^{-t})\mathbf{j} - 4\sin 2t\mathbf{k} \end{equation*}
      🔗
      🔗
    2. To find the unit tangent vector \(\mathbf{T}(0)\text{,}\) we first evaluate the velocity derivative vector at \(t=0\text{:}\)
      \begin{equation*} \mathbf{r}'(0) = 3(0)^2\mathbf{i} + (e^0 - 0)\mathbf{j} + 2\cos(0)\mathbf{k} = \mathbf{j} + 2\mathbf{k} = \langle 0, 1, 2 \rangle \end{equation*}
      Next, compute the magnitude of this tangent vector:
      \begin{equation*} |\mathbf{r}'(0)| = \sqrt{0^2 + 1^2 + 2^2} = \sqrt{5} \end{equation*}
      Divide the vector by its magnitude to obtain the final unit tangent vector:
      \begin{equation*} \mathbf{T}(0) = \frac{\mathbf{r}'(0)}{|\mathbf{r}'(0)|} = \frac{1}{\sqrt{5}}\mathbf{j} + \frac{2}{\sqrt{5}}\mathbf{k} = \left\langle 0, \frac{1}{\sqrt{5}}, \frac{2}{\sqrt{5}} \right\rangle \end{equation*}
      🔗
      🔗
    🔗
    🔗
    🔗

    Checkpoint 2.24.

    For the curve \(\mathbf{r}(t)=\sqrt{t}\mathbf{i}+(2-t)\mathbf{j}\text{,}\) find \(\mathbf{r}'(t)\) and sketch the position vector \(\mathbf{r}(1)\) and the tangent vector \(\mathbf{r}'(1)\text{.}\)
    🔗
    Solution.
    Differentiating each component with respect to \(t\) yields:
    \begin{equation*} \mathbf{r}'(t) = \frac{1}{2\sqrt{t}}\mathbf{i} - \mathbf{j} \end{equation*}
    Evaluating both the original position vector and the derivative tangent vector at \(t=1\) gives:
    \begin{equation*} \mathbf{r}(1) = \mathbf{i} + \mathbf{j} = \langle 1, 1 \rangle \end{equation*}
    \begin{equation*} \mathbf{r}'(1) = \frac{1}{2}\mathbf{i} - \mathbf{j} = \left\langle \frac{1}{2}, -1 \right\rangle \end{equation*}
    🔗
    🔗
    🔗

    Checkpoint 2.26.

    Find the parametric equations for the tangent line to the helix with the parametric equations:
    \begin{equation*} x=2\cos t, \quad y=\sin t, \quad z=t \quad \text{ at the point } (0,1,\pi/2) \end{equation*}
    🔗
    Solution.
    First, find the parameter value \(t_0\) that corresponds to the given point \((0,1,\pi/2)\text{.}\) Matching the components yields \(z = t = \pi/2\text{.}\) Testing this value reveals \(2\cos(\pi/2) = 0\) and \(\sin(\pi/2) = 1\text{,}\) which confirms that \(t_0 = \pi/2\text{.}\)
    🔗
    Next, differentiate the vector function \(\mathbf{r}(t) = \langle 2\cos t, \sin t, t \rangle\) to find the directional tangent vector:
    \begin{equation*} \mathbf{r}'(t) = \langle -2\sin t, \cos t, 1 \rangle \end{equation*}
    Evaluating the direction vector at our parameter point \(t_0 = \pi/2\) gives:
    \begin{equation*} \mathbf{r}'(\pi/2) = \langle -2(1), 0, 1 \rangle = \langle -2, 0, 1 \rangle \end{equation*}
    🔗
    Using the base point coordinates \((0,1,\pi/2)\) and the direction components \(\langle -2,0,1 \rangle\text{,}\) the parametric equations for the tangent line are:
    \begin{equation*} x = -2t, \quad y = 1, \quad z = \frac{\pi}{2} + t \end{equation*}
    🔗
    🔗
    🔗

    Checkpoint 2.27.

    Find the parametric equations for the tangent line to the twisted curve with the parametric equations:
    \begin{equation*} x = t, \quad y = t^2, \quad z = t^3 \quad \text{ at the point } t=2 \end{equation*}
    🔗
    Solution.
    Evaluate the base point on the curve at \(t=2\text{:}\)
    \begin{equation*} x_0 = 2, \quad y_0 = 2^2 = 4, \quad z_0 = 2^3 = 8 \implies (2,4,8) \end{equation*}
    Differentiate the component expressions to find the general velocity direction profile:
    \begin{equation*} \frac{dx}{dt} = 1, \quad \frac{dy}{dt} = 2t, \quad \frac{dz}{dt} = 3t^2 \end{equation*}
    Evaluating these derivatives at \(t=2\) gives the parallel directional components for our tangent line:
    \begin{equation*} \left.\frac{dx}{dt}\right|_{t=2} = 1, \quad \left.\frac{dy}{dt}\right|_{t=2} = 4, \quad \left.\frac{dz}{dt}\right|_{t=2} = 12 \end{equation*}
    🔗
    Using the point \((2,4,8)\) and the direction components vector \(\langle 1,4,12 \rangle\text{,}\) the parametric equations for the line are:
    \begin{equation*} x = 2 + t, \quad y = 4 + 4t, \quad z = 8 + 12t \end{equation*}
    🔗
    🔗
    🔗

    Definition 2.28.

    A curve given by a vector function \(\mathbf{r}(t)\) on an interval \(I\) is called smooth if \(\mathbf{r}'(t)\) is continuous and \(\mathbf{r}'(t) \neq \mathbf{0}\text{,}\) except possibly at the endpoints of \(I\text{.}\)
    🔗
    🔗

    Example 2.29.

    Determine whether the semicubical parabola \(\mathbf{r}(t)=\langle 1+t^3,t^2 \rangle\) is smooth.
    🔗
    Solution.
    First, we compute the derivative vector function component-by-component:
    \begin{equation*} \mathbf{r}'(t) = \langle 3t^2, 2t \rangle \end{equation*}
    Both component functions, \(3t^2\) and \(2t\text{,}\) are polynomials, so \(\mathbf{r}'(t)\) is continuous everywhere. Next, we determine if there are any parameter values where the derivative vector equals the zero vector:
    \begin{equation*} \mathbf{r}'(t) = \langle 0, 0 \rangle \implies 3t^2 = 0 \text{ and } 2t = 0 \end{equation*}
    This system yields the single solution \(t = 0\text{.}\) Since \(\mathbf{r}'(0) = \mathbf{0}\) at an interior point of the domain, the curve is not smooth at the origin (it features a sharp cusp).
    🔗
    🔗
    🔗
    A curve like the semicubical parabola that is made up of a finite number of smooth pieces is called piecewise smooth.
    🔗

    Example 2.30.

    Determine whether the helix \(\mathbf{r}(t)=\langle 2\cos t, \sin t, t \rangle\) is smooth.
    🔗
    Solution.
    We find the derivative vector function:
    \begin{equation*} \mathbf{r}'(t) = \langle -2\sin t, \cos t, 1 \rangle \end{equation*}
    The components are standard trigonometric and constant functions, meaning \(\mathbf{r}'(t)\) is continuous for all real numbers.
    🔗
    For the vector to be the zero vector, all three components must simultaneously equal zero. However, the \(\mathbf{k}\)-component is always \(1 \neq 0\text{.}\) Therefore, \(\mathbf{r}'(t) \neq \mathbf{0}\) for all values of \(t\text{.}\) We conclude that the helix is a smooth curve.
    🔗
    🔗
    🔗

    Theorem 2.31.

    Suppose \(\mathbf{u}\) and \(\mathbf{v}\) are differentiable vector functions, \(c\) is a scalar, and \(f\) is a real-valued function. Then:
    🔗
    1. \(\displaystyle \frac{d}{dt}[\mathbf{u}(t)+\mathbf{v}(t)]= \mathbf{u}'(t)+\mathbf{v}'(t)\)
      🔗
      🔗
    2. \(\displaystyle \frac{d}{dt}[c\mathbf{u}(t)]= c\mathbf{u}'(t)\)
      🔗
      🔗
    3. \(\displaystyle \frac{d}{dt}[f(t)\mathbf{u}(t)]= f'(t)\mathbf{u}(t)+f(t)\mathbf{u}'(t)\)
      🔗
      🔗
    4. \(\displaystyle \frac{d}{dt}[\mathbf{u}(t)\cdot\mathbf{v}(t)]= \mathbf{u}'(t)\cdot\mathbf{v}(t)+\mathbf{u}(t)\cdot\mathbf{v}'(t)\)
      🔗
      🔗
    5. \(\displaystyle \frac{d}{dt}[\mathbf{u}(t)\times\mathbf{v}(t)]= \mathbf{u}'(t)\times\mathbf{v}(t)+\mathbf{u}(t)\times\mathbf{v}'(t)\)
      🔗
      🔗
    6. \(\displaystyle \frac{d}{dt}[\mathbf{u}(f(t))]= f'(t)\mathbf{u}'(f(t)) \quad \text{(Chain Rule)}\)
      🔗
      🔗
    🔗

    Theorem 2.32.

    If \(\mathbf{r}(t)\) is differentiable and \(|\mathbf{r}(t)|=c\) (a constant), then \(\mathbf{r}'(t)\) is orthogonal to \(\mathbf{r}(t)\) for all \(t\) in the domain of \(\mathbf{r}'(t)\text{:}\)
    \begin{equation*} \mathbf{r}(t) \cdot \frac{d\mathbf{r}}{dt} = 0 \end{equation*}
    🔗
    🔗

    Proof.
    Since \(|\mathbf{r}(t)| = c\text{,}\) squaring both sides gives \(|\mathbf{r}(t)|^2 = c^2\text{.}\) Using the relationship between the magnitude and the dot product, we can rewrite this as:
    \begin{equation*} \mathbf{r}(t) \cdot \mathbf{r}(t) = c^2 \end{equation*}
    Differentiating both sides with respect to \(t\) using the dot product rule yields:
    \begin{equation*} \mathbf{r}'(t) \cdot \mathbf{r}(t) + \mathbf{r}(t) \cdot \mathbf{r}'(t) = 0 \end{equation*}
    Since the dot product is commutative, we combine the terms:
    \begin{equation*} 2\left(\mathbf{r}(t) \cdot \mathbf{r}'(t)\right) = 0 \implies \mathbf{r}(t) \cdot \frac{d\mathbf{r}}{dt} = 0 \end{equation*}
    Because their dot product is zero, the position vector \(\mathbf{r}(t)\) and the derivative vector \(\mathbf{r}'(t)\) are orthogonal.
