    1.

    Integrate the scalar function \(f(x,y,z)=\sqrt{x^2+z^2}\) over the circle path parameterization:
    \begin{equation*} \mathbf{r}(t)=(4\cos t)\mathbf{i}+(4\sin t)\mathbf{k}, \qquad 0\leq t\leq 2\pi. \end{equation*}
    🔗
    Solution.
    Step 1: Compute the Derivatives and the Arc Length Element \(ds\)
    🔗
    From the given vector path equation, the components are:
    \begin{equation*} x(t) = 4\cos t, \quad y(t) = 0, \quad z(t) = 4\sin t. \end{equation*}
    Differentiating each coordinate component with respect to the parameter \(t\) yields:
    \begin{equation*} \frac{dx}{dt} = -4\sin t, \quad \frac{dy}{dt} = 0, \quad \frac{dz}{dt} = 4\cos t. \end{equation*}
    We substitute these into the 3D arc length differential equation:
    \begin{equation*} ds = \sqrt{\left(\frac{dx}{dt}\right)^2 + \left(\frac{dy}{dt}\right)^2 + \left(\frac{dz}{dt}\right)^2}\,dt = \sqrt{(-4\sin t)^2 + 0^2 + (4\cos t)^2}\,dt \end{equation*}
    \begin{equation*} ds = \sqrt{16\sin^2 t + 16\cos^2 t}\,dt = \sqrt{16(\sin^2 t + \cos^2 t)}\,dt = 4\,dt. \end{equation*}
    🔗
    Step 2: Setup and Evaluate the Line Integral
    🔗
    Next, we substitute the component equations into the integrand function \(f(x,y,z)\text{:}\)
    \begin{equation*} f(x(t), y(t), z(t)) = \sqrt{(4\cos t)^2 + (4\sin t)^2} = \sqrt{16\cos^2 t + 16\sin^2 t} = 4. \end{equation*}
    Combining the function value, the element \(ds = 4\,dt\text{,}\) and our parameters from \(0\) to \(2\pi\text{:}\)
    \begin{equation*} \int_C f(x,y,z)\,ds = \int_{0}^{2\pi} 4 \cdot (4\,dt) = \int_{0}^{2\pi} 16 \, dt = \Big[ 16t \Big]_{0}^{2\pi} = 32\pi. \end{equation*}
    The exact value of the line integral around the vertical circle path is \(32\pi\text{.}\)

    2.

    Evaluate the line integral along a straight line path segment:
    \begin{equation*} \int_C \frac{dx+dy+dz}{\sqrt{x+y+z}}, \end{equation*}
    where \(C\) is the directed straight line segment starting from the initial coordinate point \((-1,1,1)\) and ending at the terminal point \((4,-3,0)\text{.}\)
    🔗
    Solution.
    Step 1: Parameterize the Straight Line Segment \(C\)
    🔗
    Using the vector parameterization equation \(\mathbf{r}(t) = (1-t)\mathbf{r}_0 + t\mathbf{r}_1\) for a line segment with initial point \(\mathbf{r}_0 = \langle -1,1,1 \rangle\) and terminal point \(\mathbf{r}_1 = \langle 4,-3,0 \rangle\text{:}\)
    \begin{equation*} x(t) = (1-t)(-1) + 4t = 5t - 1 \implies dx = 5\,dt, \end{equation*}
    \begin{equation*} y(t) = (1-t)(1) - 3t = 1 - 4t \implies dy = -4\,dt, \end{equation*}
    \begin{equation*} z(t) = (1-t)(1) + 0t = 1 - t \implies dz = -dt, \end{equation*}
    where the parameter varies along the continuous interval: \(0 \leq t \leq 1\text{.}\)
    🔗
    Step 2: Setup and Simplify the Integrand Components
    🔗
    We combine the differentials and coordinates into the core parameter groups:
    \begin{equation*} \text{Numerator: } dx + dy + dz = (5 - 4 - 1)\,dt = 0\,dt. \end{equation*}
    Because the combined differential vector sums perfectly to zero everywhere along this straight linear trajectory, the entire line integral drops:
    \begin{equation*} \int_C \frac{dx+dy+dz}{\sqrt{x+y+z}} = \int_{0}^{1} \frac{0}{\sqrt{(5t-1) + (1-4t) + (1-t)}} \, dt = \int_{0}^{1} 0 \, dt = 0. \end{equation*}
    The exact value of the coordinate line integral along this specific segment is \(0\text{.}\)

    3.

    Integrate the vector field \(\mathbf{F}=-(y\sin z)\mathbf{i}+(x\sin z)\mathbf{j}+(xy\cos z)\mathbf{k}\) around the circular path cut from the sphere \(x^2+y^2+z^2=5\) by the horizontal plane \(z=-1\text{,}\) oriented clockwise as viewed from above.
    🔗
    Solution.
    Step 1: Identify the Geometry and Path Parameterization
    🔗
    Substituting the plane constraint \(z = -1\) into the sphere equation reveals the radius of the circular intersection track:
    \begin{equation*} x^2 + y^2 + (-1)^2 = 5 \implies x^2 + y^2 + 1 = 5 \implies x^2 + y^2 = 4. \end{equation*}
    This describes a horizontal circle of radius \(R = 2\) resting at a height of \(z = -1\text{.}\) A standard counterclockwise orientation uses \(\langle 2\cos t, 2\sin t, -1 \rangle\text{.}\) Because the question explicitly specifies a clockwise orientation as viewed from above, we reverse the tracking parameter direction:
    \begin{equation*} x(t) = 2\cos t \implies dx = -2\sin t\,dt, \end{equation*}
    \begin{equation*} y(t) = -2\sin t \implies dy = -2\cos t\,dt, \end{equation*}
    \begin{equation*} z(t) = -1 \implies dz = 0\,dt, \end{equation*}
    where the parameter varies continuously across a full revolution: \(0 \leq t \leq 2\pi\text{.}\)
    🔗
    Step 2: Setup and Evaluate the Vector Line Integral
    🔗
    We rewrite the vector line integral in differential component form: \(\oint_C P\,dx + Q\,dy + R\,dz\text{.}\) Substituting \(z = -1\) into our component fields gives:
    \begin{equation*} P = -y\sin(-1) = y\sin(1), \quad Q = x\sin(-1) = -x\sin(1), \quad R = xy\cos(-1) = xy\cos(1). \end{equation*}
    Because \(dz = 0\text{,}\) the \(\mathbf{k}\) component drops out completely. Substituting the parametric equations into the line integral yields:
    \begin{align*} \oint_C \mathbf{F} \cdot d\mathbf{r} \amp = \int_{0}^{2\pi} \left[ \Big( (-2\sin t)\sin(1) \Big)(-2\sin t) + \Big( -(2\cos t)\sin(1) \Big)(-2\cos t) \right] \, dt\\ \amp = \int_{0}^{2\pi} \left( 4\sin^2 t\sin(1) + 4\cos^2 t\sin(1) \right) \, dt\\ \amp = 4\sin(1) \int_{0}^{2\pi} \left(\sin^2 t + \cos^2 t\right) \, dt = 4\sin(1) \int_{0}^{2\pi} 1 \, dt\\ \amp = 4\sin(1) \Big[ t \Big]_{0}^{2\pi} = 8\pi\sin(1). \end{align*}
    🔗
    Step 3: Evaluate the Double Area Integral
    🔗
    Using the parameters, we assemble the iterated region integral:
    \begin{align*} A(S) \amp = \int_{0}^{2\pi} \int_{0}^{5} 3 \, dz \, d\theta\\ \amp = 3 \left( \int_{0}^{2\pi} \, d\theta \right) \left( \int_{0}^{5} \, dz \right)\\ \amp = 3 \cdot \Big[ \theta \Big]_{0}^{2\pi} \cdot \Big[ z \Big]_{0}^{5} = 3 \cdot (2\pi) \cdot (5) = 30\pi. \end{align*}
    The lateral surface area of the cylinder is exactly \(30\pi\) (which perfectly matches our geometric formula verification: \(A = 2\pi r h = 2\pi(3)(5) = 30\pi\)).

    4.

    Evaluate the closed loop coordinate line integral by applying Green’s Theorem:
    \begin{equation*} \oint_C 8x\sin y\,dx - 8y\cos x\,dy, \end{equation*}
    where \(C\) is the perimeter boundary curve of the square cut from the first quadrant by the horizontal and vertical lines \(x=\frac{\pi}{2}\) and \(y=\frac{\pi}{2}\text{,}\) oriented counterclockwise.
    🔗
    Solution.
    Step 1: Compute Partial Derivatives and Apply Green’s Theorem
    🔗
    The path forms a closed square loop oriented counterclockwise. Evaluating each of the four edges independently would be tedious, so we apply Green’s Theorem:
    \begin{equation*} P(x,y) = 8x\sin y \quad \text{and} \quad Q(x,y) = -8y\cos x. \end{equation*}
    Next, we find their respective first-order cross-partial derivatives:
    \begin{equation*} \frac{\partial P}{\partial y} = \frac{\partial}{\partial y}(8x\sin y) = 8x\cos y, \end{equation*}
    \begin{equation*} \frac{\partial Q}{\partial x} = \frac{\partial}{\partial x}(-8y\cos x) = 8y\sin x. \end{equation*}
    By Green’s Theorem, the closed line integral transforms into a double integral over the square domain \(D\text{:}\)
    \begin{equation*} \oint_C P\,dx + Q\,dy = \iint_D \left(\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y}\right) dA = \int_{0}^{\pi/2} \int_{0}^{\pi/2} \left( 8y\sin x - 8x\cos y \right) \, dy \, dx. \end{equation*}
    🔗
    Step 2: Evaluate the Iterated Double Integral
    🔗
    First, we integrate the inner expression with respect to \(y\text{,}\) treating \(x\) as a constant factor:
    \begin{align*} \int_{0}^{\pi/2} \left( 8y\sin x - 8x\cos y \right) \, dy \amp = \left[ 4y^2\sin x - 8x\sin y \right]_{y=0}^{y=\pi/2}\\ \amp = \left( 4\left(\frac{\pi^2}{4}\right)\sin x - 8x\sin\left(\frac{\pi}{2}\right) \right) - 0\\ \amp = \pi^2\sin x - 8x. \end{align*}
    Now, substitute this polynomial expression into the outer layer and integrate with respect to \(x\text{:}\)
    \begin{align*} \int_{0}^{\pi/2} \left( \pi^2\sin x - 8x \right) \, dx \amp = \left[ -\pi^2\cos x - 4x^2 \right]_{0}^{\pi/2}\\ \amp = \left( -\pi^2\cos\left(\frac{\pi}{2}\right) - 4\left(\frac{\pi^2}{4}\right) \right) - \left( -\pi^2\cos(0) - 0 \right)\\ \amp = \left( 0 - \pi^2 \right) - \left( -\pi^2 \right) = -\pi^2 + \pi^2 = 0. \end{align*}
    The final value of the closed square line integral is exactly \(0\text{.}\)

    5.

    Determine which of the following vector fields are conservative, and which are not. Support your answers with partial derivative tests:
    🔗
    1. \(\displaystyle \mathbf{F} = x\mathbf{i} + y\mathbf{j} + z\mathbf{k}\)
      🔗
      🔗
    2. \(\displaystyle \mathbf{F} = \frac{x\mathbf{i} + y\mathbf{j} + z\mathbf{k}}{(x^2+y^2+z^2)^{3/2}}\)
      🔗
      🔗
    3. \(\displaystyle \mathbf{F} = xe^{y}\mathbf{i} + ye^z\mathbf{j} + ze^x\mathbf{k}\)
      🔗
      🔗
    4. \(\displaystyle \mathbf{F} = \frac{\mathbf{i} + z\mathbf{j} + y\mathbf{k}}{x+yz}\)
      🔗
      🔗
    Solution.
    We apply the 3D curl component test (\(\operatorname{curl} \mathbf{F} = \mathbf{0}\) for a field to be conservative) across all four cases:
    🔗
    1. Field (a): Here, \(P=x, Q=y, R=z\text{.}\) Calculating cross-partials shows:
      \begin{equation*} \frac{\partial R}{\partial y} - \frac{\partial Q}{\partial z} = 0, \quad \frac{\partial P}{\partial z} - \frac{\partial R}{\partial x} = 0, \quad \frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y} = 0. \end{equation*}
      Since \(\operatorname{curl}\mathbf{F} = \mathbf{0}\text{,}\) Field (a) is conservative (with potential function \(f = \frac{1}{2}(x^2+y^2+z^2)\)).
      🔗
      🔗
    2. Field (b): This is the classical inverse-square gravitational/electric field model. Let \(\rho = (x^2+y^2+z^2)^{1/2}\text{.}\) The components are \(P = x\rho^{-3}, Q = y\rho^{-3}, R = z\rho^{-3}\text{.}\) Differentiating the components yields symmetric cross-partials:
      \begin{equation*} \frac{\partial P}{\partial y} = x(-3\rho^{-4})\frac{y}{\rho} = -\frac{3xy}{\rho^5} \quad = \quad \frac{\partial Q}{\partial x} = y(-3\rho^{-4})\frac{x}{\rho} = -\frac{3xy}{\rho^5}. \end{equation*}
      All component pairs match identically, meaning \(\operatorname{curl}\mathbf{F} = \mathbf{0}\text{.}\) Thus, Field (b) is conservative (with potential function \(f = -\frac{1}{\sqrt{x^2+y^2+z^2}}\)).
      🔗
      🔗
    3. Field (c): Here, \(P=xe^y, Q=ye^z, R=ze^x\text{.}\) Let’s compute the cross-partial derivative of the first two components:
      \begin{equation*} \frac{\partial P}{\partial y} = \frac{\partial}{\partial y}(xe^y) = xe^y \quad \neq \quad \frac{\partial Q}{\partial x} = \frac{\partial}{\partial x}(ye^z) = 0. \end{equation*}
      Because the cross-partials are unequal (\(\frac{\partial P}{\partial y} \neq \frac{\partial Q}{\partial x}\)), the curl is non-zero. Thus, Field (c) is not conservative.
      🔗
      🔗
    4. Field (d): The components are \(P = \frac{1}{x+yz}, Q = \frac{z}{x+yz}, R = \frac{y}{x+yz}\text{.}\) Let’s check the cross-partial derivative of \(Q\) and \(R\text{:}\)
      \begin{equation*} \frac{\partial R}{\partial y} = \frac{(1)(x+yz) - y(z)}{(x+yz)^2} = \frac{x}{(x+yz)^2}, \end{equation*}
      \begin{equation*} \frac{\partial Q}{\partial z} = \frac{(1)(x+yz) - z(y)}{(x+yz)^2} = \frac{x}{(x+yz)^2}. \end{equation*}
      Checking the remaining component pairs reveals that \(\frac{\partial P}{\partial y} = \frac{\partial Q}{\partial x}\) and \(\frac{\partial P}{\partial z} = \frac{\partial R}{\partial x}\) match symmetrically everywhere on its domain. Since \(\operatorname{curl}\mathbf{F} = \mathbf{0}\text{,}\) Field (d) is conservative (with potential function \(f = \ln|x+yz|\)).
      🔗
      🔗

    6.

    Find the work done by the force field \(\mathbf{F} = 2xy\mathbf{i} + \mathbf{j} + x^2\mathbf{k}\) along a straight line segment path from the origin \((0,0,0)\) to the point \((1,1,1)\text{.}\)
    🔗
    Solution.
    Step 1: Check if the Field is Conservative
    🔗
    Let’s determine if \(\mathbf{F} = \langle 2xy, 1, x^2 \rangle\) is conservative by checking its curl components:
    \begin{equation*} \frac{\partial R}{\partial y} - \frac{\partial Q}{\partial z} = 0 - 0 = 0, \end{equation*}
    \begin{equation*} \frac{\partial P}{\partial z} - \frac{\partial R}{\partial x} = 0 - 2x = -2x \neq 0. \end{equation*}
    Because the curl is not zero (\(\operatorname{curl}\mathbf{F} \neq \mathbf{0}\)), the field is not conservative. Therefore, the work depends on the path, and we must evaluate the line integral directly along the specified straight line segment path.
    🔗
    Step 2: Parameterize the Path and Compute Differentials
    🔗
    The straight line segment connecting \((0,0,0)\) to \((1,1,1)\) can be parameterized linearly as:
    \begin{equation*} x(t) = t \implies dx = dt, \end{equation*}
    \begin{equation*} y(t) = t \implies dy = dt, \end{equation*}
    \begin{equation*} z(t) = t \implies dz = dt, \end{equation*}
    where the parameter varies continuously over the interval: \(0 \leq t \leq 1\text{.}\)
    🔗
    Step 3: Setup and Evaluate the Work Integral
    🔗
    We rewrite the work line integral in differential component form: \(W = \int_C P\,dx + Q\,dy + R\,dz\text{.}\) Substituting our parametric expressions and differentials gives:
    \begin{align*} W \amp = \int_{0}^{1} \left[ 2(t)(t) \cdot (dt) + (1) \cdot (dt) + (t)^2 \cdot (dt) \right]\\ \amp = \int_{0}^{1} \left( 2t^2 + 1 + t^2 \right) \, dt = \int_{0}^{1} \left( 3t^2 + 1 \right) \, dt\\ \amp = \Big[ t^3 + t \Big]_{0}^{1} = \left( (1)^3 + 1 \right) - 0 = 2. \end{align*}
    The total work done by the field along the straight line path is exactly \(2\text{.}\)
