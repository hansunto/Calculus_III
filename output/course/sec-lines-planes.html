    Section 1.5 Lines and Planes

    In this section, we will discuss the equations of lines and planes in three-dimensional space. We will explore how to represent lines and planes using vector equations, parametric equations, and Cartesian equations. Additionally, we will examine the relationships between lines and planes, including conditions for parallelism and intersection.
    🔗
    If a vector \(\vec{v}\) gives the direction of a line \(L\) and is written in component form as \(\vec{v} = \langle a,b,c \rangle\text{,}\) then any scalar multiple of that direction vector can be expressed as \(t\vec{v} = \langle ta,tb,tc \rangle\) for any scalar parameter \(t\text{.}\)
    🔗
    Letting \(\vec{r} = \langle x,y,z \rangle\) represent the position vector of an arbitrary point on the line, and \(\vec{r}_0 = \langle x_0,y_0,z_0 \rangle\) represent the position vector of a specific known point on the line, we can state the vector equation of a line:
    \begin{equation*} \vec{r} = \vec{r}_0 + t\vec{v} \implies \langle x,y,z \rangle = \langle x_0+ta, \, y_0+tb, \, z_0+tc \rangle \end{equation*}
    🔗

    Definition 1.81.

    Two vectors are equal if and only if their corresponding components are equal. Equating the components of the vector equation of a line yields:
    \begin{equation*} x = x_0 + at, \quad y = y_0 + bt, \quad z = z_0 + ct \end{equation*}
    where \(t \in \mathbb{R}\text{.}\) These equations are called the parametric equations of the line \(L\) passing through the point \(P_0(x_0,y_0,z_0)\) and parallel to the vector \(\vec{v} = \langle a,b,c \rangle\text{.}\) Each value of the scalar parameter \(t\) represents a unique point \((x,y,z)\) on \(L\text{.}\)
    🔗
    🔗

    Example 1.82.

    Find a vector equation and parametric equations for the line that passes through the point \((5,1,3)\) and is parallel to the vector \(\vec{i}+4\vec{j}-2\vec{k}\text{.}\) Then, find two other points on this line.
    🔗
    Solution.
    We are given a point on the line corresponding to the initial position vector \(\vec{r}_0 = \langle 5,1,3 \rangle\) and a parallel direction vector \(\vec{v} = \vec{i}+4\vec{j}-2\vec{k} = \langle 1,4,-2 \rangle\text{.}\)
    🔗
    The vector equation is given by:
    \begin{equation*} \vec{r} = \vec{r}_0 + t\vec{v} \implies \langle x,y,z \rangle = \langle 5,1,3 \rangle + t\langle 1,4,-2 \rangle \end{equation*}
    🔗
    By separating this expression into independent components, we find the parametric equations:
    \begin{equation*} x = 5 + t, \quad y = 1 + 4t, \quad z = 3 - 2t \end{equation*}
    🔗
    To find two other points on the line, we substitute any two non-zero values for the parameter \(t\text{:}\)
    • Letting \(t = 1\) yields: \(x = 5+1=6\text{,}\) \(y = 1+4(1)=5\text{,}\) and \(z = 3-2(1)=1\text{.}\) This gives the point \((6,5,1)\text{.}\)
      🔗
      🔗
    • Letting \(t = -1\) yields: \(x = 5-1=4\text{,}\) \(y = 1+4(-1)=-3\text{,}\) and \(z = 3-2(-1)=5\text{.}\) This gives the point \((4,-3,5)\text{.}\)
      🔗
      🔗
    🔗
    🔗
    🔗

    Definition 1.83.

    If a vector \(\vec{v} = \langle a,b,c \rangle\) is used to describe the direction of a line \(L\text{,}\) then the numbers \(a\text{,}\) \(b\text{,}\) and \(c\) are called the direction numbers of \(L\text{.}\) Since any vector parallel to \(\vec{v}\) can also be used, any three numbers proportional to \(a\text{,}\) \(b\text{,}\) and \(c\) can serve as a valid set of direction numbers for \(L\text{.}\)
    🔗
    If we solve each of the parametric equations for the parameter \(t\) and equate the results (assuming none of the direction numbers \(a\text{,}\) \(b\text{,}\) or \(c\) are zero), we eliminate \(t\) to obtain:
    \begin{equation*} \frac{x-x_0}{a} = \frac{y-y_0}{b} = \frac{z-z_0}{c} \end{equation*}
    These expressions are called the symmetric equations of \(L\text{.}\)
    🔗
    🔗
    For instance, if a direction number is zero, say \(a=0\text{,}\) we eliminate the parameter \(t\) for the remaining components and write the equations of \(L\) as:
    \begin{equation*} x = x_0, \quad \frac{y-y_0}{b} = \frac{z-z_0}{c} \end{equation*}
    This means geometrically that the line \(L\) lies entirely within the vertical plane \(x=x_0\text{.}\)
    🔗

    Example 1.84.

    Find parametric equations and symmetric equations for the line that passes through the points \((2,4,-3)\) and \((3,-1,1)\text{.}\) At what point does this line intersect the \(xy\)-plane?
    🔗
    Solution.
    First, find a direction vector \(\vec{v}\) for the line by subtracting the coordinates of the first point from the second point:
    \begin{equation*} \vec{v} = \langle 3-2, \, -1-4, \, 1-(-3) \rangle = \langle 1, -5, 4 \rangle \end{equation*}
    🔗
    Using the first point \((2,4,-3)\) as our base position vector \(\vec{r}_0 = \langle 2,4,-3 \rangle\text{,}\) we find the parametric equations:
    \begin{equation*} x = 2 + t, \quad y = 4 - 5t, \quad z = -3 + 4t \end{equation*}
    🔗
    By solving each equation for the parameter \(t\text{,}\) we obtain the matching symmetric equations:
    \begin{equation*} x - 2 = \frac{y - 4}{-5} = \frac{z + 3}{4} \end{equation*}
    🔗
    The line intersects the \(xy\)-plane when its height component is zero (\(z = 0\)). Substitute \(z = 0\) into the symmetric equations to solve for the coordinate positions:
    \begin{equation*} x - 2 = \frac{0 + 3}{4} \implies x = 2 + \frac{3}{4} = \frac{11}{4} \end{equation*}
    \begin{equation*} \frac{y - 4}{-5} = \frac{3}{4} \implies y - 4 = -\frac{15}{4} \implies y = 4 - \frac{15}{4} = \frac{1}{4} \end{equation*}
    Therefore, the point of intersection with the \(xy\)-plane is \(\left(\frac{11}{4}, \frac{1}{4}, 0\right)\text{.}\)
    🔗
    🔗
    🔗

    Corollary 1.85.

    A line segment starting from position vector \(\vec{r}_0\) and ending at position vector \(\vec{r}_1\) has a direction vector defined by \(\vec{v} = \vec{r}_1 - \vec{r}_0\text{.}\) Substituting this into the standard vector equation of a line \(\vec{r} = \vec{r}_0 + t\vec{v}\) yields the vector equation of a line segment:
    \begin{equation*} \vec{r}(t) = (1-t)\vec{r}_0 + t\vec{r}_1, \quad 0 \leq t \leq 1 \end{equation*}
    🔗
    🔗

    Example 1.86.

    Show that the lines \(L_1\) and \(L_2\) with the following parametric equations are skew lines (meaning they do not intersect and are not parallel):
    \begin{align*} L_1: \quad x \amp = 1+t, \quad y = -2+3t, \quad z = 4-t\\ L_2: \quad x \amp = 2s, \quad y = 3+s, \quad z = -3+4s \end{align*}
    🔗
    Solution.
    First, check if the lines are parallel by comparing their direction vectors:
    \begin{equation*} \vec{v}_1 = \langle 1, 3, -1 \rangle \quad \text{and} \quad \vec{v}_2 = \langle 2, 1, 4 \rangle \end{equation*}
    Since the components are not proportional (\(\frac{1}{2} \neq \frac{3}{1}\)), the direction vectors are not scalar multiples of each other. Therefore, the lines are not parallel.
    🔗
    Next, test if the lines intersect by equating their corresponding coordinate expressions to see if a common parameter set \((t,s)\) exists:
    \begin{align*} 1 + t \amp = 2s \quad \text{(Eq. 1)}\\ -2 + 3t \amp = 3 + s \quad \text{(Eq. 2)}\\ 4 - t \amp = -3 + 4s \quad \text{(Eq. 3)} \end{align*}
    🔗
    From Equation 1, we get \(t = 2s - 1\text{.}\) Substitute this expression into Equation 2 to solve for \(s\text{:}\)
    \begin{equation*} -2 + 3(2s - 1) = 3 + s \implies -2 + 6s - 3 = 3 + s \end{equation*}
    \begin{equation*} 5s = 8 \implies s = \frac{8}{5} \end{equation*}
    Using \(s = \frac{8}{5}\text{,}\) we find the value for \(t\text{:}\) \(t = 2\left(\frac{8}{5}\right) - 1 = \frac{16}{5} - \frac{5}{5} = \frac{11}{5}\text{.}\)
    🔗
    Now, substitute both parameters \(s = \frac{8}{5}\) and \(t = \frac{11}{5}\) into Equation 3 to see if they satisfy the vertical height component:
    \begin{equation*} \text{Left Side: } 4 - t = 4 - \frac{11}{5} = \frac{20}{5} - \frac{11}{5} = \frac{9}{5} \end{equation*}
    \begin{equation*} \text{Right Side: } -3 + 4s = -3 + 4\left(\frac{8}{5}\right) = -\frac{15}{5} + \frac{32}{5} = \frac{17}{5} \end{equation*}
    Since \(\frac{9}{5} \neq \frac{17}{5}\text{,}\) the system of linear equations has no solution. This means the lines do not intersect. Because they are neither parallel nor intersecting, \(L_1\) and \(L_2\) are skew lines.
    🔗
    🔗

    Subsection Planes

    A plane in space is uniquely determined by a fixed point \(P_0(x_0,y_0,z_0)\) lying in the plane and a vector \(\vec{n}\) that stands orthogonal to the plane surface. This perpendicular vector \(\vec{n}\) is called a normal vector. Letting \(P(x,y,z)\) be any arbitrary point in the plane, and denoting \(\vec{r}_0\) and \(\vec{r}\) as the standard position vectors of \(P_0\) and \(P\) respectively, the displacement vector \(\vec{r}-\vec{r}_0\) is represented by the geometric line segment \(\vec{P_0P}\text{.}\)
    🔗
    Since the normal vector \(\vec{n}\) is orthogonal to every vector contained within the plane, it must be orthogonal to \(\vec{r}-\vec{r}_0\text{.}\) This orthogonality condition gives us the fundamental vector equation of the plane:
    \begin{equation*} \vec{n}\cdot (\vec{r}-\vec{r}_0) = 0 \iff \vec{n}\cdot \vec{r} = \vec{n}\cdot \vec{r}_0 \end{equation*}
    🔗

    Theorem 1.88. Scalar and Linear Equations of a Plane.

    An equation of the plane passing through \(P_0(x_0,y_0,z_0)\) with normal vector \(\vec{n} = \langle a,b,c \rangle\) is derived by evaluating the dot product component-wise:
    \begin{align*} \langle a,b,c \rangle \cdot (\vec{r}-\vec{r}_0) \amp = 0\\ \langle a,b,c \rangle \cdot \langle x-x_0, y-y_0, z-z_0 \rangle \amp = 0\\ a(x-x_0) + b(y-y_0) + c(z-z_0) \amp = 0 \end{align*}
    By distributing the scalar coefficients and gathering the constants into a single value \(d = -(ax_0 + by_0 + cz_0)\text{,}\) the equation of the plane simplifies to the linear equation in three variables:
    \begin{equation*} ax + by + cz + d = 0 \end{equation*}
    where the coefficients \(a\text{,}\) \(b\text{,}\) and \(c\) are not all zero.
    🔗
    🔗

    Example 1.89.

    Find an equation of the plane through \((5,-2,4)\) with normal vector \(\vec{n} = \langle 1,2,3 \rangle\text{.}\) Find its intercepts and describe how to sketch the plane.
    🔗
    Solution.
    Substitute the given coordinates and normal components directly into the scalar equation formula:
    \begin{equation*} 1(x - 5) + 2(y - (-2)) + 3(z - 4) = 0 \implies x - 5 + 2y + 4 + 3z - 12 = 0 \end{equation*}
    Combining constant terms yields the final linear plane equation:
    \begin{equation*} x + 2y + 3z = 13 \end{equation*}
    🔗
    To calculate the coordinate intercepts, set the other two variables to zero systematically:
    • \(x\)-intercept: Set \(y=0, z=0 \implies x = 13\text{.}\) The point is \((13,0,0)\text{.}\)
      🔗
      🔗
    • \(y\)-intercept: Set \(x=0, z=0 \implies 2y = 13 \implies y = \frac{13}{2}\text{.}\) The point is \(\left(0, \frac{13}{2}, 0\right)\text{.}\)
      🔗
      🔗
    • \(z\)-intercept: Set \(x=0, y=0 \implies 3z = 13 \implies z = \frac{13}{2}\text{.}\) The point is \(\left(0, 0, \frac{13}{3}\right)\text{.}\)
      🔗
      🔗
    To sketch this plane within the first octant, plot these three intercept markers on the coordinate axes and connect them to construct a triangular boundary plane section.
    🔗
    🔗

    Example 1.90.

    Find an equation of the plane determined by the points \(P(4,-3,1)\text{,}\) \(Q(6,-4,7)\text{,}\) and \(R(1,2,2)\text{.}\)
    🔗
    Solution.
    First, construct two displacement vectors sharing vertex \(P\) that lie entirely within the plane:
    \begin{equation*} \vec{PQ} = \langle 6-4, -4-(-3), 7-1 \rangle = \langle 2, -1, 6 \rangle \end{equation*}
    \begin{equation*} \vec{PR} = \langle 1-4, 2-(-3), 2-1 \rangle = \langle -3, 5, 1 \rangle \end{equation*}
    🔗
    A normal vector \(\vec{n}\) for the plane must stand perpendicular to both vector directions. We find it by computing their cross product:
    \begin{align*} \vec{n} = \vec{PQ} \times \vec{PR} \amp = \begin{vmatrix} \vec{i} \amp \vec{j} \amp \vec{k} \\ 2 \amp -1 \amp 6 \\ -3 \amp 5 \amp 1 \end{vmatrix}\\ \amp = (-1 - 30)\vec{i} - (2 - (-18))\vec{j} + (10 - 3)\vec{k}\\ \amp = -31\vec{i} - 20\vec{j} + 7\vec{k} = \langle -31, -20, 7 \rangle \end{align*}
    🔗
    Using point \(P(4,-3,1)\) and our new normal coordinates, assemble the scalar equation layout:
    \begin{equation*} -31(x - 4) - 20(y - (-3)) + 7(z - 1) = 0 \implies -31x + 124 - 20y - 60 + 7z - 7 = 0 \end{equation*}
    Simplifying constants yields the clean linear plane model:
    \begin{equation*} -31x - 20y + 7z + 57 = 0 \quad \text{or} \quad 31x + 20y - 7z = 57 \end{equation*}
    🔗
    🔗
    🔗

    Example 1.91.

    Find the point at which the line with parametric equations \(x = 2+3t\text{,}\) \(y = -4t\text{,}\) \(z = 5+t\) intersects the plane \(4x+5y-2z = 18\text{.}\)
    🔗
    Solution.
    Substitute the parametric component equations of the line directly into the independent variable slots of the linear plane:
    \begin{equation*} 4(2 + 3t) + 5(-4t) - 2(5 + t) = 18 \end{equation*}
    Distribute coefficients and combine terms to isolate the scalar parameter \(t\text{:}\)
    \begin{equation*} 8 + 12t - 20t - 10 - 2t = 18 \implies -10t - 2 = 18 \implies -10t = 20 \implies t = -2 \end{equation*}
    🔗
    Substitute \(t = -2\) back into the original parametric expressions to recover the intersection point coordinates:
    \begin{equation*} x = 2 + 3(-2) = -4, \quad y = -4(-2) = 8, \quad z = 5 + (-2) = 3 \end{equation*}
    Thus, the line intersects the plane at the specific point coordinate \((-4, 8, 3)\text{.}\)
    🔗
    🔗
    🔗

    Definition 1.92.

    Two distinct planes containing normal vectors \(\vec{n}_1\) and \(\vec{n}_2\) are:
    1. parallel if their corresponding normal vectors are parallel (\(\vec{n}_1 = c\vec{n}_2\) for some scalar \(c\)).
      🔗
      🔗
    2. orthogonal if their corresponding normal vectors are orthogonal (\(\vec{n}_1 \cdot \vec{n}_2 = 0\)).
      🔗
      🔗
    🔗
    🔗

    Example 1.93.

    Find an equation of the plane through \(P(5,-2,4)\) that is parallel to the plane \(3x+y-6z+8 = 0\text{.}\)
    🔗
    Solution.
    Parallel planes share identical directional orientations, meaning we can adopt the normal vector of the reference plane directly. Reading the coefficients from \(3x+y-6z+8=0\) reveals \(\vec{n} = \langle 3,1,-6 \rangle\text{.}\)
    🔗
    Combine this normal direction with the coordinates of point \(P(5,-2,4)\text{:}\)
    \begin{equation*} 3(x - 5) + 1(y - (-2)) - 6(z - 4) = 0 \implies 3x - 15 + y + 2 - 6z + 24 = 0 \end{equation*}
    Simplifying the constants provides the equation of the parallel plane:
    \begin{equation*} 3x + y - 6z + 11 = 0 \end{equation*}
    🔗
    🔗
    🔗

    Example 1.94.

    Find the angle between the planes \(x+y+z = 1\) and \(x-2y+3z = 1\text{.}\) Then, find symmetric equations for the line of intersection \(L\) of these two planes.
    🔗
    Solution.
    The angle \(\theta\) between two planes matches the angle separating their normal vectors, \(\vec{n}_1 = \langle 1,1,1 \rangle\) and \(\vec{n}_2 = \langle 1,-2,3 \rangle\text{:}\)
    \begin{equation*} \vec{n}_1 \cdot \vec{n}_2 = (1)(1) + (1)(-2) + (1)(3) = 2 \end{equation*}
    \begin{equation*} \|\vec{n}_1\| = \sqrt{1^2+1^2+1^2} = \sqrt{3}, \quad \|\vec{n}_2\| = \sqrt{1^2+(-2)^2+3^2} = \sqrt{14} \end{equation*}
    \begin{equation*} \cos(\theta) = \frac{\vec{n}_1 \cdot \vec{n}_2}{\|\vec{n}_1\|\|\vec{n}_2\|} = \frac{2}{\sqrt{42}} \implies \theta = \arccos\left(\frac{2}{\sqrt{42}}\right) \approx 72.0^\circ \text{ (or } 1.26\text{ rad)} \end{equation*}
    🔗
    To track the line of intersection \(L\text{,}\) we first find its direction vector \(\vec{v}\text{,}\) which must be orthogonal to both normal paths:
    \begin{equation*} \vec{v} = \vec{n}_1 \times \vec{n}_2 = \begin{vmatrix} \vec{i} \amp \vec{j} \amp \vec{k} \\ 1 \amp 1 \amp 1 \\ 1 \amp -2 \amp 3 \end{vmatrix} = (1(3) - 1(-2))\vec{i} - (1(3) - 1(1))\vec{j} + (1(-2) - 1(1))\vec{k} = 5\vec{i} - 2\vec{j} - 3\vec{k} \end{equation*}
    \begin{equation*} \vec{v} = \langle 5, -2, -3 \rangle \end{equation*}
    🔗
    Next, we find a specific point on \(L\) by solving the system of plane equations simultaneously. We can set \(z = 0\) to simplify:
    \begin{equation*} x + y = 1 \quad \text{and} \quad x - 2y = 1 \end{equation*}
    Solving this system yields \(x = \frac{3}{2}\) and \(y = -\frac{1}{2}\text{.}\) Thus, the point \(P\left(\frac{3}{2}, -\frac{1}{2}, 0\right)\) lies on the line of intersection.
    🔗
    Finally, we can write the parametric equations for the line \(L\text{:}\)
    \begin{equation*} x = \frac{3}{2} + 5t, \quad y = -\frac{1}{2} - 2t, \quad z = -3t \end{equation*}
    And the symmetric equations are:
    \begin{equation*} \frac{x - \frac{3}{2}}{5} = \frac{y + \frac{1}{2}}{-2} = \frac{z}{-3} \end{equation*}
    🔗
    🔗
    🔗

    Fact 1.95. Symmetric Forms and Intersecting Planes.

    An arbitrary spatial point \(P(x,y,z)\) lies on a line \(L\) if and only if its coordinates satisfy the standard symmetric equations:
    \begin{equation*} \frac{x-x_1}{a} = \frac{y-y_1}{b} = \frac{z-z_1}{c} \end{equation*}
    We can interpret this line structural model as the geometric intersection path generated by two distinct, non-parallel structural planes:
    \begin{equation*} \frac{x-x_1}{a} = \frac{y-y_1}{b} \quad \text{and} \quad \frac{y-y_1}{b} = \frac{z-z_1}{c} \end{equation*}
    🔗
    🔗

    Example 1.96.

    Find a symmetric form for the line passing through \(P(3,1,-2)\) and \(Q(-2,7,-4)\text{.}\)
    🔗
    Solution.
    Calculate the baseline direction numbers by computing the displacement vector components from \(P\) to \(Q\text{:}\)
    \begin{equation*} \vec{v} = \langle -2-3, \, 7-1, \, -4-(-2) \rangle = \langle -5, 6, -2 \rangle \end{equation*}
    🔗
    Using coordinates from vertex \(P(3,1,-2)\text{,}\) populate the symmetric structural denominators:
    \begin{equation*} \frac{x - 3}{-5} = \frac{y - 1}{6} = \frac{z + 2}{-2} \end{equation*}
    🔗
    🔗
    🔗

    Example 1.97.

    Find a formula for the minimum straight-line distance \(D\) mapping from an isolated spatial point \(P_1(x_1,y_1,z_1)\) down to a linear plane surface defined by \(ax+by+cz+d=0\text{.}\)
    🔗
    Solution.
    Let \(P_0(x_0,y_0,z_0)\) be any arbitrary known point lying inside the given plane equation, which means it satisfies \(ax_0 + by_0 + cz_0 + d = 0 \implies d = -ax_0 - by_0 - cz_0\text{.}\) Construct a direction vector \(\vec{b}\) corresponding to the directed line segment \(\vec{P_0P_1}\text{:}\)
    \begin{equation*} \vec{b} = \langle x_1-x_0, \, y_1-y_0, \, z_1-z_0 \rangle \end{equation*}
    🔗
    The minimum distance \(D\) from point \(P_1\) to the plane surface equals the absolute scalar projection length of this displacement vector \(\vec{b}\) projected onto the plane’s normal vector direction \(\vec{n} = \langle a,b,c \rangle\text{:}\)
    \begin{align*} D \amp = |\operatorname{comp}_{\vec{n}}\vec{b}| = \frac{|\vec{n}\cdot\vec{b}|}{\|\vec{n}\|}\\ \amp = \frac{|a(x_1-x_0) + b(y_1-y_0) + c(z_1-z_0)|}{\sqrt{a^2+b^2+c^2}}\\ \amp = \frac{|ax_1 + by_1 + cz_1 - (ax_0 + by_0 + cz_0)|}{\sqrt{a^2+b^2+c^2}} \end{align*}
    Substituting the constant expression \(d = -ax_0 - by_0 - cz_0\) into the numerator yields the standard point-plane distance formula:
    \begin{equation*} D = \frac{|ax_1 + by_1 + cz_1 + d|}{\sqrt{a^2+b^2+c^2}} \end{equation*}
    🔗
    🔗

    Checkpoint 1.98.

    Find the distance between the parallel planes \(10x+2y-2z = 5\) and \(5x+y-z = 1\text{.}\)
    🔗
    Solution.
    To find the distance between parallel planes, pick an arbitrary test point on one plane and compute its distance to the other plane.
    🔗
    Let us find a point on the second plane \(5x+y-z=1\) by setting \(x=0\) and \(z=0\text{,}\) which gives \(y=1\text{.}\) This identifies our test point as \(P_1(0,1,0)\text{.}\)
    🔗
    Now, write the first plane in standard linear form: \(10x+2y-2z-5=0\text{.}\) Apply the coordinates of \(P_1(0,1,0)\) to the distance formula:
    \begin{equation*} D = \frac{|10(0) + 2(1) - 2(0) - 5|}{\sqrt{10^2 + 2^2 + (-2)^2}} = \frac{|2 - 5|}{\sqrt{100 + 4 + 4}} = \frac{|-3|}{\sqrt{108}} = \frac{3}{6\sqrt{3}} = \frac{1}{2\sqrt{3}} = \frac{\sqrt{3}}{6} \end{equation*}
    The exact distance separating the two planes is \(\frac{\sqrt{3}}{6}\text{.}\)

    Checkpoint 1.99.

    We showed that the lines \(L_1\) and \(L_2\) with the following parametric equations are skew lines:
    \begin{align*} L_1: \quad x \amp = 1+t, \quad y = -2+3t, \quad z = 4-t\\ L_2: \quad x \amp = 2s, \quad y = 3+s, \quad z = -3+4s \end{align*}
    Find the distance between them.
    🔗
    Solution.
    The minimum distance between two skew lines equals the distance between two parallel planes that enclose the lines. The normal vector \(\vec{n}\) for these planes must stand perpendicular to both line direction vectors: \(\vec{v}_1 = \langle 1,3,-1 \rangle\) and \(\vec{v}_2 = \langle 2,1,4 \rangle\text{.}\)
    \begin{equation*} \vec{n} = \vec{v}_1 \times \vec{v}_2 = \begin{vmatrix} \vec{i} \amp \vec{j} \amp \vec{k} \\ 1 \amp 3 \amp -1 \\ 2 \amp 1 \amp 4 \end{vmatrix} = (12 - (-1))\vec{i} - (4 - (-2))\vec{j} + (1 - 6)\vec{k} = \langle 13, -6, -5 \rangle \end{equation*}
    🔗
    Find a point on line \(L_1\) by setting \(t=0\text{,}\) giving \(P_1(1,-2,4)\text{.}\) Find a point on line \(L_2\) by setting \(s=0\text{,}\) giving \(P_2(0,3,-3)\text{.}\)
    🔗
    Construct a plane passing through \(P_2(0,3,-3)\) using our calculated normal vector components:
    \begin{equation*} 13(x - 0) - 6(y - 3) - 5(z - (-3)) = 0 \implies 13x - 6y - 5z + 3 = 0 \end{equation*}
    🔗
    Finally, calculate the distance from point \(P_1(1,-2,4)\) on line \(L_1\) to this new plane container:
    \begin{equation*} D = \frac{|13(1) - 6(-2) - 5(4) + 3|}{\sqrt{13^2 + (-6)^2 + (-5)^2}} = \frac{|13 + 12 - 20 + 3|}{\sqrt{169 + 36 + 25}} = \frac{8}{\sqrt{230}} \end{equation*}
    The minimum distance separating the two skew lines is exactly \(\frac{8}{\sqrt{230}}\text{.}\)

    Checkpoint 1.100.

    Find the distance from the point \((1,1,2)\) to the plane \(3x+2y+6z = 6\text{.}\)
    🔗
    Solution.
    First, rewrite the plane equation in standard linear form: \(3x+2y+6z-6=0\text{.}\) This provides coefficients of \(a=3, b=2, c=6,\) and \(d=-6\text{.}\)
    🔗
    Substitute these parameters along with the target point coordinates \((x_1,y_1,z_1) = (1,1,2)\) into the point-plane distance formula:
    \begin{equation*} D = \frac{|3(1) + 2(1) + 6(2) - 6|}{\sqrt{3^2 + 2^2 + 6^2}} = \frac{|3 + 2 + 12 - 6|}{\sqrt{9 + 4 + 36}} = \frac{|11|}{\sqrt{49}} = \frac{11}{7} \end{equation*}
    The minimum straight-line distance from the point to the plane is exactly \(\frac{11}{7}\text{.}\)
