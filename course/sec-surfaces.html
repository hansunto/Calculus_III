    Section 1.6 Surfaces

    In this section, we will explore the geometry of surfaces in three-dimensional space. We will learn how to represent surfaces using vector equations, parametric equations, and scalar equations. Additionally, we will discuss the relationships between surfaces and other geometric objects.
    🔗

    Definition 1.101.

    A function \(f\) of two variables is a rule that assigns to each ordered pair of real numbers \((x,y)\) in a set \(D\) a unique real number denoted by \(f(x,y)\text{.}\) The set \(D\) is the domain of \(f\) and its range is the set of values that \(f\) takes on, that is:
    \begin{equation*} \{f(x,y) \mid (x,y) \in D\} \end{equation*}
    🔗
    🔗
    We often write \(z = f(x,y)\) to make explicit the value taken on by \(f\) at the general point \((x,y)\text{.}\) The domain is a subset of \(\mathbb{R}^2\text{,}\) the \(xy\)-plane.
    🔗

    Example 1.102.

    If \(f(x,y) = 4x^2 + y^2\text{,}\) determine its domain and range.
    🔗
    Solution.
    The expression \(4x^2+y^2\) is defined for all possible ordered pairs of real numbers \((x,y)\text{.}\) Therefore, the domain is \(\mathbb{R}^2\text{,}\) which represents the entire \(xy\)-plane.
    🔗
    Since the terms \(x^2\) and \(y^2\) are always non-negative for any real numbers \(x\) and \(y\text{,}\) their linear combination \(4x^2+y^2\) must also be greater than or equal to zero. The function achieves its absolute minimum value of \(0\) exactly at the origin \((0,0)\text{.}\) Thus, the range of \(f\) is the interval \([0,\infty)\) of all non-negative real numbers.
    🔗
    🔗
    🔗

    Definition 1.104.

    If \(f\) is a function of two variables with domain \(D\text{,}\) then the graph of \(f\) is the set of all points \((x,y,z)\) in \(\mathbb{R}^3\) such that \(z = f(x,y)\) and \((x,y)\) is in \(D\text{.}\)
    🔗
    🔗

    Example 1.105.

    Sketch the graph of each of the following functions or equations in \(\mathbb{R}^3\text{:}\)
    🔗
    1. \(\displaystyle f(x,y) = 6-3x-2y\)
      🔗
      🔗
    2. An elliptic cylinder: \(\frac{x^2}{4}+\frac{y^2}{9}=1\)
      🔗
      🔗
    3. A parabolic cylinder: \(f(x,y)=x^2\)
      🔗
      🔗
    4. An ellipsoid: \(x^2+\frac{y^2}{9}+\frac{z^2}{4}=1\)
      🔗
      🔗
    Solution.
    1. Setting \(z = f(x,y)\) gives the linear equation \(z = 6-3x-2y \implies 3x + 2y + z = 6\text{.}\) This represents a flat plane in space. We can find its coordinate axis intercepts to help sketch it:
      • \(x\)-intercept: Set \(y=0, z=0 \implies 3x=6 \implies x=2\text{.}\) Point: \((2,0,0)\text{.}\)
        🔗
        🔗
      • \(y\)-intercept: Set \(x=0, z=0 \implies 2y=6 \implies y=3\text{.}\) Point: \((0,3,0)\text{.}\)
        🔗
        🔗
      • \(z\)-intercept: Set \(x=0, y=0 \implies z=6\text{.}\) Point: \((0,0,6)\text{.}\)
        🔗
        🔗
      Plotting these three points and connecting them with a triangular frame yields the trace of the plane in the first octant.
      🔗
      🔗
    2. The equation \(\frac{x^2}{4}+\frac{y^2}{9}=1\) restricts the relationship between \(x\) and \(y\) to an ellipse centered at the origin in the \(xy\)-plane, with major axis radius \(3\) along the \(y\)-axis and minor axis radius \(2\) along the \(x\)-axis. Because the variable \(z\) is completely missing from the equation, it is unconstrained and can take any real value. This projects the ellipse vertically to form an infinitely tall elliptic cylinder parallel to the \(z\)-axis.
      🔗
      🔗
    3. Setting \(z = f(x,y) = x^2\) provides a parabolic trace in the vertical \(xz\)-plane opening upwards from the origin. Since the variable \(y\) is absent, any vertical plane intersection cross-section matching \(y=k\) produces an identical copy of this parabola shifted along the horizontal axis. Connecting these parallel parabolic traces generates a sheet-like parabolic cylinder extending along the direction of the \(y\)-axis.
      🔗
      🔗
    4. The equation \(x^2+\frac{y^2}{9}+\frac{z^2}{4}=1\) matches the standard form of an ellipsoid centered at the origin: \(\frac{x^2}{a^2}+\frac{y^2}{b^2}+\frac{z^2}{c^2}=1\) with radii \(a=1\text{,}\) \(b=3\text{,}\) and \(c=2\text{.}\) Its traces in all three coordinate planes are ellipses:
      • \(xy\)-plane trace (\(z=0\)): \(x^2 + \frac{y^2}{9} = 1\) (an ellipse of radii 1 and 3).
        🔗
        🔗
      • \(xz\)-plane trace (\(y=0\)): \(x^2 + \frac{z^2}{4} = 1\) (an ellipse of radii 1 and 2).
        🔗
        🔗
      • \(yz\)-plane trace (\(x=0\)): \(\frac{y^2}{9} + \frac{z^2}{4} = 1\) (an ellipse of radii 3 and 2).
        🔗
        🔗
      Sketching these three cross-sectional traces outlines a 3D egg-like oval surface shell.
      🔗
      🔗
    🔗
    🔗
    If we keep \(x\) fixed by putting \(x=k\) and letting \(y\) vary, the resulting function is a function of one variable, \(z=f(k,y)\text{,}\) whose graph is the curve that results when we intersect the surface \(z=f(x,y)\) with the vertical plane \(x=k\text{.}\) This type of curve is called a trace of the surface. Standard quadric surfaces classified by their traces include:
    🔗
    • A linear function: \(f(x,y)=ax+by+c\)
      🔗
      🔗
    • An elliptic paraboloid: \(\frac{z}{c}=\frac{x^2}{a^2}+\frac{y^2}{b^2}\)
      🔗
      🔗
    • A hyperbolic paraboloid: \(\frac{z}{c}=\frac{x^2}{a^2}-\frac{y^2}{b^2}\)
      🔗
      🔗
    • An ellipsoid: \(\frac{x^2}{a^2}+\frac{y^2}{b^2}+\frac{z^2}{c^2}=1\)
      🔗
      🔗
    • A hyperboloid of one sheet: \(\frac{x^2}{a^2}+\frac{y^2}{b^2}-\frac{z^2}{c^2}=1\)
      🔗
      🔗
    • A hyperboloid of two sheets: \(-\frac{x^2}{a^2}-\frac{y^2}{b^2}+\frac{z^2}{c^2}=1\)
      🔗
      🔗
    • An elliptic cone: \(\frac{z^2}{c^2}=\frac{x^2}{a^2}+\frac{y^2}{b^2}\)
      🔗
      🔗

    Example 1.117.

    Classify the surface \(x^2+2z^2-6x-y+10=0\text{.}\)
    🔗
    Solution.
    To classify the surface, we isolate the linear variable \(y\) and complete the square for the \(x\) terms:
    \begin{align*} y \amp = x^2 - 6x + 2z^2 + 10\\ y \amp = (x^2 - 6x + 9) + 2z^2 + 10 - 9\\ y - 1 \amp = (x - 3)^2 + 2z^2 \end{align*}
    🔗
    We can rewrite this in standard form as:
    \begin{equation*} y - 1 = \frac{(x - 3)^2}{1} + \frac{z^2}{1/2} \end{equation*}
    Comparing this to our standard models, this equation represents an elliptic paraboloid. Its vertex is shifted to the coordinates \((3, 1, 0)\text{,}\) and it opens along the positive direction of the \(y\)-axis.
