    Section 2.4 Arc Length and Curvature

    In this section, we will discuss the concept of arc length for curves defined by vector-valued functions. The arc length of a curve is the distance along the curve between two points. We will derive a formula for computing the arc length of a curve defined by a vector function and provide examples to illustrate the process.
    🔗

    Definition 2.39. Arc Length of a Curve.

    The length \(L\) of a smooth curve \(C\) traced out by a vector function \(\mathbf{r}(t)=x(t)\mathbf{i}+y(t)\mathbf{j}+z(t)\mathbf{k}\) on the interval \(a \le t \le b\) is defined as:
    \begin{equation*} L = \int^b_a\sqrt{[x'(t)]^2+[y'(t)]^2+[z'(t)]^2}\, dt = \int^b_a\sqrt{\left[\frac{dx}{dt}\right]^2+\left[\frac{dy}{dt}\right]^2+\left[\frac{dz}{dt}\right]^2}\, dt = \int^b_a|\mathbf{r}'(t)|\, dt \end{equation*}
    🔗
    🔗

    Example 2.40.

    Find the length of the arc of the circular helix with vector equation:
    \begin{equation*} \mathbf{r}(t)=\cos t\ \mathbf{i}+\sin t\ \mathbf{j}+t\ \mathbf{k} \end{equation*}
    from the point \((1,0,0)\) to the point \((1,0,2\pi)\text{.}\)
    🔗
    Solution.
    First, we determine the parameter limits \(t = a\) and \(t = b\) corresponding to the given points. Comparing the \(\mathbf{k}\)-component \(z = t\) to the points yields:
    \begin{align*} \text{At } (1,0,0) \amp \implies t = 0\\ \text{At } (1,0,2\pi) \amp \implies t = 2\pi \end{align*}
    Next, we compute the derivative vector function \(\mathbf{r}'(t)\) and its magnitude:
    \begin{equation*} \mathbf{r}'(t) = \langle -\sin t, \cos t, 1 \rangle \end{equation*}
    \begin{equation*} |\mathbf{r}'(t)| = \sqrt{(-\sin t)^2 + (\cos t)^2 + 1^2} = \sqrt{\sin^2 t + \cos^2 t + 1} = \sqrt{1 + 1} = \sqrt{2} \end{equation*}
    🔗
    Applying the arc length formula over the interval \([0, 2\pi]\) gives:
    \begin{equation*} L = \int_{0}^{2\pi} |\mathbf{r}'(t)|\, dt = \int_{0}^{2\pi} \sqrt{2}\, dt = \left[ \sqrt{2}t \right]_{0}^{2\pi} = 2\pi\sqrt{2} \end{equation*}
    🔗
    🔗
    🔗

    Checkpoint 2.41.

    A curve \(C\) can be represented by more than one vector function. For instance, the twisted cubic can be parameterized as:
    \begin{equation*} \mathbf{r}_1(t)=t\,\mathbf{i}+ t^2\,\mathbf{j}+t^3\,\mathbf{k} \quad \text{for } 1 \le t \le 2 \end{equation*}
    or as:
    \begin{equation*} \mathbf{r}_2(u)=e^u\,\mathbf{i}+e^{2u}\,\mathbf{j}+e^{3u}\,\mathbf{k} \quad \text{for } 0 \le u \le \ln 2 \end{equation*}
    Therefore, the vector functions representing a space curve are not uniquely determined.
    🔗
    Solution.
    To show that \(\mathbf{r}_1(t)\) and \(\mathbf{r}_2(u)\) trace out the exact same geometric path in space, we can perform a change of variables (reparameterization).
    🔗
    Let us define a substitution relationship between the two parameters \(t\) and \(u\text{:}\)
    \begin{equation*} t = e^u \end{equation*}
    If we take the natural logarithm of both sides, this is equivalent to writing \(u = \ln t\text{.}\)
    🔗
    Now, we verify two things to ensure the representations are identical: the component equations and the domain boundaries.
    🔗
    1. Component Transformation: Substitute \(t = e^u\) directly into the component equations of \(\mathbf{r}_1(t)\text{:}\)
      \begin{align*} x(t) \amp = t \implies x(u) = e^u\\ y(t) \amp = t^2 \implies y(u) = (e^u)^2 = e^{2u}\\ z(t) \amp = t^3 \implies z(u) = (e^u)^3 = e^{3u} \end{align*}
      This matches the vector structure of \(\mathbf{r}_2(u) = \langle e^u, e^{2u}, e^{3u} \rangle\) perfectly.
      🔗
      🔗
    2. Domain Mapping: We must map the parameter boundaries from \(t\) to \(u\) using our transition formula \(u = \ln t\text{:}\)
      \begin{align*} \text{Lower bound: } t = 1 \amp \implies u = \ln(1) = 0\\ \text{Upper bound: } t = 2 \amp \implies u = \ln(2) \end{align*}
      This confirms that as \(t\) travels over the continuous interval \([1, 2]\text{,}\) the new parameter \(u\) travels exactly over the continuous interval \([0, \ln 2]\text{.}\)
      🔗
      🔗
    Both vector functions start at the point \((1,1,1)\) and terminate at the point \((2,4,8)\text{,}\) tracking the exact same space path. This illustrates that a single space curve can have infinitely many valid parametric representations.
    🔗
    🔗
    🔗
    We say that those equations are parameterizations of the curve \(C\text{.}\)
    🔗

    Definition 2.42.

    Suppose that \(C\) is a piecewise-smooth curve given by a vector function \(\mathbf{r}(t)=f(t)\mathbf{i}+g(t)\mathbf{j}+h(t)\mathbf{k}\) for \(a \le t \le b\text{,}\) and \(C\) is traversed exactly once as \(t\) increases from \(a\) to \(b\text{.}\) The arc length function \(s\) is defined as:
    \begin{equation*} s(t)=\int^t_a|\mathbf{r}'(u)|\, du = \int^t_a\sqrt{\left[\frac{dx}{du}\right]^2+\left[\frac{dy}{du}\right]^2+\left[\frac{dz}{du}\right]^2}\, du \end{equation*}
    So, \(s(t)\) represents the accumulated length of the part of \(C\) between the starting point \(\mathbf{r}(a)\) and the variable position \(\mathbf{r}(t)\text{.}\)
    🔗
    🔗
    By using the Fundamental Theorem of Calculus Part 1 to differentiate the arc length function \(s(t)\text{,}\) we get:
    \begin{equation*} \frac{ds}{dt}=|\mathbf{r}'(t)| \end{equation*}
    This structural relationship allows us to parametrize a curve with respect to arc length. Arc length parameterization arises naturally from the geometry of the curve itself and remains invariant across all chosen external coordinate systems.
    🔗

    Example 2.43.

    Reparametrize the helix \(\mathbf{r}(t)=\cos t\ \mathbf{i}+\sin t\ \mathbf{j}+t\ \mathbf{k}\) with respect to arc length measured from the point \((1,0,0)\) in the direction of increasing \(t\) (a positive orientation).
    🔗
    Solution.
    The initial baseline point \((1,0,0)\) maps directly to the parameter value \(t=0\text{.}\) Using our previous evaluation, we know that:
    \begin{equation*} \frac{ds}{dt}=|\mathbf{r}'(t)|=\sqrt{(-\sin t)^2 + (\cos t)^2 + 1^2} = \sqrt{2} \end{equation*}
    Now, compute the active arc length function \(s(t)\) by integration:
    \begin{equation*} s = s(t) = \int^t_0|\mathbf{r}'(u)|\, du = \int^t_0\sqrt{2} \, du = \sqrt{2}t \end{equation*}
    Isolating the original domain parameter yields the substitution profile \(t=\frac{s}{\sqrt{2}}\text{.}\) Substituting this back into our primary position function gives the desired arc length parameterization:
    \begin{equation*} \mathbf{r}(t(s)) = \left\langle \cos\left(\frac{s}{\sqrt{2}}\right), \ \sin\left(\frac{s}{\sqrt{2}}\right), \ \frac{s}{\sqrt{2}} \right\rangle \end{equation*}
    🔗
    🔗
    🔗

    Checkpoint 2.44.

    Find the arc length parametrization for a circle \(x^2+y^2=2^2\text{.}\)
    🔗
    Solution.
    Let \(A(2,0)\) represent the fixed initial coordinate point and let \(P(x,y)\) match any coordinate along the circle. The standard parametric representation tracking counterclockwise rotation gives:
    \begin{equation*} x=2\cos \theta, \quad y=2\sin \theta \quad \text{for } 0 \le \theta \le 2\pi \end{equation*}
    The magnitude of the derivative tracking vector evaluates to:
    \begin{equation*} |\mathbf{r}'(\theta)| = |\langle -2\sin\theta, 2\cos\theta \rangle| = \sqrt{4\sin^2\theta + 4\cos^2\theta} = 2 \end{equation*}
    Setting up our accumulated arc length function \(s\) yields:
    \begin{equation*} s(\theta) = \int^\theta_0 2\, du = 2\theta \implies \theta = \frac{s}{2} \end{equation*}
    Mapping the boundaries into the new parametric frame maps \(\theta=0 \implies s=0\) and \(\theta=2\pi \implies s=4\pi\text{.}\) Substituting the transformation pattern into our vector structure outputs:
    \begin{equation*} \mathbf{r}(\theta(s)) = \left\langle 2\cos\left(\frac{s}{2}\right), \ 2\sin\left(\frac{s}{2}\right) \right\rangle \quad \text{for } 0 \le s \le 4\pi \end{equation*}
    🔗
    🔗
    🔗
    If \(\mathbf{r}(t)\) is a vector-valued function and the space curve \(C\) determined by \(\mathbf{r}(t)\) is smooth, then \(\mathbf{r}'(t)\) represents a tangent vector pointing in the direction of the curve’s orientation. Provided \(\mathbf{r}'(t) \neq \mathbf{0}\text{,}\) the unit tangent vector \(\mathbf{T}(t)\) is defined as:
    \begin{equation*} \mathbf{T}(t)=\frac{\mathbf{r}'(t)}{|\mathbf{r}'(t)|} \end{equation*}
    Because \(|\mathbf{T}(t)|=1\) remains constant for all values of \(t\text{,}\) if \(\mathbf{T}'(t)\) is differentiable, then \(\mathbf{T}'(t)\) must be orthogonal to \(\mathbf{T}(t)\text{.}\)
    🔗

    The Unit Normal Vector.

    The unit normal vector \(\mathbf{N}(t)\) to \(C\) is defined as the unit vector pointing in the same direction as \(\mathbf{T}'(t)\text{:}\)
    \begin{equation*} \mathbf{N}(t)= \frac{\mathbf{T}'(t)}{|\mathbf{T}'(t)|} \end{equation*}
    🔗
    🔗

    Example 2.45.

    Let \(C\) be a curve determined by the vector function \(\mathbf{r}(t)=t^2\mathbf{i}+t\mathbf{j}\text{.}\)
    🔗
    1. Find the unit tangent and unit normal vectors \(\mathbf{T}(t)\) and \(\mathbf{N}(t)\text{.}\)
      🔗
      🔗
    2. Find \(\mathbf{T}(1)\) and \(\mathbf{N}(1)\text{.}\)
      🔗
      🔗
    3. Check the orthogonality of \(\mathbf{T}(t)\) and \(\mathbf{N}(t)\text{.}\)
      🔗
      🔗
    Solution.
    1. First, differentiate the position vector function to locate our general velocity vector profile:
      \begin{equation*} \mathbf{r}'(t) = 2t\mathbf{i} + \mathbf{j} \implies |\mathbf{r}'(t)| = \sqrt{4t^2 + 1} \end{equation*}
      Dividing the derivative vector by its magnitude gives the unit tangent vector function:
      \begin{equation*} \mathbf{T}(t) = \frac{2t}{\sqrt{4t^2+1}}\mathbf{i} + \frac{1}{\sqrt{4t^2+1}}\mathbf{j} \end{equation*}
      Next, differentiate \(\mathbf{T}(t)\) using the quotient rule or chain rule to find the normal vector direction:
      \begin{equation*} \mathbf{T}'(t) = \frac{2(4t^2+1)^{-1/2} - 2t \cdot \frac{1}{2}(4t^2+1)^{-1/2}(8t)}{4t^2+1}\mathbf{i} + \left(-\frac{1}{2}(4t^2+1)^{-3/2}(8t)\right)\mathbf{j} \end{equation*}
      \begin{equation*} \mathbf{T}'(t) = \frac{2}{(4t^2+1)^{3/2}}\mathbf{i} - \frac{4t}{(4t^2+1)^{3/2}}\mathbf{j} \implies |\mathbf{T}'(t)| = \frac{\sqrt{4 + 16t^2}}{(4t^2+1)^{3/2}} = \frac{2}{\sqrt{4t^2+1}} \end{equation*}
      Normalizing this vector tracking profile produces our final unit normal vector function:
      \begin{equation*} \mathbf{N}(t) = \frac{\mathbf{T}'(t)}{|\mathbf{T}'(t)|} = \frac{1}{\sqrt{4t^2+1}}\mathbf{i} - \frac{2t}{\sqrt{4t^2+1}}\mathbf{j} \end{equation*}
      🔗
      🔗
    2. Evaluating both unit vector function equations at our specific target parameter point \(t=1\) yields:
      \begin{equation*} \mathbf{T}(1) = \frac{2}{\sqrt{5}}\mathbf{i} + \frac{1}{\sqrt{5}}\mathbf{j} = \left\langle \frac{2}{\sqrt{5}}, \frac{1}{\sqrt{5}} \right\rangle \end{equation*}
      \begin{equation*} \mathbf{N}(1) = \frac{1}{\sqrt{5}}\mathbf{i} - \frac{2}{\sqrt{5}}\mathbf{j} = \left\langle \frac{1}{\sqrt{5}}, -\frac{2}{\sqrt{5}} \right\rangle \end{equation*}
      🔗
      🔗
    3. To check orthogonality, we compute the algebraic dot product of our two unit vector functions:
      \begin{equation*} \mathbf{T}(t) \cdot \mathbf{N}(t) = \left(\frac{2t}{\sqrt{4t^2+1}}\right)\left(\frac{1}{\sqrt{4t^2+1}}\right) + \left(\frac{1}{\sqrt{4t^2+1}}\right)\left(-\frac{2t}{\sqrt{4t^2+1}}\right) \end{equation*}
      \begin{equation*} \mathbf{T}(t) \cdot \mathbf{N}(t) = \frac{2t}{4t^2+1} - \frac{2t}{4t^2+1} = 0 \end{equation*}
      Because their dot product evaluates to exactly zero for all values of \(t\text{,}\) the unit tangent vector \(\mathbf{T}(t)\) and unit normal vector \(\mathbf{N}(t)\) are verified as orthogonal.
      🔗
      🔗
    🔗
    🔗

    Checkpoint 2.46.

    Let \(C\) be determined by the helix \(\mathbf{r}(t)=\cos t\ \mathbf{i}+\sin t\ \mathbf{j}+t\ \mathbf{k}\) for \(t \ge 0\text{.}\) Find the unit vector functions \(\mathbf{T}(t)\) and \(\mathbf{N}(t)\text{.}\)
    🔗
    Solution.
    Differentiate the helix position components vector to extract our general velocity direction path:
    \begin{equation*} \mathbf{r}'(t) = \langle -\sin t, \cos t, 1 \rangle \implies |\mathbf{r}'(t)| = \sqrt{\sin^2 t + \cos^2 t + 1} = \sqrt{2} \end{equation*}
    Dividing the derivative components by this magnitude gives our unit tangent vector function:
    \begin{equation*} \mathbf{T}(t) = \frac{\mathbf{r}'(t)}{|\mathbf{r}'(t)|} = \left\langle -\frac{1}{\sqrt{2}}\sin t, \frac{1}{\sqrt{2}}\cos t, \frac{1}{\sqrt{2}} \right\rangle \end{equation*}
    🔗
    Next, differentiate our unit tangent function with respect to \(t\) to trace the geometric turn path:
    \begin{equation*} \mathbf{T}'(t) = \left\langle -\frac{1}{\sqrt{2}}\cos t, -\frac{1}{\sqrt{2}}\sin t, 0 \right\rangle \end{equation*}
    Computing the magnitude of this secondary differentiation vector yields:
    \begin{equation*} |\mathbf{T}'(t)| = \sqrt{\left(-\frac{1}{\sqrt{2}}\cos t\right)^2 + \left(-\frac{1}{\sqrt{2}}\sin t\right)^2 + 0^2} = \sqrt{\frac{1}{2}(\cos^2 t + \sin^2 t)} = \frac{1}{\sqrt{2}} \end{equation*}
    Finally, normalize \(\mathbf{T}'(t)\) to calculate our primary unit normal vector function:
    \begin{equation*} \mathbf{N}(t) = \frac{\mathbf{T}'(t)}{|\mathbf{T}'(t)|} = \frac{1}{1/\sqrt{2}}\left\langle -\frac{1}{\sqrt{2}}\cos t, -\frac{1}{\sqrt{2}}\sin t, 0 \right\rangle = \langle -\cos t, -\sin t, 0 \rangle \end{equation*}
    This shows that for a circular helix, the unit normal vector always points directly inward toward the central vertical axis of the cylinder.
    🔗
    🔗

    Subsection The Curvature

    The curvature of a curve \(C\) at a given point is a measure of how quickly the curve changes direction at that point, or how much the curve bends at each point.
    🔗

    Definition 2.47. Curvature.

    The curvature of a curve is defined as:
    \begin{equation*} \kappa=\left|\frac{d\mathbf{T}}{ds}\right| \end{equation*}
    where \(\mathbf{T}\) is the unit tangent vector. Curvature is the magnitude of the rate of change of the unit tangent vector with respect to arc length.
    🔗
    🔗

    Theorem 2.48. Formulas for Curvature.

    Let \(C\) be a smooth curve and let \(\theta\) be the angle between the unit tangent vector \(\mathbf{T}\) and the standard basis vector \(\mathbf{i}\text{.}\)
    🔗
    1. By applying the chain rule to the unit tangent vector with respect to time \(t\text{:}\)
      \begin{equation*} \frac{d\mathbf{T}}{dt}=\frac{d\mathbf{T}}{ds}\frac{ds}{dt}, \quad \text{and} \quad \kappa=\left|\frac{d\mathbf{T}}{ds}\right|=\left|\frac{d\mathbf{T}/dt}{ds/dt}\right| \end{equation*}
      Since \(\frac{ds}{dt}=|\mathbf{r}'(t)|\text{,}\) the curvature as a function of the parameter \(t\) simplifies to:
      \begin{equation*} \kappa(t)=\frac{|\mathbf{T}'(t)|}{|\mathbf{r}'(t)|} \end{equation*}
      🔗
      🔗
    2. If a 2D curve has an arc length parameterization given by \(x=f(s)\) and \(y=g(s)\text{,}\) then the curvature \(\kappa\) at a point \(P(x,y)\) is given geometrically by:
      \begin{equation*} \kappa=\left|\frac{d\theta}{ds}\right| \end{equation*}
      🔗
      🔗
    3. Suppose a 2D curve is defined explicitly by a Cartesian function \(y=f(x)\text{,}\) where \(f'(x)\) is continuous on an interval. Since \(y'\) is the slope of the tangent line at \(P(x,y)\text{,}\) we have \(\tan \theta=y'\) (or \(\theta=\tan^{-1}y'\)), and the arc length function is \(s(x)=\int^x_a\sqrt{1+(y')^2}\, dx\text{.}\) If \(y''\) exists, applying the chain rule yields:
      \begin{equation*} \frac{d\theta}{dx} = \frac{d}{dx}(\tan^{-1}y') = \frac{y''}{1+(y')^2} \quad \text{and} \quad \frac{ds}{dx} = \sqrt{1+(y')^2} \end{equation*}
      Thus, the curvature \(\kappa\) at the point \(P(x,y)\) is defined as:
      \begin{equation*} \kappa=\frac{|y''|}{\left(1+(y')^2\right)^{3/2}} \end{equation*}
      🔗
      🔗
    4. Suppose a 2D curve is given parametrically by \(x=f(t)\) and \(y=g(t)\text{,}\) where \(f''\) and \(g''\) exist on an interval \(I\text{.}\) Since the slope of the tangent line is \(\tan \theta = \frac{g'(t)}{f'(t)}\) for \(f'(t) \neq 0\text{,}\) differentiating \(\theta = \tan^{-1}\left(\frac{g'(t)}{f'(t)}\right)\) with respect to \(t\) and dividing by \(\frac{ds}{dt}\) yields:
      \begin{equation*} \kappa =\frac{|f'(t)g''(t)-g'(t)f''(t)|}{\left[(f'(t))^2+(g'(t))^2\right]^{3/2}} \end{equation*}
      🔗
      🔗
    5. For a curve in space defined by a 3D vector function \(\mathbf{r}(t)\text{,}\) the curvature can be computed directly without finding the unit tangent vector via the cross product:
      \begin{equation*} \kappa = \frac{|\mathbf{r}'(t)\times\mathbf{r}''(t)|}{|\mathbf{r}'(t)|^3} \end{equation*}
      🔗
      🔗
    🔗

    Example 2.49.

    Show that the curvature of a circle of radius \(a\) is \(1/a\) at every point on the circle using the vector formula from Item 1.
    🔗
    Solution.
    Parameterize the circle of radius \(a\) as \(\mathbf{r}(t) = \langle a\cos t, a\sin t, 0 \rangle\text{.}\) Then, the velocity vector is \(\mathbf{r}'(t) = \langle -a\sin t, a\cos t, 0 \rangle\text{,}\) and its magnitude is \(|\mathbf{r}'(t)| = \sqrt{a^2\sin^2 t + a^2\cos^2 t} = a\text{.}\)
    🔗
    The unit tangent vector is:
    \begin{equation*} \mathbf{T}(t) = \frac{\mathbf{r}'(t)}{|\mathbf{r}'(t)|} = \langle -\sin t, \cos t, 0 \rangle \end{equation*}
    Differentiating with respect to \(t\) gives \(\mathbf{T}'(t) = \langle -\cos t, -\sin t, 0 \rangle\text{,}\) so \(|\mathbf{T}'(t)| = 1\text{.}\) Applying the formula, we find:
    \begin{equation*} \kappa(t) = \frac{|\mathbf{T}'(t)|}{|\mathbf{r}'(t)|} = \frac{1}{a} \end{equation*}
    🔗
    🔗
    🔗

    Example 2.50.

    Show that the curvature of a line \(l\) is \(0\) at every point on \(l\) using the geometric formulation in Item 2.
    🔗
    Solution.
    A straight line maintains a constant direction along its entire trajectory. Therefore, the tangent angle \(\theta\) is constant relative to the \(x\)-axis, which implies \(\frac{d\theta}{ds} = 0\text{.}\) Thus, \(\kappa = |0| = 0\text{.}\)
    🔗
    🔗
    🔗

    Example 2.51.

    Sketch the graph of \(y=1-x^2\) and find the curvature at general points \((x,y)\text{,}\) and specifically at the points \((0,1)\text{,}\) \((1,0)\text{,}\) and \((2,-3)\) using the explicit function method of Item 3.
    🔗
    Solution.
    1. For \(y = 1 - x^2\text{,}\) we compute the first and second derivatives with respect to \(x\text{:}\)
      \begin{equation*} y' = -2x \quad \text{and} \quad y'' = -2 \end{equation*}
      Substituting these into the explicit Cartesian curvature formula yields the general formula for this parabola:
      \begin{equation*} \kappa(x) = \frac{|-2|}{\left(1 + (-2x)^2\right)^{3/2}} = \frac{2}{(1 + 4x^2)^{3/2}} \end{equation*}
      🔗
      🔗
    2. Evaluating this expression at the specific requested coordinates:
      1. At \((0,1)\text{,}\) \(x=0 \implies \kappa(0) = \frac{2}{(1+0)^{3/2}} = 2\text{.}\)
        🔗
        🔗
      2. At \((1,0)\text{,}\) \(x=1 \implies \kappa(1) = \frac{2}{(1+4)^{3/2}} = \frac{2}{5\sqrt{5}}\text{.}\)
        🔗
        🔗
      3. At \((2,-3)\text{,}\) \(x=2 \implies \kappa(2) = \frac{2}{(1+16)^{3/2}} = \frac{2}{17\sqrt{17}}\text{.}\)
        🔗
        🔗
      🔗
      🔗
    🔗
    🔗

    Checkpoint 2.52.

    Prove that the curvature of a circle of radius \(a\) is \(1/a\) at every point on the circle using the 2D parametric formula derived in Item 4.
    🔗
    Solution.
    Using the standard parametric representation \(x = a\cos t\) and \(y = a\sin t\text{,}\) we calculate the first and second derivatives:
    \begin{align*} f'(t) = -a\sin t, \amp \quad g'(t) = a\cos t\\ f''(t) = -a\cos t, \amp \quad g''(t) = -a\sin t \end{align*}
    Substitute these components into the parametric numerator and denominator arrays:
    \begin{equation*} |f'(t)g''(t) - g'(t)f''(t)| = |(-a\sin t)(-a\sin t) - (a\cos t)(-a\cos t)| = a^2\sin^2 t + a^2\cos^2 t = a^2 \end{equation*}
    \begin{equation*} [(f'(t))^2 + (g'(t))^2]^{3/2} = [(-a\sin t)^2 + (a\cos t)^2]^{3/2} = (a^2)^{3/2} = a^3 \end{equation*}
    Dividing these values yields a constant curvature of:
    \begin{equation*} \kappa = \frac{a^2}{a^3} = \frac{1}{a} \end{equation*}

    Checkpoint 2.53.

    A curve \(C\) has the parameterization \(x=t^2\text{,}\) \(y=t^3\) for \(t\in \mathbb{R}\text{.}\) Find the curvature at the point \(P\) corresponding to \(t=1/2\) using the 2D parametric formula derived in Item 4.
    🔗
    Solution.
    Differentiating the component functions gives:
    \begin{align*} f'(t) = 2t, \amp \quad g'(t) = 3t^2\\ f''(t) = 2, \amp \quad g''(t) = 6t \end{align*}
    Substitute these into the 2D parametric curvature formula:
    \begin{equation*} \kappa(t) = \frac{|(2t)(6t) - (3t^2)(2)|}{[(2t)^2 + (3t^2)^2]^{3/2}} = \frac{|12t^2 - 6t^2|}{(4t^2 + 9t^4)^{3/2}} = \frac{6t^2}{(4t^2 + 9t^4)^{3/2}} \end{equation*}
    Evaluating the curvature profile at the requested input parameter \(t = 1/2\text{:}\)
    \begin{equation*} \kappa\left(\frac{1}{2}\right) = \frac{6(1/4)}{\left(4(1/4) + 9(1/16)\right)^{3/2}} = \frac{3/2}{\left(1 + 9/16\right)^{3/2}} = \frac{3/2}{(25/16)^{3/2}} = \frac{3/2}{125/64} = \frac{3}{2} \cdot \frac{64}{125} = \frac{96}{125} \end{equation*}

    Checkpoint 2.54.

    Find the curvature of the twisted cubic \(\mathbf{r}(t)=\langle t,t^2,t^3 \rangle\) at a general parameter point and specifically at the origin \((0,0,0)\) using the cross product formulation of Item 5.
    🔗
    Solution.
    First, compute the velocity vector \(\mathbf{r}'(t)\) and acceleration vector \(\mathbf{r}''(t)\text{:}\)
    \begin{equation*} \mathbf{r}'(t) = \langle 1, 2t, 3t^2 \rangle \implies |\mathbf{r}'(t)| = \sqrt{1 + 4t^2 + 9t^4} \end{equation*}
    \begin{equation*} \mathbf{r}''(t) = \langle 0, 2, 6t \rangle \end{equation*}
    Next, evaluate the vector cross product of these two derivative vectors:
    \begin{equation*} \mathbf{r}'(t) \times \mathbf{r}''(t) = \begin{vmatrix} \mathbf{i} \amp \mathbf{j} \amp \mathbf{k} \\ 1 \amp 2t \amp 3t^2 \\ 0 \amp 2 \amp 6t \end{vmatrix} = (12t^2 - 6t^2)\mathbf{i} - (6t - 0)\mathbf{j} + (2 - 0)\mathbf{k} = \langle 6t^2, -6t, 2 \rangle \end{equation*}
    Find the magnitude of this cross product vector:
    \begin{equation*} |\mathbf{r}'(t) \times \mathbf{r}''(t)| = \sqrt{(6t^2)^2 + (-6t)^2 + 2^2} = \sqrt{36t^4 + 36t^2 + 4} = 2\sqrt{9t^4 + 9t^2 + 1} \end{equation*}
    🔗
    Substituting these components back into our 3D vector curvature equation yields the general expression:
    \begin{equation*} \kappa(t) = \frac{2\sqrt{9t^4 + 9t^2 + 1}}{(1 + 4t^2 + 9t^4)^{3/2}} \end{equation*}
    To find the specific curvature at the origin \((0,0,0)\text{,}\) substitute the parameter value \(t=0\text{:}\)
    \begin{equation*} \kappa(0) = \frac{2\sqrt{0 + 0 + 1}}{(1 + 0 + 0)^{3/2}} = \frac{2(1)}{1} = 2 \end{equation*}
