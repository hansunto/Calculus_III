    Section 3.5 Chain Rules

    The chain rule is a fundamental tool in calculus that allows us to compute the derivative of a composite function. In this section, we will derive the chain rule for functions of several variables and apply it to various cases.
    🔗
    Recall that if \(y=f(x)\) and \(x=g(t)\text{,}\) where \(f\) and \(g\) are differentiable functions, then \(y\) is indirectly a differentiable function of \(t\text{,}\) and the single-variable chain rule applies:
    \begin{equation*} \frac{dy}{dt} = \frac{dy}{dx}\frac{dx}{dt} \end{equation*}
    🔗

    Theorem 3.48. The Chain Rule: Case I.

    Suppose that \(z=f(x,y)\) is a differentiable function of \(x\) and \(y\text{,}\) where \(x=g(t)\) and \(y=h(t)\) are both differentiable functions of \(t\text{.}\) Then \(z\) is a differentiable function of \(t\) and satisfies the formula:
    \begin{equation*} \frac{dz}{dt} = \frac{\partial f}{\partial x}\frac{dx}{dt} + \frac{\partial f}{\partial y}\frac{dy}{dt} \end{equation*}
    🔗
    🔗

    Example 3.50.

    If \(z=x^2y+3xy^4\text{,}\) where \(x=\sin(2t)\) and \(y=\cos(t)\text{,}\) find \(\frac{dz}{dt}\) when \(t=0\text{.}\)
    🔗
    Solution.
    We first compute all the necessary components for the Case I Chain Rule formula. The first-order partial derivatives of \(z\) with respect to \(x\) and \(y\) are:
    \begin{equation*} \frac{\partial z}{\partial x} = 2xy + 3y^4 \quad \text{and} \quad \frac{\partial z}{\partial y} = x^2 + 12xy^3 \end{equation*}
    🔗
    The single-variable derivatives of \(x\) and \(y\) with respect to the parameter \(t\) are:
    \begin{equation*} \frac{dx}{dt} = \frac{d}{dt}[\sin(2t)] = 2\cos(2t) \quad \text{and} \quad \frac{dy}{dt} = \frac{d}{dt}[\cos(t)] = -\sin(t) \end{equation*}
    🔗
    Now, we substitute these components into the chain rule formula:
    \begin{align*} \frac{dz}{dt} \amp = \frac{\partial z}{\partial x}\frac{dx}{dt} + \frac{\partial z}{\partial y}\frac{dy}{dt}\\ \amp = (2xy + 3y^4)(2\cos(2t)) + (x^2 + 12xy^3)(-\sin(t)) \end{align*}
    🔗
    To find the value of \(\frac{dz}{dt}\) at the specific parameter point \(t = 0\text{,}\) we first evaluate the corresponding coordinate locations for \(x\) and \(y\text{:}\)
    \begin{equation*} x = \sin(2(0)) = 0 \quad \text{and} \quad y = \cos(0) = 1 \end{equation*}
    🔗
    Finally, substituting \(t=0\text{,}\) \(x=0\text{,}\) and \(y=1\) into our derivative expression yields:
    \begin{align*} \frac{dz}{dt}\Big|_{t=0} \amp = \left(2(0)(1) + 3(1)^4\right)\left(2\cos(0)\right) + \left(0^2 + 12(0)(1)^3\right)\left(-\sin(0)\right)\\ \amp = (0 + 3)(2(1)) + (0 + 0)(0)\\ \amp = (3)(2) + 0\\ \amp = 6 \end{align*}
    🔗
    🔗
    🔗

    Theorem 3.52. The Chain Rule: Case II.

    Suppose that \(z=f(x,y)\) is a differentiable function of \(x\) and \(y\text{,}\) where \(x=g(s,t)\) and \(y=h(s,t)\) are both differentiable functions of \(s\) and \(t\text{.}\) Then \(z\) is a differentiable function of \(s\) and \(t\text{,}\) satisfying the partial derivative formulas:
    \begin{equation*} \frac{\partial z}{\partial s} = \frac{\partial z}{\partial x}\frac{\partial x}{\partial s} + \frac{\partial z}{\partial y}\frac{\partial y}{\partial s} \quad \text{and} \quad \frac{\partial z}{\partial t} = \frac{\partial z}{\partial x}\frac{\partial x}{\partial t} + \frac{\partial z}{\partial y}\frac{\partial y}{\partial t} \end{equation*}
    🔗
    🔗

    Example 3.54.

    If \(z=e^x\sin(y)\text{,}\) where \(x=st^2\) and \(y=s^2t\text{,}\) find \(\frac{\partial z}{\partial s}\) and \(\frac{\partial z}{\partial t}\text{.}\)
    🔗
    Solution.
    We first calculate the required first-order partial derivatives of the dependent surface function \(z\text{:}\)
    \begin{equation*} \frac{\partial z}{\partial x} = e^x\sin(y) \quad \text{and} \quad \frac{\partial z}{\partial y} = e^x\cos(y) \end{equation*}
    🔗
    Next, we calculate the partial derivatives for the intermediate variable functions \(x\) and \(y\) with respect to both independent parameters \(s\) and \(t\text{:}\)
    \begin{equation*} \frac{\partial x}{\partial s} = t^2, \quad \frac{\partial x}{\partial t} = 2st \end{equation*}
    \begin{equation*} \frac{\partial y}{\partial s} = 2st, \quad \frac{\partial y}{\partial t} = s^2 \end{equation*}
    🔗
    Applying the Case II Chain Rule formula for the parameter \(s\text{,}\) we substitute our partial derivatives:
    \begin{align*} \frac{\partial z}{\partial s} \amp = \frac{\partial z}{\partial x}\frac{\partial x}{\partial s} + \frac{\partial z}{\partial y}\frac{\partial y}{\partial s}\\ \amp = \left(e^x\sin(y)\right)(t^2) + \left(e^x\cos(y)\right)(2st)\\ \amp = t^2 e^{st^2}\sin(s^2t) + 2st e^{st^2}\cos(s^2t) \end{align*}
    🔗
    Applying the Case II Chain Rule formula for the parameter \(t\text{,}\) we substitute our partial derivatives:
    \begin{align*} \frac{\partial z}{\partial t} \amp = \frac{\partial z}{\partial x}\frac{\partial x}{\partial t} + \frac{\partial z}{\partial y}\frac{\partial y}{\partial t}\\ \amp = \left(e^x\sin(y)\right)(2st) + \left(e^x\cos(y)\right)(s^2)\\ \amp = 2st e^{st^2}\sin(s^2t) + s^2 e^{st^2}\cos(s^2t) \end{align*}
    🔗
    🔗
    🔗

    Example 3.56.

    Use the chain rule to find \(\frac{\partial w}{\partial z}\text{,}\) if
    \begin{equation*} w=r^2+sv+t^3, \quad r=x^2+y^2+z^2, \quad s=xyz, \quad v=xe^y, \quad t=yz^2 \end{equation*}
    🔗
    Solution.
    The dependent variable \(w\) is directly a function of four intermediate variables: \(r\text{,}\) \(s\text{,}\) \(v\text{,}\) and \(t\text{.}\) By checking our dependency tree, we see that \(r\text{,}\) \(s\text{,}\) and \(t\) depend on the independent variable \(z\text{,}\) whereas \(v\) does not contain \(z\) (meaning \(\frac{\partial v}{\partial z} = 0\)). Therefore, the chain rule extension for this system is:
    \begin{equation*} \frac{\partial w}{\partial z} = \frac{\partial w}{\partial r}\frac{\partial r}{\partial z} + \frac{\partial w}{\partial s}\frac{\partial s}{\partial z} + \frac{\partial w}{\partial v}\frac{\partial v}{\partial z} + \frac{\partial w}{\partial t}\frac{\partial t}{\partial z} \end{equation*}
    🔗
    We find each required component derivative independently:
    \begin{equation*} \frac{\partial w}{\partial r} = 2r = 2(x^2+y^2+z^2) \quad \text{and} \quad \frac{\partial r}{\partial z} = 2z \end{equation*}
    \begin{equation*} \frac{\partial w}{\partial s} = v = xe^y \quad \text{and} \quad \frac{\partial s}{\partial z} = xy \end{equation*}
    \begin{equation*} \frac{\partial w}{\partial t} = 3t^2 = 3(yz^2)^2 = 3y^2z^4 \quad \text{and} \quad \frac{\partial t}{\partial z} = 2yz \end{equation*}
    🔗
    Substituting these components back into our general chain rule formula yields:
    \begin{align*} \frac{\partial w}{\partial z} \amp = \left[2(x^2+y^2+z^2)\right](2z) + \left[xe^y\right](xy) + (xyz)(0) + \left[3y^2z^4\right](2yz)\\ \amp = 4z(x^2+y^2+z^2) + x^2ye^y + 6y^3z^5 \end{align*}
    🔗
    🔗
    🔗

    Example 3.58.

    If \(g(s,t)=f(s^2-t^2,t^2-s^2)\) and \(f\) is differentiable, show that \(g\) satisfies the partial differential equation:
    \begin{equation*} t\frac{\partial g}{\partial s}+s\frac{\partial g}{\partial t}=0 \end{equation*}
    🔗
    Solution.
    Let us define the intermediate variables for the inner functions as \(x = s^2 - t^2\) and \(y = t^2 - s^2\text{,}\) so that \(g(s,t) = f(x,y)\text{.}\) By applying the Case II Chain Rule, we compute the general first-order partial derivatives of \(g\) with respect to the parameters \(s\) and \(t\text{:}\)
    🔗
    For the parameter \(s\text{:}\)
    \begin{align*} \frac{\partial g}{\partial s} \amp = \frac{\partial f}{\partial x}\frac{\partial x}{\partial s} + \frac{\partial f}{\partial y}\frac{\partial y}{\partial s}\\ \amp = \frac{\partial f}{\partial x}(2s) + \frac{\partial f}{\partial y}(-2s)\\ \amp = 2s\frac{\partial f}{\partial x} - 2s\frac{\partial f}{\partial y} \end{align*}
    🔗
    For the parameter \(t\text{:}\)
    \begin{align*} \frac{\partial g}{\partial t} \amp = \frac{\partial f}{\partial x}\frac{\partial x}{\partial t} + \frac{\partial f}{\partial y}\frac{\partial y}{\partial t}\\ \amp = \frac{\partial f}{\partial x}(-2t) + \frac{\partial f}{\partial y}(2t)\\ \amp = -2t\frac{\partial f}{\partial x} + 2t\frac{\partial f}{\partial y} \end{align*}
    🔗
    Now, we substitute these expressions directly into the left-hand side of the target verification equation:
    \begin{align*} t\frac{\partial g}{\partial s} + s\frac{\partial g}{\partial t} \amp = t\left(2s\frac{\partial f}{\partial x} - 2s\frac{\partial f}{\partial y}\right) + s\left(-2t\frac{\partial f}{\partial x} + 2t\frac{\partial f}{\partial y}\right)\\ \amp = 2st\frac{\partial f}{\partial x} - 2st\frac{\partial f}{\partial y} - 2st\frac{\partial f}{\partial x} + 2st\frac{\partial f}{\partial y}\\ \amp = 0 \end{align*}
    🔗
    Since the expression simplifies identically to \(0\text{,}\) we have successfully shown that the function \(g\) satisfies the given equation.
    🔗
    🔗
    🔗

    Example 3.59.

    If \(z=f(x,y)\) has continuous second-order partial derivatives, \(x=r^2+s^2\text{,}\) and \(y=2rs\text{,}\) find \(\frac{\partial z}{\partial r}\) and \(\frac{\partial^2 z}{\partial r^2}\text{.}\)
    🔗
    Solution.
    To find the first partial derivative \(\frac{\partial z}{\partial r}\text{,}\) we apply the Case II Multivariate Chain Rule:
    \begin{align*} \frac{\partial z}{\partial r} \amp = \frac{\partial z}{\partial x}\frac{\partial x}{\partial r} + \frac{\partial z}{\partial y}\frac{\partial y}{\partial r}\\ \amp = \frac{\partial z}{\partial x}(2r) + \frac{\partial z}{\partial y}(2s)\\ \amp = 2r\frac{\partial z}{\partial x} + 2s\frac{\partial z}{\partial y} \end{align*}
    🔗
    To find the second partial derivative \(\frac{\partial^2 z}{\partial r^2}\text{,}\) we differentiate \(\frac{\partial z}{\partial r}\) with respect to \(r\text{.}\) This requires applying both the single-variable product rule and a nested multivariate chain rule since the internal terms \(\frac{\partial z}{\partial x}\) and \(\frac{\partial z}{\partial y}\) still implicitly depend on \(x\) and \(y\) (which depend on \(r\)):
    \begin{align*} \frac{\partial^2 z}{\partial r^2} \amp = \frac{\partial}{\partial r}\left( 2r\frac{\partial z}{\partial x} + 2s\frac{\partial z}{\partial y} \right)\\ \amp = \left[ 2\frac{\partial z}{\partial x} + 2r\frac{\partial}{\partial r}\left(\frac{\partial z}{\partial x}\right) \right] + 2s\frac{\partial}{\partial r}\left(\frac{\partial z}{\partial y}\right) \end{align*}
    🔗
    We expand the nested partial derivatives \(\frac{\partial}{\partial r}\left(\frac{\partial z}{\partial x}\right)\) and \(\frac{\partial}{\partial r}\left(\frac{\partial z}{\partial y}\right)\) using the chain rule:
    \begin{align*} \frac{\partial}{\partial r}\left(\frac{\partial z}{\partial x}\right) \amp = \frac{\partial^2 z}{\partial x^2}\frac{\partial x}{\partial r} + \frac{\partial^2 z}{\partial y\,\partial x}\frac{\partial y}{\partial r} = \frac{\partial^2 z}{\partial x^2}(2r) + \frac{\partial^2 z}{\partial y\,\partial x}(2s)\\ \frac{\partial}{\partial r}\left(\frac{\partial z}{\partial y}\right) \amp = \frac{\partial^2 z}{\partial x\,\partial y}\frac{\partial x}{\partial r} + \frac{\partial^2 z}{\partial y^2}\frac{\partial y}{\partial r} = \frac{\partial^2 z}{\partial x\,\partial y}(2r) + \frac{\partial^2 z}{\partial y^2}(2s) \end{align*}
    🔗
    Substituting these expansions back into our main differential expression yields:
    \begin{align*} \frac{\partial^2 z}{\partial r^2} \amp = 2\frac{\partial z}{\partial x} + 2r\left[ 2r\frac{\partial^2 z}{\partial x^2} + 2s\frac{\partial^2 z}{\partial y\,\partial x} \right] + 2s\left[ 2r\frac{\partial^2 z}{\partial x\,\partial y} + 2s\frac{\partial^2 z}{\partial y^2} \right]\\ \amp = 2\frac{\partial z}{\partial x} + 4r^2\frac{\partial^2 z}{\partial x^2} + 4rs\frac{\partial^2 z}{\partial y\,\partial x} + 4rs\frac{\partial^2 z}{\partial x\,\partial y} + 4s^2\frac{\partial^2 z}{\partial y^2} \end{align*}
    Since the second-order partial derivatives are continuous, Clairaut’s Theorem guarantees that the mixed partials are equal (\(\frac{\partial^2 z}{\partial y\,\partial x} = \frac{\partial^2 z}{\partial x\,\partial y}\)). Combining the like terms gives the final result:
    \begin{equation*} \frac{\partial^2 z}{\partial r^2} = 2\frac{\partial z}{\partial x} + 4r^2\frac{\partial^2 z}{\partial x^2} + 8rs\frac{\partial^2 z}{\partial x\,\partial y} + 4s^2\frac{\partial^2 z}{\partial y^2} \end{equation*}
    🔗
    🔗
    🔗
    Suppose an equation of the form \(F(x,y)=0\) defines \(y\) implicitly as a differentiable function of \(x\text{,}\) such that \(y=f(x)\text{,}\) where \(F(x,f(x))=0\) for all \(x\) in the domain of \(f\text{.}\) If \(F\) is differentiable, we can apply the multivariate chain rule to differentiate both sides of the equation with respect to \(x\text{,}\) treating \(x\) as both an independent parameter and a base variable path:
    \begin{equation*} \frac{\partial F}{\partial x}\frac{dx}{dx} + \frac{\partial F}{\partial y}\frac{dy}{dx} = 0 \implies \frac{\partial F}{\partial x} + \frac{\partial F}{\partial y}\frac{dy}{dx} = 0 \end{equation*}
    If \(\frac{\partial F}{\partial y} \neq 0\text{,}\) we can isolate and solve for the single-variable derivative \(\frac{dy}{dx}\text{.}\)
    🔗

    Theorem 3.60. Implicit Differentiation: Case I.

    If an equation \(F(x,y)=0\) defines \(y\) implicitly as a differentiable function of \(x\text{,}\) then:
    \begin{equation*} \frac{dy}{dx} = -\frac{F_x(x,y)}{F_y(x,y)} = -\frac{\frac{\partial F}{\partial x}}{\frac{\partial F}{\partial y}} \end{equation*}
    🔗
    🔗

    Example 3.61.

    Find \(y'\) if \(x^3+y^3=6xy\text{.}\)
    🔗
    Solution.
    First, we move all terms to one side of the equation to define our multi-variable identity function \(F(x,y) = 0\text{:}\)
    \begin{equation*} F(x,y) = x^3 + y^3 - 6xy = 0 \end{equation*}
    🔗
    Next, we calculate the first-order partial derivatives \(F_x\) and \(F_y\text{:}\)
    \begin{equation*} F_x(x,y) = \frac{\partial}{\partial x}(x^3 + y^3 - 6xy) = 3x^2 - 6y \end{equation*}
    \begin{equation*} F_y(x,y) = \frac{\partial}{\partial y}(x^3 + y^3 - 6xy) = 3y^2 - 6x \end{equation*}
    🔗
    Applying the Theorem for Implicit Differentiation, we substitute these partials into the formula:
    \begin{align*} y' = \frac{dy}{dx} \amp = -\frac{F_x(x,y)}{F_y(x,y)}\\ \amp = -\frac{3x^2 - 6y}{3y^2 - 6x}\\ \amp = -\frac{3(x^2 - 2y)}{3(y^2 - 2x)}\\ \amp = -\frac{x^2 - 2y}{y^2 - 2x} \end{align*}
    🔗
    🔗
    🔗

    Theorem 3.62. Implicit Differentiation: Case II.

    If an equation \(F(x,y,z)=0\) defines \(z\) implicitly as a differentiable function of two variables \(x\) and \(y\text{,}\) such that \(z=f(x,y)\) for every \((x,y)\) in the domain of \(f\text{,}\) then:
    \begin{equation*} \frac{\partial z}{\partial x} = -\frac{F_x(x,y,z)}{F_z(x,y,z)} \quad \text{and} \quad \frac{\partial z}{\partial y} = -\frac{F_y(x,y,z)}{F_z(x,y,z)} \end{equation*}
    🔗
    🔗

    Example 3.63.

    Find \(\frac{\partial z}{\partial x}\) and \(\frac{\partial z}{\partial y}\) if \(z\) is defined implicitly as a function of \(x\) and \(y\) by the equation:
    \begin{equation*} x^3+y^3+z^3+6xyz=1 \end{equation*}
    🔗
    Solution.
    First, we move all terms to one side of the equation to define our three-variable identity function \(F(x,y,z) = 0\text{:}\)
    \begin{equation*} F(x,y,z) = x^3 + y^3 + z^3 + 6xyz - 1 = 0 \end{equation*}
    🔗
    Next, we find the three required first-order partial derivatives of \(F\text{:}\)
    \begin{equation*} F_x(x,y,z) = \frac{\partial}{\partial x}(x^3 + y^3 + z^3 + 6xyz - 1) = 3x^2 + 6yz \end{equation*}
    \begin{equation*} F_y(x,y,z) = \frac{\partial}{\partial y}(x^3 + y^3 + z^3 + 6xyz - 1) = 3y^2 + 6xz \end{equation*}
    \begin{equation*} F_z(x,y,z) = \frac{\partial}{\partial z}(x^3 + y^3 + z^3 + 6xyz - 1) = 3z^2 + 6xy \end{equation*}
    🔗
    To find \(\frac{\partial z}{\partial x}\text{,}\) we substitute the partial derivatives \(F_x\) and \(F_z\) into the implicit differentiation formula and simplify by factoring out a common factor of 3:
    \begin{align*} \frac{\partial z}{\partial x} \amp = -\frac{F_x(x,y,z)}{F_z(x,y,z)}\\ \amp = -\frac{3x^2 + 6yz}{3z^2 + 6xy}\\ \amp = -\frac{3(x^2 + 2yz)}{3(z^2 + 2xy)}\\ \amp = -\frac{x^2+2yz}{z^2+2xy} \end{align*}
    🔗
    Similarly, to find \(\frac{\partial z}{\partial y}\text{,}\) we substitute the partial derivatives \(F_y\) and \(F_z\) into the corresponding formula and simplify:
    \begin{align*} \frac{\partial z}{\partial y} \amp = -\frac{F_y(x,y,z)}{F_z(x,y,z)}\\ \amp = -\frac{3y^2 + 6xz}{3z^2 + 6xy}\\ \amp = -\frac{3(y^2 + 2xz)}{3(z^2 + 2xy)}\\ \amp = -\frac{y^2+2xz}{z^2+2xy} \end{align*}
    🔗
    🔗
    🔗

    Checkpoint 3.64.

    Use the chain rule to find \(\frac{\partial w}{\partial p}\) and \(\frac{\partial w}{\partial q}\) if:
    \begin{equation*} w=r^3+s^2 \quad \text{with} \quad r=pq^2, \ \ s=p^2\sin(q) \end{equation*}
    🔗
    Solution.
    The dependent variable \(w\) is a function of intermediate variables \(r\) and \(s\text{,}\) which both depend on the independent parameters \(p\) and \(q\text{.}\) We apply the Case II Multivariate Chain Rule.
    🔗
    First, we find the baseline partial derivatives of \(w\text{:}\)
    \begin{equation*} \frac{\partial w}{\partial r} = 3r^2 = 3(pq^2)^2 = 3p^2q^4 \quad \text{and} \quad \frac{\partial w}{\partial s} = 2s = 2(p^2\sin(q)) = 2p^2\sin(q) \end{equation*}
    🔗
    Next, we calculate the partial derivatives of \(r\) and \(s\) with respect to the parameter \(p\text{:}\)
    \begin{equation*} \frac{\partial r}{\partial p} = q^2 \quad \text{and} \quad \frac{\partial s}{\partial p} = 2p\sin(q) \end{equation*}
    Substituting these into the chain rule formula for \(\frac{\partial w}{\partial p}\) yields:
    \begin{align*} \frac{\partial w}{\partial p} \amp = \frac{\partial w}{\partial r}\frac{\partial r}{\partial p} + \frac{\partial w}{\partial s}\frac{\partial s}{\partial p}\\ \amp = (3p^2q^4)(q^2) + (2p^2\sin(q))(2p\sin(q))\\ \amp = 3p^2q^6 + 4p^3\sin^2(q) \end{align*}
    🔗
    Now, we calculate the partial derivatives of \(r\) and \(s\) with respect to the parameter \(q\text{:}\)
    \begin{equation*} \frac{\partial r}{\partial q} = 2pq \quad \text{and} \quad \frac{\partial s}{\partial q} = p^2\cos(q) \end{equation*}
    Substituting these into the chain rule formula for \(\frac{\partial w}{\partial q}\) yields:
    \begin{align*} \frac{\partial w}{\partial q} \amp = \frac{\partial w}{\partial r}\frac{\partial r}{\partial q} + \frac{\partial w}{\partial s}\frac{\partial s}{\partial q}\\ \amp = (3p^2q^4)(2pq) + (2p^2\sin(q))(p^2\cos(q))\\ \amp = 6p^3q^5 + 2p^4\sin(q)\cos(q) \end{align*}
    🔗
    🔗
    🔗

    Checkpoint 3.65.

    Use the chain rule to find \(\frac{\partial w}{\partial t}\) if:
    \begin{equation*} w=x^2 \quad \text{with} \quad x=3t^2+1, \ \ y=2t-4, \ \ z=t^3 \end{equation*}
    🔗
    Solution.
    As defined by the given equations, the function \(w\) explicitly depends only on the variable \(x\text{.}\) Since \(x\) is a function of the single parameter \(t\text{,}\) this simplifies directly to the standard single-variable chain rule:
    \begin{equation*} \frac{dw}{dt} = \frac{dw}{dx}\frac{dx}{dt} \end{equation*}
    🔗
    We compute the component derivatives independently:
    \begin{equation*} \frac{dw}{dx} = 2x = 2(3t^2+1) = 6t^2+2 \quad \text{and} \quad \frac{dx}{dt} = 6t \end{equation*}
    🔗
    Multiplying these expressions together yields the rate of change with respect to \(t\text{:}\)
    \begin{align*} \frac{dw}{dt} \amp = (6t^2+2)(6t)\\ \amp = 36t^3 + 12t \end{align*}
    🔗
    🔗
    🔗

    Checkpoint 3.66.

    Find \(\frac{dy}{dx}\) if \(y=f(x)\) is determined implicitly by the equation:
    \begin{equation*} y^4+3y-4x^3-5x-1=0 \end{equation*}
    🔗
    Solution.
    We define our multi-variable identity function \(F(x,y) = 0\) as:
    \begin{equation*} F(x,y) = y^4 + 3y - 4x^3 - 5x - 1 = 0 \end{equation*}
    🔗
    Next, we evaluate the first-order partial derivatives \(F_x\) and \(F_y\text{:}\)
    \begin{equation*} F_x(x,y) = \frac{\partial}{\partial x}(y^4 + 3y - 4x^3 - 5x - 1) = -12x^2 - 5 \end{equation*}
    \begin{equation*} F_y(x,y) = \frac{\partial}{\partial y}(y^4 + 3y - 4x^3 - 5x - 1) = 4y^3 + 3 \end{equation*}
    🔗
    Applying the Implicit Differentiation Theorem Case I, we find:
    \begin{align*} \frac{dy}{dx} \amp = -\frac{F_x(x,y)}{F_y(x,y)}\\ \amp = -\frac{-12x^2 - 5}{4y^3 + 3}\\ \amp = \frac{12x^2 + 5}{4y^3 + 3} \end{align*}

    Checkpoint 3.67.

    Find \(\frac{\partial z}{\partial x}\) and \(\frac{\partial z}{\partial y}\) if \(z\) is defined implicitly by the equation:
    \begin{equation*} x^2z^2+xy^2-z^3+4yz-5=0 \end{equation*}
    🔗
    Solution.
    We define our three-variable identity function \(F(x,y,z) = 0\) as:
    \begin{equation*} F(x,y,z) = x^2z^2 + xy^2 - z^3 + 4yz - 5 = 0 \end{equation*}
    🔗
    We calculate the three first-order partial derivatives of \(F\text{:}\)
    \begin{equation*} F_x(x,y,z) = 2xz^2 + y^2 \end{equation*}
    \begin{equation*} F_y(x,y,z) = 2xy + 4z \end{equation*}
    \begin{equation*} F_z(x,y,z) = 2x^2z - 3z^2 + 4y \end{equation*}
    🔗
    Applying the Implicit Differentiation Theorem Case II for the partial derivative with respect to \(x\text{:}\)
    \begin{equation*} \frac{\partial z}{\partial x} = -\frac{F_x(x,y,z)}{F_z(x,y,z)} = -\frac{2xz^2 + y^2}{2x^2z - 3z^2 + 4y} \end{equation*}
    🔗
    Similarly, applying the formula for the partial derivative with respect to \(y\text{:}\)
    \begin{equation*} \frac{\partial z}{\partial y} = -\frac{F_y(x,y,z)}{F_z(x,y,z)} = -\frac{2xy + 4z}{2x^2z - 3z^2 + 4y} \end{equation*}
