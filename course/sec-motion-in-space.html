    Section 2.5 Motion in Space: Velocity and Acceleration

    In this section, we will study the motion of objects in three-dimensional space. We will define the concepts of velocity and acceleration for vector-valued functions, and we will explore how these concepts relate to the motion of objects along curves in space.
    🔗
    We will also discuss the concepts of speed and acceleration, and how they can be used to analyze the motion of objects in space. Finally, we will look at some examples of motion in space, including the motion of projectiles and the motion of objects under the influence of gravity.
    🔗
    Suppose a particle moves through space so that its position vector at time \(t\) is \(\mathbf{r}(t)\text{.}\) For small values of \(h\text{,}\) the vector:
    \begin{equation*} \frac{\mathbf{r}(t+h)-\mathbf{r}(t)}{h} \longrightarrow \text{the direction of the particle moving along the curve } \mathbf{r}(t) \end{equation*}
    🔗

    Definition 2.55. Velocity Vector.

    The vector \(\frac{\mathbf{r}(t+h)-\mathbf{r}(t)}{h}\) gives the average velocity over a time interval of length \(h\text{,}\) and its limit is the velocity vector \(\mathbf{v}(t)\) at time \(t\text{:}\)
    \begin{equation*} \mathbf{v}(t)=\lim_{h \to 0} \frac{\mathbf{r}(t+h)-\mathbf{r}(t)}{h}= \mathbf{r}'(t) \end{equation*}
    🔗
    🔗

    Definition 2.56. Speed.

    The speed of the particle at time \(t\) is the magnitude of the velocity vector, \(|\mathbf{v}(t)|\text{,}\) defined as:
    \begin{equation*} |\mathbf{v}(t)|=|\mathbf{r}'(t)|=\frac{ds}{dt}= \text{rate of change of distance with respect to time.} \end{equation*}
    🔗
    🔗

    Definition 2.57. Acceleration.

    In multi-dimensional motion, the acceleration \(\mathbf{a}(t)\) of the particle is defined as the derivative of the velocity vector:
    \begin{equation*} \mathbf{a}(t)=\mathbf{v}'(t)=\mathbf{r}''(t) \end{equation*}
    🔗
    🔗

    Example 2.58.

    The position vector of an object moving in a plane is given by \(\mathbf{r}(t)=(t^2+t)\mathbf{i}+t^3\mathbf{j}\) for \(0 \le t \le 2\text{.}\) Find its velocity, speed, and acceleration when \(t=1\text{.}\)
    🔗
    Solution.
    First, we differentiate the position vector function \(\mathbf{r}(t)\) with respect to \(t\) to find the velocity vector function \(\mathbf{v}(t)\text{:}\)
    \begin{equation*} \mathbf{v}(t) = \mathbf{r}'(t) = (2t+1)\mathbf{i} + 3t^2\mathbf{j} \end{equation*}
    Evaluating the velocity vector at the specific parameter point \(t=1\) yields:
    \begin{equation*} \mathbf{v}(1) = (2(1)+1)\mathbf{i} + 3(1)^2\mathbf{j} = 3\mathbf{i} + 3\mathbf{j} = \langle 3, 3 \rangle \end{equation*}
    🔗
    Next, the speed of the object is the scalar magnitude of its velocity vector. Evaluating at \(t=1\) gives:
    \begin{equation*} |\mathbf{v}(1)| = \sqrt{3^2 + 3^2} = \sqrt{18} = 3\sqrt{2} \end{equation*}
    🔗
    Finally, we differentiate the velocity vector function to find the acceleration vector function \(\mathbf{a}(t)\text{:}\)
    \begin{equation*} \mathbf{a}(t) = \mathbf{v}'(t) = 2\mathbf{i} + 6t\mathbf{j} \end{equation*}
    Evaluating the acceleration vector at \(t=1\) yields:
    \begin{equation*} \mathbf{a}(1) = 2\mathbf{i} + 6(1)\mathbf{j} = 2\mathbf{i} + 6\mathbf{j} = \langle 2, 6 \rangle \end{equation*}
    🔗
    🔗
    🔗

    Example 2.59.

    A moving particle starts at an initial position \(\mathbf{r}(0)=\langle 1,0,0 \rangle\) with initial velocity \(\mathbf{v}(0)=\mathbf{i}-\mathbf{j}+\mathbf{k}\text{.}\) Its acceleration is \(\mathbf{a}(t)=4t\mathbf{i}+6t\mathbf{j}+\mathbf{k}\text{.}\) Find its velocity and position at time \(t\text{.}\)
    🔗
    Solution.
    We find the velocity vector function \(\mathbf{v}(t)\) by integrating the given acceleration vector function:
    \begin{equation*} \mathbf{v}(t) = \int \mathbf{a}(t)\, dt = \left( \int 4t\, dt \right)\mathbf{i} + \left( \int 6t\, dt \right)\mathbf{j} + \left( \int 1\, dt \right)\mathbf{k} = 2t^2\mathbf{i} + 3t^2\mathbf{j} + t\mathbf{k} + \mathbf{C}_1 \end{equation*}
    Using our initial velocity vector condition \(\mathbf{v}(0) = \mathbf{i}-\mathbf{j}+\mathbf{k} = \langle 1, -1, 1 \rangle\) at \(t=0\) reveals \(\mathbf{C}_1 = \mathbf{i}-\mathbf{j}+\mathbf{k}\text{.}\) Combining these gives the final velocity function:
    \begin{equation*} \mathbf{v}(t) = (2t^2+1)\mathbf{i} + (3t^2-1)\mathbf{j} + (t+1)\mathbf{k} = \langle 2t^2+1, 3t^2-1, t+1 \rangle \end{equation*}
    🔗
    Next, we find the position vector function \(\mathbf{r}(t)\) by integrating the newly solved velocity vector function:
    \begin{equation*} \mathbf{r}(t) = \int \mathbf{v}(t)\, dt = \left( \frac{2}{3}t^3 + t \right)\mathbf{i} + \left( t^3 - t \right)\mathbf{j} + \left( \frac{1}{2}t^2 + t \right)\mathbf{k} + \mathbf{C}_2 \end{equation*}
    Using our initial position vector condition \(\mathbf{r}(0) = \langle 1,0,0 \rangle = 1\mathbf{i} + 0\mathbf{j} + 0\mathbf{k}\) at \(t=0\) reveals \(\mathbf{C}_2 = \mathbf{i}\text{.}\) Combining these gives the final position vector function:
    \begin{equation*} \mathbf{r}(t) = \left( \frac{2}{3}t^3 + t + 1 \right)\mathbf{i} + \left( t^3 - t \right)\mathbf{j} + \left( \frac{1}{2}t^2 + t \right)\mathbf{k} = \left\langle \frac{2}{3}t^3 + t + 1, \, t^3 - t, \, \frac{1}{2}t^2 + t \right\rangle \end{equation*}
    🔗
    🔗
    🔗

    Example 2.60.

    An object with mass \(m\) that moves in a circular path with constant angular speed \(\omega\) has position vector \(\mathbf{r}(t)=a\cos \omega t\mathbf{i}+a\sin \omega t\mathbf{j}\text{.}\) Find the force acting on the object and show that it is directed toward the origin.
    🔗
    Solution.
    To locate the total force acting on the particle, we must first find its acceleration profile vector. Differentiating the position vector function with respect to time using the chain rule yields velocity:
    \begin{equation*} \mathbf{v}(t) = \mathbf{r}'(t) = -a\omega\sin\omega t\mathbf{i} + a\omega\cos\omega t\mathbf{j} \end{equation*}
    Differentiating a second time yields the acceleration vector function:
    \begin{equation*} \mathbf{a}(t) = \mathbf{v}'(t) = -a\omega^2\cos\omega t\mathbf{i} - a\omega^2\sin\omega t\mathbf{j} \end{equation*}
    Factoring out the shared constant parameter scalar \(-\omega^2\) reveals an explicit relationship to our primary position vector:
    \begin{equation*} \mathbf{a}(t) = -\omega^2\left( a\cos\omega t\mathbf{i} + a\sin\omega t\mathbf{j} \right) = -\omega^2\mathbf{r}(t) \end{equation*}
    🔗
    By applying Newton’s Second Law of Motion (\(\mathbf{F} = m\mathbf{a}\)), we multiply this acceleration expression by the mass scalar:
    \begin{equation*} \mathbf{F}(t) = m\mathbf{a}(t) = -m\omega^2\mathbf{r}(t) \end{equation*}
    Because \(m\omega^2 \gt 0\) is a strictly positive scalar quantity, the negative sign indicates that the net force vector \(\mathbf{F}(t)\) points in the exact opposite direction of the outward displacement vector \(\mathbf{r}(t)\text{.}\) Therefore, the centripetal force is directed directly inward toward the origin.
    🔗
    🔗
    🔗

    Remark 2.62. Newton’s Second Law of Motion.

    If the force that acts on a particle is known, then its acceleration can be found from Newton’s Second Law of Motion. The vector version of this law states that if, at any time \(t\text{,}\) a force \(\mathbf{F}(t)\) acts on an object of mass \(m\) producing an acceleration \(\mathbf{a}(t)\text{,}\) then:
    \begin{equation*} \mathbf{F}(t)=m\mathbf{a}(t) \end{equation*}
    🔗
    🔗

    Example 2.63.

    Show that if a particle \(P\) moves around a circle of radius \(a\) at a constant speed \(v\text{,}\) then the acceleration vector has a constant magnitude \(v^2/a\) and is directed from \(P\) toward the center of the circle.
    🔗
    Solution.
    We can parameterize uniform circular motion of radius \(a\) with constant angular speed \(\omega\) as:
    \begin{equation*} \mathbf{r}(t) = \langle a\cos(\omega t), a\sin(\omega t) \rangle \end{equation*}
    Differentiating once yields the velocity vector function:
    \begin{equation*} \mathbf{v}(t) = \mathbf{r}'(t) = \langle -a\omega\sin(\omega t), a\omega\cos(\omega t) \rangle \end{equation*}
    The constant speed \(v\) is the magnitude of this velocity vector:
    \begin{equation*} v = |\mathbf{v}(t)| = \sqrt{(-a\omega\sin(\omega t))^2 + (a\omega\cos(\omega t))^2} = \sqrt{a^2\omega^2(\sin^2(\omega t) + \cos^2(\omega t))} = a\omega \end{equation*}
    This gives us the relationship \(\omega = \frac{v}{a}\text{.}\)
    🔗
    Next, differentiate the velocity vector function to find the acceleration vector function \(\mathbf{a}(t)\text{:}\)
    \begin{equation*} \mathbf{a}(t) = \mathbf{v}'(t) = \langle -a\omega^2\cos(\omega t), -a\omega^2\sin(\omega t) \rangle = -\omega^2\mathbf{r}(t) \end{equation*}
    Substituting \(\omega = \frac{v}{a}\) into our acceleration expression yields:
    \begin{equation*} \mathbf{a}(t) = -\left(\frac{v}{a}\right)^2\mathbf{r}(t) = -\frac{v^2}{a^2}\mathbf{r}(t) \end{equation*}
    🔗
    Now we compute the magnitude of the acceleration vector:
    \begin{equation*} |\mathbf{a}(t)| = \left|-\frac{v^2}{a^2}\right| |\mathbf{r}(t)| = \frac{v^2}{a^2} \cdot a = \frac{v^2}{a} \end{equation*}
    Because \(\frac{v^2}{a^2} \gt 0\) is a strictly positive scalar, the negative sign in \(\mathbf{a}(t) = -\frac{v^2}{a^2}\mathbf{r}(t)\) dictates that the acceleration points in the exact opposite direction of the outward radial position vector \(\mathbf{r}(t)\text{.}\) Therefore, the acceleration vector has a constant magnitude of \(v^2/a\) and points directly inward from \(P\) toward the center of the circle.
    🔗
    🔗
    🔗

    Example 2.64.

    The position vector of a particle \(P\) is given by \(\mathbf{r}(t)=2t\mathbf{i}+3t^2\mathbf{j}+t^3\mathbf{k}\) for \(0 \le t \le 2\text{.}\) Find the velocity and acceleration of \(P\) at time \(t\text{.}\) Sketch the path of \(P\text{.}\)
    🔗
    Solution.
    We compute the velocity vector function \(\mathbf{v}(t)\) by taking the first derivative of the position components with respect to \(t\text{:}\)
    \begin{equation*} \mathbf{v}(t) = \mathbf{r}'(t) = 2\mathbf{i} + 6t\mathbf{j} + 3t^2\mathbf{k} = \langle 2, 6t, 3t^2 \rangle \end{equation*}
    We compute the acceleration vector function \(\mathbf{a}(t)\) by taking the derivative of our velocity profile:
    \begin{equation*} \mathbf{a}(t) = \mathbf{v}'(t) = 6\mathbf{j} + 6t\mathbf{k} = \langle 0, 6, 6t \rangle \end{equation*}
    🔗
    The trajectory begins at the origin \(\mathbf{r}(0) = \langle 0,0,0 \rangle\) and ends at \(\mathbf{r}(2) = \langle 4,12,8 \rangle\text{.}\) As time increases, the path arches upward across the first octant space.
    🔗
    🔗

    Example 2.66.

    A point \(P\) is rotating about the \(z\)-axis on a circle of radius \(a\) that lies in the plane \(z=h\text{.}\) The angular speed \(d\theta/dt\) is a constant \(\omega\text{.}\) The vector \(\boldsymbol{\omega}=\omega\mathbf{k}\) directed along the \(z\)-axis and having magnitude \(\omega\) is the angular velocity of \(P\text{.}\) Show that the velocity \(\mathbf{v}(t)\) of \(P\) is the cross product of \(\boldsymbol{\omega}\) and the position vector \(\mathbf{r}(t)\) for \(P\text{.}\)
    🔗
    Solution.
    Since the path is a circle of radius \(a\) sitting in the horizontal plane \(z=h\text{,}\) we can express the position vector of the moving point \(P\) explicitly as:
    \begin{equation*} \mathbf{r}(t) = \langle a\cos(\omega t), a\sin(\omega t), h \rangle \end{equation*}
    Differentiating this position profile with respect to \(t\) yields the standard velocity vector:
    \begin{equation*} \mathbf{v}(t) = \mathbf{r}'(t) = \langle -a\omega\sin(\omega t), a\omega\cos(\omega t), 0 \rangle \end{equation*}
    🔗
    Now, let us calculate the vector cross product of the angular velocity vector \(\boldsymbol{\omega} = \langle 0, 0, \omega \rangle\) and our position vector \(\mathbf{r}(t)\text{:}\)
    \begin{align*} \boldsymbol{\omega} \times \mathbf{r}(t) \amp = \begin{vmatrix} \mathbf{i} \amp \mathbf{j} \amp \mathbf{k} \\ 0 \amp 0 \amp \omega \\ a\cos(\omega t) \amp a\sin(\omega t) \amp h \end{vmatrix}\\ \amp = (0 \cdot h - \omega \cdot a\sin(\omega t))\mathbf{i} - (0 \cdot h - \omega \cdot a\cos(\omega t))\mathbf{j} + (0 \cdot a\sin(\omega t) - 0 \cdot a\cos(\omega t))\mathbf{k}\\ \amp = \langle -a\omega\sin(\omega t), a\omega\cos(\omega t), 0 \rangle \end{align*}
    Comparing this coordinate result directly to our derivative calculation proves that:
    \begin{equation*} \mathbf{v}(t) = \boldsymbol{\omega} \times \mathbf{r}(t) \end{equation*}
    🔗
    🔗
    🔗

    Example 2.67.

    A projectile is fired with an initial velocity \(\mathbf{v}_0\) from a point \(h_0\) feet above the ground. If the only force acting on the projectile is caused by gravitational acceleration \(\mathbf{g}\text{,}\) determine its position after \(t\) seconds.
    🔗
    Solution.
    Let the motion take place in the 2D \(xy\)-plane, where the positive \(y\)-axis points vertically upward from the ground level. The constant downward gravitational acceleration vector is expressed as:
    \begin{equation*} \mathbf{a}(t) = -g\mathbf{j} \end{equation*}
    where \(g \approx 32 \, \text{ft/sec}^2\text{.}\) Integrating this acceleration vector with respect to time \(t\) yields the velocity vector function:
    \begin{equation*} \mathbf{v}(t) = \int \mathbf{a}(t)\, dt = \int -g\mathbf{j}\, dt = -gt\mathbf{j} + \mathbf{C}_1 \end{equation*}
    Using the initial velocity condition at \(t=0\) gives \(\mathbf{v}(0) = \mathbf{C}_1 = \mathbf{v}_0\text{.}\) Thus:
    \begin{equation*} \mathbf{v}(t) = -gt\mathbf{j} + \mathbf{v}_0 \end{equation*}
    🔗
    Next, we integrate the velocity vector function to determine the position vector function \(\mathbf{r}(t)\text{:}\)
    \begin{equation*} \mathbf{r}(t) = \int \mathbf{v}(t)\, dt = \int (-gt\mathbf{j} + \mathbf{v}_0)\, dt = -\frac{1}{2}gt^2\mathbf{j} + \mathbf{v}_0t + \mathbf{C}_2 \end{equation*}
    Using our initial position conditions at \(t=0\text{,}\) the object starts at a height \(h_0\) above the origin, meaning \(\mathbf{r}(0) = h_0\mathbf{j}\text{.}\) This establishes \(\mathbf{C}_2 = h_0\mathbf{j}\text{.}\) Combining all components gives the general position vector function for projectile motion:
    \begin{equation*} \mathbf{r}(t) = \mathbf{v}_0t + \left(h_0 - \frac{1}{2}gt^2\right)\mathbf{j} \end{equation*}
    🔗
    🔗
    🔗

    Remark 2.68. Projectile Path Components.

    A unit vector \(\mathbf{u}\) pointing in the same direction as the launch trajectory vector \(\mathbf{v}_0\) with an angle of elevation \(\alpha\) is given by:
    \begin{equation*} \mathbf{u}=\cos \alpha \mathbf{i}+\sin \alpha \mathbf{j} \end{equation*}
    Letting \(\|\mathbf{v}_0\| = v_0\) denote the initial launch speed of the projectile, the initial velocity vector breaks down into its component components as:
    \begin{equation*} \mathbf{v}_0=v_0\mathbf{u}=(v_0\cos \alpha)\mathbf{i}+(v_0\sin \alpha)\mathbf{j} \end{equation*}
    Provided the projectile launches from ground level (\(h_0=0\)), substituting these components into our position equation yields:
    \begin{equation*} \mathbf{r}(t)=(v_0\cos \alpha)t\mathbf{i}+\left[(v_0\sin \alpha)t-\frac{1}{2}gt^2\right]\mathbf{j} \end{equation*}
    Therefore, the parametric equations describing the projectile’s trajectory path are:
    \begin{equation*} x=(v_0\cos \alpha)t, \quad\quad y=(v_0\sin \alpha)t-\frac{1}{2}gt^2 \end{equation*}
    🔗
    🔗

    Example 2.69.

    A projectile is fired from ground level with an initial speed of \(1500 \, \text{ft/sec}\) and an angle of elevation of \(30^\circ\text{.}\) Find:
    🔗
    1. The velocity vector at time \(t\text{.}\)
      🔗
      🔗
    2. The maximum altitude reached.
      🔗
      🔗
    3. The horizontal range of the projectile.
      🔗
      🔗
    4. The speed at which the projectile strikes the ground.
      🔗
      🔗
    Solution.
    Using the parameters provided, we have the initial speed \(v_0 = 1500\text{,}\) launch angle \(\alpha = 30^\circ\text{,}\) and constant gravitational acceleration \(g = 32 \, \text{ft/sec}^2\text{.}\) The initial horizontal and vertical velocity components evaluate to:
    \begin{equation*} v_{0x} = 1500\cos(30^\circ) = 1500\left(\frac{\sqrt{3}}{2}\right) = 750\sqrt{3} \, \text{ft/sec} \end{equation*}
    \begin{equation*} v_{0y} = 1500\sin(30^\circ) = 1500\left(\frac{1}{2}\right) = 750 \, \text{ft/sec} \end{equation*}
    Thus, the parametric tracking coordinates of the position are \(x(t) = 750\sqrt{3}t\) and \(y(t) = 750t - 16t^2\text{.}\)
    🔗
    1. Differentiating the position components gives the general velocity vector function at time \(t\text{:}\)
      \begin{equation*} \mathbf{v}(t) = 750\sqrt{3}\mathbf{i} + (750 - 32t)\mathbf{j} = \langle 750\sqrt{3}, \, 750 - 32t \rangle \end{equation*}
      🔗
      🔗
    2. The maximum altitude occurs when the vertical velocity component drops to zero (\(v_y(t) = 0\)):
      \begin{equation*} 750 - 32t = 0 \implies t = \frac{750}{32} = \frac{375}{16} \approx 23.44 \, \text{seconds} \end{equation*}
      Substituting this peak time back into our vertical tracking position function yields the maximum altitude:
      \begin{equation*} y_{\max} = 750\left(\frac{375}{16}\right) - 16\left(\frac{375}{16}\right)^2 = \frac{281250}{16} - \frac{140625}{16} = \frac{140625}{16} \approx 8789.06 \, \text{feet} \end{equation*}
      🔗
      🔗
    3. The horizontal range is the distance traveled when the projectile returns to the ground level (\(y(t) = 0\)):
      \begin{equation*} 750t - 16t^2 = 0 \implies t(750 - 16t) = 0 \implies t_{\text{impact}} = \frac{750}{16} = \frac{375}{8} \approx 46.88 \, \text{seconds} \end{equation*}
      Evaluating our horizontal position tracking function at this total flight time gives the range:
      \begin{equation*} x_{\text{range}} = 750\sqrt{3}\left(\frac{375}{8}\right) = \frac{281250\sqrt{3}}{8} \approx 60892.10 \, \text{feet} \end{equation*}
      🔗
      🔗
    4. Because the projectile launches and lands at the same horizontal altitude elevation, the vertical landing speed matches the initial vertical launching speed but points downward. Evaluating the components at impact yields \(\mathbf{v}(t_{\text{impact}}) = \langle 750\sqrt{3}, -750 \rangle\text{.}\) The total scalar speed on impact is:
      \begin{equation*} \|\mathbf{v}(t_{\text{impact}})\| = \sqrt{(750\sqrt{3})^2 + (-750)^2} = \sqrt{1687500 + 562500} = \sqrt{2250000} = 1500 \, \text{ft/sec} \end{equation*}
      🔗
      🔗
