Item 3.
Suppose a 2D curve is defined explicitly by a Cartesian function \(y=f(x)\text{,}\) where \(f'(x)\) is continuous on an interval. Since \(y'\) is the slope of the tangent line at \(P(x,y)\text{,}\) we have \(\tan \theta=y'\) (or \(\theta=\tan^{-1}y'\)), and the arc length function is \(s(x)=\int^x_a\sqrt{1+(y')^2}\, dx\text{.}\) If \(y''\) exists, applying the chain rule yields:
\begin{equation*} \frac{d\theta}{dx} = \frac{d}{dx}(\tan^{-1}y') = \frac{y''}{1+(y')^2} \quad \text{and} \quad \frac{ds}{dx} = \sqrt{1+(y')^2} \end{equation*}
Thus, the curvature \(\kappa\) at the point \(P(x,y)\) is defined as:
\begin{equation*} \kappa=\frac{|y''|}{\left(1+(y')^2\right)^{3/2}} \end{equation*}
