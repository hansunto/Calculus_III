Theorem 5.29. Component Test for Conservative Vector Fields.

Let \(\mathbf{F}(x,y) = P(x,y)\mathbf{i} + Q(x,y)\mathbf{j}\) be a vector field on a simply-connected domain \(D\text{,}\) where \(P\) and \(Q\) have continuous first-order partial derivatives. Then \(\mathbf{F}\) is a conservative vector field if and only if:
\begin{equation*} \frac{\partial P}{\partial y} = \frac{\partial Q}{\partial x} \end{equation*}
throughout \(D\text{.}\)
