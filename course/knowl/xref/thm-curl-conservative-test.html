Theorem 5.47. The Curl of a Gradient Field.

If \(f\) is a function of three variables that has continuous second-order partial derivatives, then:
\begin{equation*} \operatorname{curl}(\nabla f) = \mathbf{0}. \end{equation*}
Since a conservative vector field is defined as one for which there exists a potential scalar function such that \(\mathbf{F}=\nabla f\text{,}\) it follows that if \(\mathbf{F}\) is a conservative vector field, then:
\begin{equation*} \operatorname{curl} \mathbf{F} = \mathbf{0}. \end{equation*}
