    Section 4.3 Double Integrals in Polar coordinates

    In this section, we will explore the use of polar coordinates to evaluate double integrals over regions that are more naturally described in polar form. Polar coordinates provide a convenient way to describe circular and radial symmetry in two-dimensional space.
    🔗
    We will derive the formula for converting a double integral from Cartesian coordinates to polar coordinates and apply it to various examples.
    🔗

    Theorem 4.35. Evaluation Theorem.

    \begin{equation*} \iint_R f(r,\theta)\,dA = \int^{\beta}_{\alpha} \int^{g_2(\theta)}_{g_1(\theta)} f(r,\theta)r\,drd\theta. \end{equation*}
    🔗
    🔗

    Example 4.36.

    Find the area of the region \(R\) that lies outside the circle \(r=a\) and inside the circle \(r=2a\sin\theta\text{.}\)
    🔗
    Solution.
    Step 1: Find the Points of Intersection and Limits
    🔗
    We find the angular intersection points of the two curves by setting their expressions equal to each other:
    \begin{equation*} 2a\sin\theta = a \implies \sin\theta = \frac{1}{2}. \end{equation*}
    In the standard first and second quadrants, this yields the angular bounds:
    \begin{equation*} \alpha = \frac{\pi}{6} \quad \text{and} \quad \beta = \frac{5\pi}{6}. \end{equation*}
    For any angle \(\theta\) between these limits, the radial variable \(r\) starts on the inner circle boundary \(r = a\) and sweeps outward to the outer circle boundary \(r = 2a\sin\theta\text{.}\) Thus, our region \(R\) is bounded by:
    \begin{equation*} \frac{\pi}{6} \leq \theta \leq \frac{5\pi}{6} \quad \text{and} \quad a \leq r \leq 2a\sin\theta. \end{equation*}
    🔗
    Step 2: Evaluate the Double Area Integral
    🔗
    Using the Evaluation Theorem with \(f(r,\theta) = 1\) to calculate area:
    \begin{align*} A \amp = \int_{\pi/6}^{5\pi/6} \int_{a}^{2a\sin\theta} r \, drd\theta = \int_{\pi/6}^{5\pi/6} \left[ \frac{1}{2}r^2 \right]_{a}^{2a\sin\theta} \, d\theta\\ \amp = \frac{1}{2} \int_{\pi/6}^{5\pi/6} \left( 4a^2\sin^2\theta - a^2 \right) \, d\theta\\ \amp = \frac{a^2}{2} \int_{\pi/6}^{5\pi/6} \left( 4\left(\frac{1-\cos(2\theta)}{2}\right) - 1 \right) \, d\theta\\ \amp = \frac{a^2}{2} \int_{\pi/6}^{5\pi/6} \left( 2 - 2\cos(2\theta) - 1 \right) \, d\theta = \frac{a^2}{2} \int_{\pi/6}^{5\pi/6} \left( 1 - 2\cos(2\theta) \right) \, d\theta\\ \amp = \frac{a^2}{2} \Big[ \theta - \sin(2\theta) \Big]_{\pi/6}^{5\pi/6} \end{align*}
    Evaluating at the definite boundary constraints:
    \begin{align*} A \amp = \frac{a^2}{2} \left[ \left( \frac{5\pi}{6} - \sin\left(\frac{5\pi}{3}\right) \right) - \left( \frac{\pi}{6} - \sin\left(\frac{\pi}{3}\right) \right) \right]\\ \amp = \frac{a^2}{2} \left[ \left( \frac{5\pi}{6} - \left(-\frac{\sqrt{3}}{2}\right) \right) - \left( \frac{\pi}{6} - \frac{\sqrt{3}}{2} \right) \right]\\ \amp = \frac{a^2}{2} \left[ \frac{4\pi}{6} + \frac{2\sqrt{3}}{2} \right] = \frac{a^2}{2} \left( \frac{2\pi}{3} + \sqrt{3} \right) = a^2\left(\frac{\pi}{3} + \frac{\sqrt{3}}{2}\right). \end{align*}
    The area of the region is exactly \(a^2\left(\frac{\pi}{3} + \frac{\sqrt{3}}{2}\right)\text{.}\)

    Checkpoint 4.38.

    Find the area of the region \(R\) bounded by one loop of the lemniscate \(r^2=a^2\sin(2\theta)\text{,}\) where \(a \gt 0\text{.}\)
    🔗
    Solution.
    Step 1: Set Up the Integration Loop Bounds
    🔗
    A single loop of the lemniscate \(r^2 = a^2\sin(2\theta)\) is traced out as the radius starts at the origin, sweeps outward, and returns to the origin. This occurs when:
    \begin{equation*} a^2\sin(2\theta) \geq 0 \implies 0 \leq 2\theta \leq \pi \implies 0 \leq \theta \leq \frac{\pi}{2}. \end{equation*}
    For any angle \(\theta\) in the first quadrant, the variable \(r\) spans from the origin \(r=0\) to the boundary line curve \(r = \sqrt{a^2\sin(2\theta)} = a\sqrt{\sin(2\theta)}\text{.}\)
    🔗
    Step 2: Evaluate the Iterated Area Integral
    🔗
    Using the Evaluation Theorem to compute the area of this region:
    \begin{align*} A \amp = \int_{0}^{\pi/2} \int_{0}^{a\sqrt{\sin(2\theta)}} r \, drd\theta\\ \amp = \int_{0}^{\pi/2} \left[ \frac{1}{2}r^2 \right]_{0}^{a\sqrt{\sin(2\theta)}} \, d\theta\\ \amp = \int_{0}^{\pi/2} \frac{1}{2}a^2\sin(2\theta) \, d\theta\\ \amp = \frac{a^2}{2} \left[ -\frac{1}{2}\cos(2\theta) \right]_{0}^{\pi/2}\\ \amp = -\frac{a^2}{4} \Big( \cos(\pi) - \cos(0) \Big)\\ \amp = -\frac{a^2}{4} \Big( -1 - 1 \Big) = -\frac{a^2}{4}(-2) = \frac{a^2}{2}. \end{align*}
    The area enclosed by one loop of the lemniscate is exactly \(\frac{a^2}{2}\text{.}\)

    Theorem 4.40. Change of Variables Formula.

    \begin{equation*} \iint_R f(x,y)\,dydx = \iint_{R} f(r\cos\theta, r\sin\theta)r\,drd\theta. \end{equation*}
    🔗
    🔗

    Example 4.41.

    Use polar coordinates to evaluate
    \begin{equation*} \int^a_{-a}\int^{\sqrt{a^2-x^2}}_0(x^2+y^2)^{3/2}\,dydx. \end{equation*}
    🔗
    Solution.
    Step 1: Analyze the Region of Integration \(R\)
    🔗
    The original limits are given in Cartesian coordinates where:
    \begin{equation*} -a \leq x \leq a \quad \text{and} \quad 0 \leq y \leq \sqrt{a^2-x^2}. \end{equation*}
    The upper boundary curve \(y = \sqrt{a^2-x^2}\) represents the upper half of a circle centered at the origin with radius \(a\text{.}\) Since \(x\) runs from \(-a\) to \(a\text{,}\) the region \(R\) is a complete upper semicircle sitting in the first and second quadrants.
    🔗
    Converting this region to polar coordinates, the radius \(r\) starts at the origin and sweeps outward to the boundary circle \(r = a\text{.}\) The angle \(\theta\) opens from the positive \(x\)-axis (\(0\)) all the way around to the negative \(x\)-axis (\(\pi\)). Therefore, the polar boundaries are:
    \begin{equation*} 0 \leq \theta \leq \pi \quad \text{and} \quad 0 \leq r \leq a. \end{equation*}
    🔗
    Step 2: Convert and Evaluate the Integral
    🔗
    Using the relationship \(x^2 + y^2 = r^2\) and changing the differential element to polar form (\(dA = r\,drd\theta\)), we substitute our values into the formula:
    \begin{equation*} \int^a_{-a}\int^{\sqrt{a^2-x^2}}_0 (x^2+y^2)^{3/2} \,dydx = \int_{0}^{\pi} \int_{0}^{a} (r^2)^{3/2} \cdot r \,drd\theta \end{equation*}
    Simplifying the inner integrand components gives:
    \begin{equation*} = \int_{0}^{\pi} \int_{0}^{a} r^3 \cdot r \,drd\theta = \int_{0}^{\pi} \int_{0}^{a} r^4 \,drd\theta \end{equation*}
    🔗
    Since the limits of integration are entirely constants and the variables inside the integrand are completely independent, we can separate this double integral into the product of two single integrals:
    \begin{align*} \int_{0}^{\pi} \int_{0}^{a} r^4 \,drd\theta \amp = \left( \int_{0}^{\pi} \,d\theta \right) \left( \int_{0}^{a} r^4 \,dr \right)\\ \amp = \Big[ \theta \Big]_{0}^{\pi} \cdot \left[ \frac{1}{5}r^5 \right]_{0}^{a}\\ \amp = (\pi - 0) \cdot \left( \frac{1}{5}a^5 - 0 \right)\\ \amp = \frac{\pi a^5}{5}. \end{align*}
    The value of the double integral evaluated using polar coordinates is exactly \(\frac{\pi a^5}{5}\text{.}\)

    Example 4.43.

    Find the volume \(V\) of the solid that is bounded by the paraboloid \(z=4-x^2-y^2\) and the \(xy\)-plane.
    🔗
    Solution.
    Step 1: Identify the Base Region \(R\) in the \(xy\)-plane
    🔗
    The solid sits beneath the paraboloid ceiling and is bounded below by the \(xy\)-plane, where \(z = 0\text{.}\) We find the footprint or boundary curve of this base region \(R\) by setting the paraboloid equation to zero:
    \begin{equation*} 4 - x^2 - y^2 = 0 \implies x^2 + y^2 = 4. \end{equation*}
    This represents a complete circle centered at the origin with a radius of \(r = 2\text{.}\)
    🔗
    Converting this full circular disk to polar coordinates gives us constant boundaries that span all four quadrants:
    \begin{equation*} 0 \leq \theta \leq 2\pi \quad \text{and} \quad 0 \leq r \leq 2. \end{equation*}
    🔗
    Step 2: Convert and Evaluate the Volume Integral
    🔗
    The height function inside the double integral can be converted to polar coordinates using \(x^2 + y^2 = r^2\text{,}\) which yields \(z = 4 - r^2\text{.}\) Substituting this ceiling function and the polar area differential element (\(dA = r\,drd\theta\)) into our volume formula gives:
    \begin{equation*} V = \iint_R (4 - x^2 - y^2) \, dA = \int_{0}^{2\pi} \int_{0}^{2} (4 - r^2)r \, dr d\theta \end{equation*}
    Distributing the inner radial factor across the integrand parameters:
    \begin{equation*} = \int_{0}^{2\pi} \int_{0}^{2} (4r - r^3) \, dr d\theta \end{equation*}
    🔗
    Since the boundary parameters are entirely composed of constants and the individual components inside the expression are mathematically independent, we can factor the double integral into a simple product of two independent single integrals:
    \begin{align*} V \amp = \left( \int_{0}^{2\pi} \,d\theta \right) \left( \int_{0}^{2} (4r - r^3) \,dr \right)\\ \amp = \Big[ \theta \Big]_{0}^{2\pi} \cdot \left[ 2r^2 - \frac{1}{4}r^4 \right]_{0}^{2}\\ \amp = (2\pi - 0) \cdot \left( \left(2(2)^2 - \frac{1}{4}(2)^4\right) - 0 \right)\\ \amp = 2\pi \cdot \left( 8 - \frac{16}{4} \right)\\ \amp = 2\pi \cdot (8 - 4) = 2\pi \cdot 4 = 8\pi. \end{align*}
    The total volume of the solid enclosed under the paraboloid is exactly \(8\pi\text{.}\)

    1.

    Change the Cartesian integral into an equivalent polar integral then evaluate the integral:
    🔗
    1. \(\displaystyle \displaystyle\int^0_{-1}\int^0_{-\sqrt{1-x^2}} \frac{2}{1+\sqrt{x^2+y^2}}\,dydx\)
      🔗
      🔗
    2. \(\displaystyle \displaystyle\int^1_{-1}\int^{\sqrt{1-y^2}}_{-\sqrt{1-y^2}}\ln(x^2+y^2+1)\,dxdy\)
      🔗
      🔗
    Solution.
    1. Solution to Part (a):
      🔗
      The original Cartesian limits restrict the domain to:
      \begin{equation*} -1 \leq x \leq 0 \quad \text{and} \quad -\sqrt{1-x^2} \leq y \leq 0. \end{equation*}
      The curve \(y = -\sqrt{1-x^2}\) describes the lower half of a unit circle centered at the origin. Because both \(x\) and \(y\) values are exclusively negative or zero, the integration domain \(R\) is a quarter-circle resting entirely inside the third quadrant.
      🔗
      Converting to polar variables, the radial sweep starts at the origin \(r=0\) and extends out to the curve boundary \(r=1\text{.}\) The angle \(\theta\) sweeps across the third quadrant from \(\pi\) to \(\frac{3\pi}{2}\text{:}\)
      \begin{equation*} \pi \leq \theta \leq \frac{3\pi}{2} \quad \text{and} \quad 0 \leq r \leq 1. \end{equation*}
      🔗
      Substituting the polar translation rules (\(\sqrt{x^2+y^2}=r\) and \(dydx = r\,drd\theta\)) transforms the expression into:
      \begin{equation*} \int_{\pi}^{3\pi/2} \int_{0}^{1} \frac{2}{1+r} \cdot r \, dr d\theta = \left( \int_{\pi}^{3\pi/2} \,d\theta \right) \left( \int_{0}^{1} \frac{2r}{1+r} \,dr \right) \end{equation*}
      Evaluating the angular integral yields \(\frac{3\pi}{2} - \pi = \frac{\pi}{2}\text{.}\) For the radial integral, we apply algebraic long division:
      \begin{equation*} \int_{0}^{1} \frac{2r}{1+r} \,dr = 2\int_{0}^{1} \frac{(r+1)-1}{1+r} \,dr = 2\int_{0}^{1} \left(1 - \frac{1}{1+r}\right) \,dr \end{equation*}
      \begin{equation*} = 2 \Big[ r - \ln|1+r| \Big]_{0}^{1} = 2 \Big( (1 - \ln 2) - (0 - \ln 1) \Big) = 2 - 2\ln 2. \end{equation*}
      Multiplying our two separate solutions together gives:
      \begin{equation*} \text{Value} = \frac{\pi}{2}(2 - 2\ln 2) = \pi(1 - \ln 2). \end{equation*}
      🔗
      🔗
    2. Solution to Part (b):
      🔗
      The original integration walls are given by:
      \begin{equation*} -1 \leq y \leq 1 \quad \text{and} \quad -\sqrt{1-y^2} \leq x \leq \sqrt{1-y^2}. \end{equation*}
      This maps out the complete full unit circular disk centered at the origin. Converting to polar tracking variables yields:
      \begin{equation*} 0 \leq \theta \leq 2\pi \quad \text{and} \quad 0 \leq r \leq 1. \end{equation*}
      Substituting \(x^2+y^2 = r^2\) and the differential multiplier element (\(dxdy = r\,drd\theta\)) gives:
      \begin{equation*} \int_{0}^{2\pi} \int_{0}^{1} \ln(r^2+1) \cdot r \, dr d\theta = \left( \int_{0}^{2\pi} \,d\theta \right) \left( \int_{0}^{1} r\ln(r^2+1) \,dr \right) \end{equation*}
      The outer integral maps directly to \(2\pi\text{.}\) For the inner single radial integral, we perform a quick \(u\)-substitution:
      \begin{equation*} u = r^2+1 \implies du = 2r\,dr \implies \frac{1}{2}\,du = r\,dr. \end{equation*}
      Updating limits: when \(r=0 \implies u=1\text{,}\) and when \(r=1 \implies u=2\text{.}\)
      \begin{equation*} \int_{0}^{1} r\ln(r^2+1) \,dr = \frac{1}{2} \int_{1}^{2} \ln(u) \,du = \frac{1}{2} \Big[ u\ln(u) - u \Big]_{1}^{2} \end{equation*}
      \begin{equation*} = \frac{1}{2} \Big( (2\ln 2 - 2) - (1\ln 1 - 1) \Big) = \frac{1}{2} (2\ln 2 - 2 + 1) = \ln 2 - \frac{1}{2}. \end{equation*}
      Combining the structural components yields:
      \begin{equation*} \text{Value} = 2\pi \left(\ln 2 - \frac{1}{2}\right) = 2\pi\ln 2 - \pi. \end{equation*}
      🔗
      🔗

    2.

    Find the area of the region cut from the first quadrant (first octant base) by the cardioid \(r=1+\sin\theta\text{.}\)
    🔗
    Solution.
    Step 1: Set Up the Polar Integration Boundaries
    🔗
    The phrase "cut from the first quadrant" restricts our angle tracking bounds to \(0 \leq \theta \leq \frac{\pi}{2}\text{.}\) Inside this angular sweep, the radius starts at the origin \(r=0\) and pushes out to hit the outer wall profile of the cardioid \(r = 1+\sin\theta\text{:}\)
    \begin{equation*} 0 \leq \theta \leq \frac{\pi}{2} \quad \text{and} \quad 0 \leq r \leq 1+\sin\theta. \end{equation*}
    🔗
    Step 2: Evaluate the Area Integration
    🔗
    Using the polar double integral definition for finding area (\(dA = r\,drd\theta\)):
    \begin{equation*} A = \int_{0}^{\pi/2} \int_{0}^{1+\sin\theta} r \, dr d\theta = \int_{0}^{\pi/2} \left[ \frac{1}{2}r^2 \right]_{0}^{1+\sin\theta} \, d\theta \end{equation*}
    \begin{equation*} = \frac{1}{2} \int_{0}^{\pi/2} (1 + \sin\theta)^2 \, d\theta = \frac{1}{2} \int_{0}^{\pi/2} (1 + 2\sin\theta + \sin^2\theta) \, d\theta \end{equation*}
    Using the half-angle reduction identity \(\sin^2\theta = \frac{1-\cos(2\theta)}{2}\text{:}\)
    \begin{equation*} = \frac{1}{2} \int_{0}^{\pi/2} \left( 1 + 2\sin\theta + \frac{1}{2} - \frac{1}{2}\cos(2\theta) \right) \, d\theta = \frac{1}{2} \int_{0}^{\pi/2} \left( \frac{3}{2} + 2\sin\theta - \frac{1}{2}\cos(2\theta) \right) \, d\theta \end{equation*}
    \begin{equation*} = \frac{1}{2} \left[ \frac{3}{2}\theta - 2\cos\theta - \frac{1}{4}\sin(2\theta) \right]_{0}^{\pi/2} \end{equation*}
    \begin{equation*} = \frac{1}{2} \left[ \left( \frac{3\pi}{4} - 0 - 0 \right) - \left( 0 - 2 - 0 \right) \right] = \frac{1}{2} \left( \frac{3\pi}{4} + 2 \right) = \frac{3\pi}{8} + 1. \end{equation*}
    The total area cut inside the first quadrant is exactly \(\frac{3\pi}{8} + 1\text{.}\)

    3.

    Find the average height of the hemisphere surface \(z=\sqrt{2^2-x^2-y^2}\) above the disk \(x^2+y^2\leq 4\) in the \(xy\)-plane.
    🔗
    Solution.
    Step 1: Determine the Base Area
    🔗
    The base region \(R\) is a complete circular disk described by \(x^2+y^2 \leq 4\text{,}\) which yields a clear radius of \(R_0 = 2\text{.}\) The geometric area of this full circular base region is:
    \begin{equation*} \text{Area}(R) = \pi R_0^2 = \pi (2)^2 = 4\pi. \end{equation*}
    Converted to polar coordinate conditions, our tracking parameters are: \(0 \leq \theta \leq 2\pi\) and \(0 \leq r \leq 2\text{.}\)
    🔗
    Step 2: Set Up and Evaluate the Average Value Theorem Form
    🔗
    According to the average value theorem, the average height \(\bar{z}\) is computed by finding the total volume under the dome surface and dividing it by the total area of the floor disk:
    \begin{equation*} \bar{z} = \frac{1}{\text{Area}(R)} \iint_R z \, dA = \frac{1}{4\pi} \int_{0}^{2\pi} \int_{0}^{2} \sqrt{4-r^2} \cdot r \, dr d\theta \end{equation*}
    Since the bounds are completely constants and the inner function components are independent, we can pull apart the double integral:
    \begin{equation*} \bar{z} = \frac{1}{4\pi} \left( \int_{0}^{2\pi} \,d\theta \right) \left( \int_{0}^{2} r\sqrt{4-r^2} \,dr \right) = \frac{2\pi}{4\pi} \int_{0}^{2} r(4-r^2)^{1/2} \,dr = \frac{1}{2} \int_{0}^{2} r(4-r^2)^{1/2} \,dr \end{equation*}
    We compute the remaining single integral using \(u\)-substitution:
    \begin{equation*} u = 4-r^2 \implies du = -2r\,dr \implies -\frac{1}{2}\,du = r\,dr. \end{equation*}
    Updating parameters: when \(r=0 \implies u=4\text{,}\) and when \(r=2 \implies u=0\text{.}\)
    \begin{equation*} \bar{z} = \frac{1}{2} \int_{4}^{0} \left(-\frac{1}{2} u^{1/2}\right) \,du = \frac{1}{4} \int_{0}^{4} u^{1/2} \,du \end{equation*}
    \begin{equation*} = \frac{1}{4} \left[ \frac{2}{3}u^{3/2} \right]_{0}^{4} = \frac{1}{4} \left( \frac{2}{3}(4)^{3/2} - 0 \right) = \frac{1}{4} \left( \frac{2}{3} \cdot 8 \right) = \frac{16}{12} = \frac{4}{3}. \end{equation*}
    The average height of the hemisphere surface above the floor base disk is exactly \(\frac{4}{3}\text{.}\)
