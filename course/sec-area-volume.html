    Section 4.2 Area and Volume

    In this section, we will explore how to use double integrals to calculate the area of a region in the plane and the volume of a solid in three-dimensional space. We will also discuss the applications of these concepts in various fields.
    🔗

    Definition 4.21. Area.

    The area of a closed, bounded plane region \(R\) is
    \begin{equation*} A = \iint_R \,dA. \end{equation*}
    🔗
    🔗

    Example 4.22.

    Find the area of the region \(R\) bounded by \(y=x\) and \(y=x^2\) in the first quadrant.
    🔗
    Solution.
    Step 1: Find Points of Intersection and Setup Boundaries
    🔗
    We find where the two curves intersect by setting them equal to each other:
    \begin{equation*} x = x^2 \implies x^2 - x = 0 \implies x(x-1) = 0 \end{equation*}
    This gives intersection boundaries at \(x = 0\) and \(x = 1\text{.}\) On the interval \([0,1]\text{,}\) the line \(y = x\) lies above the parabola \(y = x^2\text{.}\) Thus, our region is described by:
    \begin{equation*} 0 \leq x \leq 1 \quad \text{and} \quad x^2 \leq y \leq x \end{equation*}
    🔗
    Step 2: Evaluate the Double Integral
    🔗
    Using the definition of area, we set up the iterated integral with respect to \(y\) first:
    🔗
    \begin{align*} A \amp = \int_{0}^{1} \int_{x^2}^{x} \,dy\,dx\\ \amp = \int_{0}^{1} \Big[ y \Big]_{y=x^2}^{y=x} \,dx\\ \amp = \int_{0}^{1} (x - x^2) \,dx\\ \amp = \left[ \frac{1}{2}x^2 - \frac{1}{3}x^3 \right]_{0}^{1}\\ \amp = \left( \frac{1}{2}(1)^2 - \frac{1}{3}(1)^3 \right) - 0\\ \amp = \frac{1}{2} - \frac{1}{3} = \frac{1}{6} \end{align*}
    The area of the region is exactly \(\frac{1}{6}\text{.}\)
    🔗
    🔗
    🔗

    Example 4.24.

    Find the area of the region \(R\) enclosed by the parabola \(y=x^2\) and the line \(y=x+2\text{.}\)
    🔗
    Solution.
    Step 1: Find Points of Intersection and Setup Boundaries
    🔗
    We find where the line and the parabola intersect by setting their equations equal to each other:
    \begin{equation*} x^2 = x + 2 \implies x^2 - x - 2 = 0 \implies (x-2)(x+1) = 0 \end{equation*}
    This gives intersection boundaries at \(x = -1\) and \(x = 2\text{.}\) On the interval \([-1, 2]\text{,}\) the line \(y = x + 2\) lies above the parabola \(y = x^2\text{.}\) Thus, our region \(R\) is described by:
    \begin{equation*} -1 \leq x \leq 2 \quad \text{and} \quad x^2 \leq y \leq x + 2 \end{equation*}
    🔗
    Step 2: Evaluate the Area Double Integral
    🔗
    Using the definition of area, we set up the iterated integral with respect to \(y\) first:
    🔗
    \begin{align*} A \amp = \int_{-1}^{2} \int_{x^2}^{x+2} \,dy\,dx\\ \amp = \int_{-1}^{2} \Big[ y \Big]_{y=x^2}^{y=x+2} \,dx\\ \amp = \int_{-1}^{2} (x + 2 - x^2) \,dx\\ \amp = \left[ \frac{1}{2}x^2 + 2x - \frac{1}{3}x^3 \right]_{-1}^{2} \end{align*}
    Now, evaluating at the upper and lower limits:
    🔗
    \begin{align*} \text{Upper limit } (x=2): \amp \quad \frac{1}{2}(2)^2 + 2(2) - \frac{1}{3}(2)^3 = 2 + 4 - \frac{8}{3} = \frac{10}{3}\\ \text{Lower limit } (x=-1): \amp \quad \frac{1}{2}(-1)^2 + 2(-1) - \frac{1}{3}(-1)^3 = \frac{1}{2} - 2 + \frac{1}{3} = -\frac{7}{6} \end{align*}
    Subtracting the lower limit evaluation from the upper limit evaluation yields:
    \begin{align*} A \amp = \frac{10}{3} - \left( -\frac{7}{6} \right)\\ \amp = \frac{20}{6} + \frac{7}{6} = \frac{27}{6} = \frac{9}{2} \end{align*}
    The total area of the enclosed region is exactly \(\frac{9}{2}\) (or \(4.5\)).
    🔗
    🔗
    🔗

    Definition 4.26. Average Value.

    The average value of \(f\) over \(R\) is
    \begin{equation*} \text{Average Value} = \frac{1}{\text{area of } R} \iint_R f \,dA. \end{equation*}
    🔗
    🔗

    Example 4.27.

    Find the average value of \(f(x,y)=x\cos(xy)\) over the rectangle \(R: 0\leq x \leq \pi, \quad 0\leq y \leq 1\text{.}\)
    🔗
    Solution.
    Step 1: Calculate the Area of Region \(R\)
    🔗
    The region \(R\) is a simple rectangle bounded by constant lines. The width along the \(x\)-axis is \(\pi\) and the height along the \(y\)-axis is \(1\text{.}\) Therefore, the area of \(R\) is:
    \begin{equation*} \text{Area}(R) = \text{width} \times \text{height} = \pi \times 1 = \pi. \end{equation*}
    🔗
    Step 2: Evaluate the Double Integral Over \(R\)
    🔗
    To evaluate \(\iint_R x\cos(xy) \,dA\text{,}\) we integrate with respect to \(y\) first because the inner derivative factor \(x\) is already present in the integrand, simplifying the anti-differentiation:
    🔗
    \begin{align*} \iint_R x\cos(xy) \,dA \amp = \int_{0}^{\pi} \int_{0}^{1} x\cos(xy) \,dy\,dx\\ \amp = \int_{0}^{\pi} \Big[ \sin(xy) \Big]_{y=0}^{y=1} \,dx\\ \amp = \int_{0}^{\pi} \left( \sin(x \cdot 1) - \sin(x \cdot 0) \right) \,dx\\ \amp = \int_{0}^{\pi} \sin(x) \,dx\\ \amp = \Big[ -\cos(x) \Big]_{0}^{\pi}\\ \amp = -\cos(\pi) - (-\cos(0))\\ \amp = -(-1) + 1 = 1 + 1 = 2 \end{align*}
    Step 3: Calculate the Final Average Value
    🔗
    Using the definition, we divide the value of the double integral by the total area computed in Step 1:
    \begin{equation*} \text{Average Value} = \frac{1}{\pi} \cdot 2 = \frac{2}{\pi}. \end{equation*}
    The average value of the function over the given rectangle is exactly \(\frac{2}{\pi}\text{.}\)
    🔗
    🔗
    🔗

    Definition 4.29. Volume.

    The volume of a solid lying under a surface \(z = f(x,y)\) over a closed, bounded plane region \(R\) is
    \begin{equation*} V = \iint_R f(x,y)\,dA. \end{equation*}
    🔗
    🔗

    Example 4.30.

    Find the volume \(V\) of the solid in the first octant bounded by the coordinate planes, the paraboloid \(z=x^2+y^2+1\text{,}\) and the plane \(2x+y=2\text{.}\)
    🔗
    Solution.
    Step 1: Identify the Base Region \(R\)
    🔗
    In the first octant, the coordinate plane boundaries give \(x \geq 0\text{,}\) \(y \geq 0\text{,}\) and \(z \geq 0\text{.}\) The base region \(R\) sits in the \(xy\)-plane (\(z=0\)) and is a triangle bounded by the lines \(x = 0\) (the \(y\)-axis), \(y = 0\) (the \(x\)-axis), and the intersection line \(2x + y = 2\text{.}\)
    🔗
    Rewriting the bounding line in terms of \(y\) gives \(y = 2 - 2x\text{.}\) Finding the intercepts of this line:
    • When \(y = 0\text{:}\) \(2x = 2 \implies x = 1\)
      🔗
      🔗
    • When \(x = 0\text{:}\) \(y = 2\)
      🔗
      🔗
    Therefore, setting this up as a Type I region, the boundaries are:
    \begin{equation*} 0 \leq x \leq 1 \quad \text{and} \quad 0 \leq y \leq 2 - 2x \end{equation*}
    🔗
    Step 2: Set Up and Evaluate the Iterated Integral
    🔗
    The ceiling of the solid is the paraboloid \(z = x^2 + y^2 + 1\text{.}\) We integrate this function over the triangular region \(R\text{:}\)
    \begin{equation*} V = \int_{0}^{1} \int_{0}^{2-2x} (x^2 + y^2 + 1) \, dy \, dx \end{equation*}
    First, we integrate the inner expression with respect to \(y\text{:}\)
    🔗
    \begin{align*} \int_{0}^{2-2x} (x^2 + y^2 + 1) \, dy \amp = \left[ x^2y + \frac{1}{3}y^3 + y \right]_{y=0}^{y=2-2x}\\ \amp = x^2(2-2x) + \frac{1}{3}(2-2x)^3 + (2-2x)\\ \amp = 2x^2 - 2x^3 + \frac{1}{3}(8 - 24x + 24x^2 - 8x^3) + 2 - 2x\\ \amp = 2x^2 - 2x^3 + \frac{8}{3} - 8x + 8x^2 - \frac{8}{3}x^3 + 2 - 2x\\ \amp = \left(-2 - \frac{8}{3}\right)x^3 + (2 + 8)x^2 + (-8 - 2)x + \left(\frac{8}{3} + 2\right)\\ \amp = -\frac{14}{3}x^3 + 10x^2 - 10x + \frac{14}{3} \end{align*}
    Next, we integrate this simplified single variable polynomial expression with respect to \(x\) over the outer boundary:
    🔗
    \begin{align*} V \amp = \int_{0}^{1} \left( -\frac{14}{3}x^3 + 10x^2 - 10x + \frac{14}{3} \right) \, dx\\ \amp = \left[ -\frac{14}{12}x^4 + \frac{10}{3}x^3 - 5x^2 + \frac{14}{3}x \right]_{0}^{1}\\ \amp = \left( -\frac{7}{6}(1)^4 + \frac{10}{3}(1)^3 - 5(1)^2 + \frac{14}{3}(1) \right) - 0\\ \amp = -\frac{7}{6} + \frac{10}{3} - 5 + \frac{14}{3}\\ \amp = -\frac{7}{6} + \frac{24}{3} - 5\\ \amp = -\frac{7}{6} + 8 - 5 = -\frac{7}{6} + 3\\ \amp = -\frac{7}{6} + \frac{18}{6} = \frac{11}{6} \end{align*}
    The total volume of the solid in the first octant is exactly \(\frac{11}{6}\text{.}\)

    Exercises Exercises

    1.

    Find the volume \(V\) of the solid that lies in the first octant and is bounded by the three coordinate planes and the cylinders \(x^2+y^2=9\) and \(y^2+z^2=9\text{.}\)
    🔗
    Solution.
    Step 1: Identify the Base Region \(R\)
    🔗
    In the first octant, the coordinate planes give \(x \geq 0\text{,}\) \(y \geq 0\text{,}\) and \(z \geq 0\text{.}\) The base region \(R\) in the \(xy\)-plane is bounded by the circular cylinder base \(x^2+y^2=9\text{.}\) Isolating \(x\) gives \(x = \sqrt{9-y^2}\text{.}\) The region can be described as a Type II domain tracking from \(y=0\) to \(y=3\text{:}\)
    \begin{equation*} 0 \leq y \leq 3 \quad \text{and} \quad 0 \leq x \leq \sqrt{9-y^2} \end{equation*}
    🔗
    The ceiling surface is given by isolating \(z\) from the second cylinder equation: \(z = \sqrt{9-y^2}\text{.}\)
    🔗
    Step 2: Evaluate the Volume Integral
    🔗
    Using a double integral, we have:
    \begin{equation*} V = \iint_R \sqrt{9-y^2} \, dA = \int_{0}^{3} \int_{0}^{\sqrt{9-y^2}} \sqrt{9-y^2} \, dx \, dy \end{equation*}
    Integrating with respect to \(x\) first:
    \begin{equation*} \int_{0}^{\sqrt{9-y^2}} \sqrt{9-y^2} \, dx = \left[ x\sqrt{9-y^2} \right]_{x=0}^{x=\sqrt{9-y^2}} = (\sqrt{9-y^2})^2 - 0 = 9 - y^2 \end{equation*}
    Now, integrating the remaining expression with respect to \(y\text{:}\)
    \begin{equation*} V = \int_{0}^{3} (9-y^2) \, dy = \left[ 9y - \frac{1}{3}y^3 \right]_{0}^{3} = \left( 9(3) - \frac{1}{3}(27) \right) - 0 = 27 - 9 = 18 \end{equation*}
    The volume of the intersection solid in the first octant is exactly \(18\text{.}\)
    🔗
    🔗
    🔗

    2.

    Reverse the order of integration and evaluate the integral
    \begin{equation*} \int^\pi_0 \int^\pi_x \frac{\sin y}{y} \, dydx \end{equation*}
    🔗
    Solution.
    Step 1: Analyze and Reverse the Bounds
    🔗
    The original Type I description is bounded by \(0 \leq x \leq \pi\) and \(x \leq y \leq \pi\text{.}\) This represents a triangular region bounded by the lines \(y = x\text{,}\) \(x = 0\text{,}\) and \(y = \pi\text{.}\) Reversing this to a Type II region (\(dxdy\)) gives:
    \begin{equation*} 0 \leq y \leq \pi \quad \text{and} \quad 0 \leq x \leq y \end{equation*}
    🔗
    Step 2: Evaluate the Transformed Integral
    🔗
    Applying the reversed limits of integration:
    \begin{equation*} \int_{0}^{\pi} \int_{0}^{y} \frac{\sin y}{y} \, dx \, dy \end{equation*}
    Evaluating the inner integral with respect to \(x\text{:}\)
    \begin{equation*} \int_{0}^{y} \frac{\sin y}{y} \, dx = \left[ x \frac{\sin y}{y} \right]_{x=0}^{x=y} = y \left(\frac{\sin y}{y}\right) - 0 = \sin y \end{equation*}
    Now evaluating the outer integral with respect to \(y\text{:}\)
    \begin{equation*} \int_{0}^{\pi} \sin y \, dy = \Big[ -\cos y \Big]_{0}^{\pi} = -\cos(\pi) - (-\cos 0) = -(-1) + 1 = 2 \end{equation*}
    The value of the integral is exactly \(2\text{.}\)
    🔗
    🔗
    🔗

    3.

    Find the volume of the solid whose base is a region in the \(xy\)-plane that is bounded by the parabola \(y=4-x^2\) and the line \(y=3x\text{,}\) while the top of the solid is bounded by the plane \(z=x+4\text{.}\)
    🔗
    Solution.
    Step 1: Find the Intersection Boundaries of the Base
    🔗
    We set the base boundaries equal to find their intersection coordinates:
    \begin{equation*} 4 - x^2 = 3x \implies x^2 + 3x - 4 = 0 \implies (x+4)(x-1) = 0 \end{equation*}
    The intersection points occur at \(x = -4\) and \(x = 1\text{.}\) On the interval \([-4, 1]\text{,}\) the parabola \(y = 4-x^2\) stays above the line \(y = 3x\text{.}\) The Type I boundaries are:
    \begin{equation*} -4 \leq x \leq 1 \quad \text{and} \quad 3x \leq y \leq 4-x^2 \end{equation*}
    🔗
    Step 2: Evaluate the Volume Integral
    🔗
    We compute the volume by integrating the ceiling function \(z = x+4\) over \(R\text{:}\)
    \begin{equation*} V = \int_{-4}^{1} \int_{3x}^{4-x^2} (x+4) \, dy \, dx \end{equation*}
    Integrating with respect to \(y\text{:}\)
    \begin{equation*} \int_{3x}^{4-x^2} (x+4) \, dy = (x+4)\Big[ y \Big]_{3x}^{4-x^2} = (x+4)(4 - x^2 - 3x) \end{equation*}
    Expanding the polynomial product results in:
    \begin{equation*} = 4x - x^3 - 3x^2 + 16 - 4x^2 - 12x = -x^3 - 7x^2 - 8x + 16 \end{equation*}
    Now, evaluating the single integral with respect to \(x\text{:}\)
    \begin{align*} V \amp = \int_{-4}^{1} (-x^3 - 7x^2 - 8x + 16) \, dx = \left[ -\frac{1}{4}x^4 - \frac{7}{3}x^3 - 4x^2 + 16x \right]_{-4}^{1}\\ \amp = \left( -\frac{1}{4} - \frac{7}{3} - 4 + 16 \right) - \left( -\frac{256}{4} - \frac{7(-64)}{3} - 4(16) + 16(-4) \right)\\ \amp = \left( \frac{113}{12} \right) - \left( -64 + \frac{448}{3} - 64 - 64 \right) = \frac{113}{12} - \left( -\frac{128}{3} \right) = \frac{113 + 512}{12} = \frac{625}{12} \end{align*}
    The volume of the solid is exactly \(\frac{625}{12}\text{.}\)

    4.

    Find the volume of the wedge cut from the first octant by the cylinder \(z=12-3y^2\) and the plane \(x+y=2\text{.}\)
    🔗
    Solution.
    Step 1: Determine the Bounds in the Base Plane
    🔗
    In the first octant, \(x \geq 0\) and \(y \geq 0\text{.}\) The base triangle \(R\) is bounded by the line \(x+y=2 \implies x = 2-y\text{.}\) Setting \(x=0\) gives the upper limit for \(y\) at \(y=2\text{.}\) Setting this up as a Type II integral gives:
    \begin{equation*} 0 \leq y \leq 2 \quad \text{and} \quad 0 \leq x \leq 2-y \end{equation*}
    🔗
    Step 2: Reverse the Limits to Type I (\(dydx\))
    🔗
    To switch to a Type I layout, we examine the vertical bounds of the region across a horizontal span. The region spans along the \(x\)-axis from \(x = 0\) to \(x = 1\text{.}\) For a fixed value of \(x\text{,}\) a vertical strip starts at the bottom boundary line \(y = 0\) and sweeps upward until it reaches the line \(y = x\text{.}\) The reversed boundary descriptions are:
    \begin{equation*} 0 \leq x \leq 1 \quad \text{and} \quad 0 \leq y \leq x. \end{equation*}
    🔗
    Step 3: Setup and Evaluate the Transformed Integral
    🔗
    Using our new boundary configurations, the original unintegrable problem transitions into a solvable iterated integral:
    \begin{equation*} \int_{0}^{1} \int_{0}^{x} e^{x^2} \, dy \, dx \end{equation*}
    We compute the inner integral with respect to \(y\text{,}\) treating \(e^{x^2}\) as a constant factor:
    \begin{equation*} \int_{0}^{x} e^{x^2} \, dy = \left[ y e^{x^2} \right]_{y=0}^{y=x} = x e^{x^2} - 0 = x e^{x^2}. \end{equation*}
    Next, we substitute this result back into the outer structural layer to integrate with respect to \(x\text{:}\)
    \begin{equation*} \int_{0}^{1} x e^{x^2} \, dx \end{equation*}
    We evaluate this final layer using \(u\)-substitution. Let:
    \begin{equation*} u = x^2 \implies du = 2x \, dx \implies \frac{1}{2}\,du = x \, dx. \end{equation*}
    Updating our definite limits of integration:
    • When \(x = 0\text{:}\) \(u = 0^2 = 0\)
      🔗
      🔗
    • When \(x = 1\text{:}\) \(u = 1^2 = 1\)
      🔗
      🔗
    Substituting these elements back into the equation yields:
    \begin{align*} \int_{0}^{1} \frac{1}{2} e^u \, du \amp = \left[ \frac{1}{2} e^u \right]_{0}^{1}\\ \amp = \frac{1}{2}e^1 - \frac{1}{2}e^0\\ \amp = \frac{1}{2}(e - 1). \end{align*}
    The exact value of the reversed double integral is \(\frac{e - 1}{2}\text{.}\)
